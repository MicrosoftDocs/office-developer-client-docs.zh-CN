# <a name="doclastedit-function"></a>DOCLASTEDIT 函数

按序列值返回上次编辑文档的日期和时间。
  
## <a name="syntax"></a>语法

DOCLASTEDIT()
  
## <a name="remarks"></a>注解

使用[FORMAT](format-function.md)函数设置该值。
