# <a name="sin-function-visioshapesheet"></a>SIN 函数 (VisioShapeSheet)

返回角度的正弦值。 
  
## <a name="syntax"></a>语法

SIN (* * *angle* * *) 
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _angle_ <br/> |必需  <br/> |**数值** <br/> |要获取其正弦值的角度。  <br/> |
   
### <a name="return-value"></a>返回值

数字
  
## <a name="remarks"></a>说明

SIN 函数使用弧度, 除非您以_角度_指定了不同的度量单位。
  
## <a name="example"></a>示例

SIN(45 deg) 
  
返回 0.7071。
