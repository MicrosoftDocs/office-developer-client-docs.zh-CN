# <a name="int-function"></a>INT 函数

将数字向下舍入到下一个整数。
  
## <a name="syntax"></a>语法

INT (* **数字** *) 
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _number_ <br/> |必需  <br/> |**Number** <br/> |要向下舍入的数。  <br/> |
   
## <a name="example-1"></a>示例 1

INT (7.2)
  
返回 7。
  
## <a name="example-2"></a>示例 2

INT (-7.2)
  
返回 -8。
