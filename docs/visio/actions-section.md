# <a name="actions-section"></a>“Actions”部分

包含描述形状或页的快捷菜单或动作标记菜单上的菜单项的行。 在早期版本的 Microsoft Visio，动作标记称为智能标记。**TagName**单元格用于将操作关联的**动作标记**部分，其中定义动作标记行。 
  
> [!NOTE]
> 在以前版本的 Microsoft Visio 中，动作标记称为“智能标记”。 
  
## <a name="remarks"></a>注解

**TagName**单元格用于将操作关联的**动作标记**部分，其中定义动作标记行。
