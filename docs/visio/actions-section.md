# <a name="actions-section"></a>“Actions”内容

包含描述形状或页的快捷菜单或动作标记菜单上的菜单项的行。 在早期版本的 Microsoft Visio 中, 操作标记称为智能标记。**TagName**单元格用于将操作与**操作标记**部分中的行相关联, 其中定义了操作标记。 
  
> [!NOTE]
> 在以前版本的 Microsoft Visio 中，动作标记称为“智能标记”。 
  
## <a name="remarks"></a>注解

**TagName**单元格用于将操作与**操作标记**部分中的行相关联, 其中定义了操作标记。
