# <a name="acos-function"></a>ACOS 函数

返回数字的反正弦值，例如，其反正弦值是  *数字的角度*  。 
  
## <a name="syntax"></a>语法

ACOS ( *number*)  
  
## <a name="remarks"></a>备注

输入值的范围必须为 -1 = \< 数字 \< = 1，否则#NUM！ 错误。 结果角度的范围为 0 = 角度 = PI 弧 \< \< 度， (0 \< = angle = \< 180 度) 。
  
## <a name="example"></a>示例

ACOS (0)  
  
返回 90 度
