# <a name="acos-function"></a>ACOS 函数

返回反余弦值的数目，例如，余弦为*数字*的角度。 
  
## <a name="syntax"></a>语法

ACOS （*数量*） 
  
## <a name="remarks"></a>说明

输入的值必须位于范围介于-1 \<= 号码\<= 1 或 #NUM ！ 将返回错误。 生成的角度是在 0 \<= 角度\<= π (0 \<= 角度\<= 180 度)。
  
## <a name="example"></a>示例

ACOS(0) 
  
返回 90 度
