# <a name="edit-a-part-of-a-formula"></a>编辑公式的某一部分

1. 双击单元格，或者在选择单元格后按 F2。
    
2. 在单元格或者编辑栏中，运用下表中的技巧编辑公式。
    
|**目标**|**措施**|
|:-----|:-----|
| 放置插入点  <br/> | 在希望插入点出现的位置单击。  <br/> |
| 移动插入点  <br/> | 按左箭头键或右箭头键。  <br/> |
| 删除左侧的字符  <br/> | 按 Backspace。  <br/> |
| 删除右侧的字符  <br/> | 按 Delete。  <br/> |
| 选择文本  <br/> | 在文本上拖动鼠标指针。  <br/> |
| 将选择范围向左延伸  <br/> | 按 Shift+向左键。  <br/> |
| 将选择范围向右延伸  <br/> | 按 Shift+向右键。  <br/> |
| 选择字词  <br/> | 按 Ctrl+Shift+向右键，或者双击。  <br/> |
| 选择到行末  <br/> | 按 Shift+End。  <br/> |
| 选择到行首  <br/> | 按 Shift+Home。  <br/> |
| 替换选择的文本  <br/> | 键入新文本。  <br/> |
| 删除选择的文本  <br/> | 按 Delete。 或在"主页 **"选项卡上**，单击"剪切 **"。**  <br/> |
