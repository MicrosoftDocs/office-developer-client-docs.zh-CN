# <a name="iferror-function"></a>IFERROR 函数

返回主表达式的计算结果 (如果未计算出错误)。 否则返回备用表达式的计算结果。
  
## <a name="version-information"></a>版本信息

添加的版本： Visio 2010
 
  
## <a name="syntax"></a>语法

IFERROR (* **主表达式** *, * **替代表达式** *) 
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _主表达式_ <br/> |必需  <br/> |**String** <br/> |要计算的第一个表达式。  <br/> |
| _替代表达式_ <br/> |必需  <br/> |**String** <br/> |基本表达式计算错误时用于计算的备用表达式。  <br/> |
   
### <a name="return-value"></a>返回值

各不相同
