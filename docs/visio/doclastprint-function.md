# <a name="doclastprint-function"></a>DOCLASTPRINT 函数

以序列值的形式返回上次打印文档的日期和时间。
  
## <a name="syntax"></a>语法

DOCLASTPRINT ()
  
## <a name="remarks"></a>注解

使用 FORMAT 函数设置该值的格式。
