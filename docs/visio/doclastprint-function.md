# <a name="doclastprint-function"></a>DOCLASTPRINT 函数

以串行值返回上次打印文档的日期和时间。
  
## <a name="syntax"></a>语法

DOCLASTPRINT ()
  
## <a name="remarks"></a>备注

使用 FORMAT 函数设置该值的格式。
