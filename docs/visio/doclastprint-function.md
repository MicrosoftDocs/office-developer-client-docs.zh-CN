# <a name="doclastprint-function"></a>DOCLASTPRINT 函数

返回日期和时间上次打印文档，作为序列值。
  
## <a name="syntax"></a>语法

DOCLASTPRINT ()
  
## <a name="remarks"></a>注解

使用 FORMAT 函数设置该值的格式。
