# <a name="nurbsto-row-geometry-section"></a>NURBSTo 行（“Geometry”部分）

包含*x*和*y* -坐标、 最后一个节点到第二个位置，最后一个权重的位置，的第一个节点的位置的第一个权重和非均匀有理 B 样条 (NURBS) 公式的位置。 
  
NURBSTo 行包含以下单元格。
  
|**Cell**|**说明**|
|:-----|:-----|
|[X](x-cell-geometry-section.md) <br/> |*X* -NURBS 的最后一个控制点的坐标。  <br/> |
|[Y](y-cell-geometry-section.md) <br/> |*Y* -NURBS 的最后一个控制点的坐标。  <br/> |
|[A](a-cell-geometry-section.md) <br/> |NURBS 的倒数第二个节点。  <br/> |
|[B](b-cell-geometry-section.md) <br/> |NURBS 的最后一个权重。  <br/> |
|[C](c-cell-geometry-section.md) <br/> |NURBS 的第一个节点。  <br/> |
|[D](d-cell-geometry-section.md) <br/> |NURBS 的第一个权重。  <br/> |
|[E](e-cell-geometry-section.md) <br/> |NURBS 公式。  <br/> |
   
## <a name="remarks"></a>注解

NURBS 是表示曲线的通用数学方法。可用 **“自由绘制”** 工具创建 NURBS。
