# <a name="nurbsto-row-geometry-section"></a>NURBSTo 行（“Geometry”内容）

包含*x*坐标和*y*坐标、第二个节点的位置、最后一个权重的位置、第一个节点的位置、第一个节点的位置、第一个权重的位置以及非均匀有理 B 样条 (NURBS) 的公式。 
  
NURBSTo 行包含以下单元格。
  
|**Cell**|**Description**|
|:-----|:-----|
|[X](x-cell-geometry-section.md) <br/> |NURBS 的最后一个控制点的*x*坐标。  <br/> |
|[Y](y-cell-geometry-section.md) <br/> |NURBS 的最后一个控制点的*y*坐标。  <br/> |
|[A](a-cell-geometry-section.md) <br/> |NURBS 的倒数第二个节点。  <br/> |
|[黑白](b-cell-geometry-section.md) <br/> |NURBS 的最后一个权重。  <br/> |
|[lc](c-cell-geometry-section.md) <br/> |NURBS 的第一个节点。  <br/> |
|[D](d-cell-geometry-section.md) <br/> |NURBS 的第一个权重。  <br/> |
|[E](e-cell-geometry-section.md) <br/> |NURBS 公式。  <br/> |
   
## <a name="remarks"></a>注解

NURBS 是表示曲线的通用数学方法。可用 **“自由绘制”** 工具创建 NURBS。
