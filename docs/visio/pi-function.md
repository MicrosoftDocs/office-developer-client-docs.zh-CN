# <a name="pi-function"></a>PI 函数

返回数学常量*pi* ，即大约 3.1415926535898。 
  
## <a name="syntax"></a>语法

PI( )
  
### <a name="return-value"></a>返回值

Number
  
## <a name="example"></a>示例

(14 英寸 / 2) ^2 \* PI （) 
  
返回 153.9380 英寸，即直径为 14 英寸的圆的面积。
