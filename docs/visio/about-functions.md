# <a name="about-functions"></a>关于函数

函数用于执行单个明确任务。大多数函数都使用若干参数作为输入。虽然参数的类型和数目取决于函数，但所有函数都遵循同一通用语法：
  
 **函数 (** _argument1_， _argument2_，...  _argumentN_[， _argumentA_ |  _参数_ **])**
  
|**语法元素**|**说明**|
|:-----|:-----|
| ( )  <br/> | 如果函数没有参数，则必须紧跟一对空括号（）。  <br/> |
| ,  <br/> | 各参数用逗号分隔。  <br/> |
| ...  <br/> | 仅用作注释；不纳入函数。  <br/> |
| [ ]  <br/> | 可选参数。仅用作注释；不纳入函数。  <br/> |
| |  <br/> | 选择;您可以包括_argumentA_或_参数_。 仅用作注释；不纳入函数。  <br/> |
   
在公式中可使用的很多函数都类似于您在电子表格程序中见到的函数：数学函数，如 SUM 或 SQRT；三角函数，如 SIN 或 COS；逻辑函数，如 IF 或 NOT。还有很多函数是 Microsoft Office Visio 独有的，如 GUARD、GRAVITY 和 RUNADDON。
  
有关特定函数的详细信息，请参阅“ShapeSheet 参考”。
  
> [!NOTE]
>  某些功能出现在 Visio，生成的公式，但不是显示在**编辑公式**对话框中或因为它们仅供内部使用，不应在其他公式中使用此参考中所述。 下面是示例： ELLIPSE_THETA，_ELLIPSE_ECC，_UCON_C1，和 _SHAPEMIN。
