# <a name="deg-function"></a>DEG 函数

将角度的值从弧度转换为度。
  
## <a name="syntax"></a>语法

DEG (** *angle* ** )  
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _angle_ <br/> |必需  <br/> |**String** <br/> |角度的弧度值。  <br/> |
   
## <a name="example"></a>示例

DEG(PI( )/4) 
  
返回 45
