# <a name="add-change-or-delete-rows-in-the-shapesheet"></a>在 ShapeSheet 中添加、更改或删除行

您可以在 Microsoft Visio 中添加新行、更改现有行的类型或删除 ShapeSheet 中的行。
  
### <a name="to-add-a-new-row-to-a-section-in-the-shapesheet"></a>向 ShapeSheet 中的某一节添加新行

1. 在现有行中选择一个单元格。
    
2. 在 "**设计**" 选项卡上 " **ShapeSheet 工具**" 下的 "**行**" 组中, 单击 "**插入**" 以插入现有行的后面的行。 
    
    或者右键单击该单元格, 然后单击快捷菜单上的 "**插入行**" 或 "**在其后插入行**" (仅限 " [Geometry](geometry-section.md) " 部分)。 
    
### <a name="to-change-a-rows-type-in-the-shapesheet"></a>在 ShapeSheet 中更改行的类型

- 右键单击行标签或单元格, 然后单击快捷菜单上的 "**更改行类型**"。 
    
    > [!NOTE]
    > 如果 "**更改行类型**" 不可用, 则 ShapeSheet 部分可能不允许其他行类型。 
  
### <a name="to-delete-a-row-in-the-shapesheet"></a>删除 ShapeSheet 中的行

1. 在您要删除的行中选取一个单元格。
    
2. 在 " **ShapeSheet 工具**" 下 "**设计**" 选项卡的 "**行**" 组中, 单击 "**删除**"。 或者右键单击该单元格, 然后单击快捷菜单上的 "**删除行**"。 以下各行上移一节中的一行。
