# <a name="distancefromground-cell-3-d-rotation-properties"></a>DistanceFromGround 单元格（“3-D Rotation Properties”）

确定此对象以磅为单位时旋转三维从头引发的距离。
  
## <a name="remarks"></a>说明

要从另一个公式，由**N** **单元格**元素的属性的值或使用**CellsU**属性从某个程序按名称获取对**DistanceFromGround**单元格的引用，请使用： 
  
|||
|:-----|:-----|
|单元格名称：  <br/> |DistanceFromGround  <br/> |
   
若要从某个程序按索引获取对**DistanceFromGround**单元格的引用，请使用带下列参数的**CellsSRC**属性： 
  
|||
|:-----|:-----|
|内容索引：  <br/> |**visSectionObject** <br/> |
|行索引：  <br/> |**visRow3DRotationProperties** <br/> |
|单元格索引：  <br/> |**visDistanceFromGround** <br/> |
