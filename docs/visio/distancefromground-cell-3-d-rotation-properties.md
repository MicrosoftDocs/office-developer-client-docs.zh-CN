# <a name="distancefromground-cell-3-d-rotation-properties"></a>DistanceFromGround 单元格（“三维旋转属性”）

确定以磅为单位在三维中旋转时的对象的底边距离。
  
## <a name="remarks"></a>注解

若要从另一个公式按名称获取对**DistanceFromGround**单元格的引用、 **cell**元素的**N**属性值, 或从使用**CellsU**属性的某个程序获取对该单元格的引用, 请使用: 
  
|||
|:-----|:-----|
|单元格名称：  <br/> |DistanceFromGround  <br/> |
   
若要从某个程序按索引获取对**DistanceFromGround**单元格的引用, 请使用带下列参数的**CellsSRC**属性: 
  
|||
|:-----|:-----|
|内容索引：  <br/> |**visSectionObject** <br/> |
|行索引：  <br/> |**visRow3DRotationProperties** <br/> |
|单元格索引：  <br/> |**visDistanceFromGround** <br/> |
