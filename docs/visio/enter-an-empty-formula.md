# <a name="enter-an-empty-formula"></a>输入空公式

1. 选择要包含公式的单元格。
    
2. 按 Delete，然后按 Enter。
    
> [!NOTE]
> 尝试此操作在**草稿**单元格中插入局部公式之后。 由于没有从主控形状继承的公式，单元格将恢复为无公式。
