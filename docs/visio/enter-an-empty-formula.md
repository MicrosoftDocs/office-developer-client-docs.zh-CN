# <a name="enter-an-empty-formula"></a>输入空公式

1. 选择要包含公式的单元格。
    
2. 按 Delete，然后按 Enter。
    
> [!NOTE]
> 请在插入局部公式后在**暂存**单元格中尝试此项。 单元格将恢复为 "无公式", 因为没有要从主控形状继承的公式。
