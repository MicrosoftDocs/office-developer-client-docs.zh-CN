|||
|:-----|:-----|
|**元素类型** <br/> |[DynamicGridEnabled_Type](dynamicgridenabled_type-complextypevisio-xml.md) <br/> |

|**架构文件** <br/> |VisioSchema15.xsd  <br/> |
|**文档部件** <br/> |windows.xml、document.xml  <br/> |

```XML

</xs:element >
```
