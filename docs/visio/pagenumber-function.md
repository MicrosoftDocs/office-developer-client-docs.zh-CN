# <a name="pagenumber-function"></a>PAGENUMBER 函数

返回包含形状的页码。 
  
## <a name="syntax"></a>语法

PAGENUMBER () 
  
### <a name="return-value"></a>返回值

帐号
  
## <a name="remarks"></a>备注

如果当前页为背景页（未编号），或者发生了错误，则 PageNumber 函数将返回零。
