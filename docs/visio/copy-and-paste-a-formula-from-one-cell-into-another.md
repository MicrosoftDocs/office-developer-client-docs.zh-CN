# <a name="copy-and-paste-a-formula-from-one-cell-into-another"></a>将公式从一个单元格复制并粘贴到另一个单元格

1. 选择包含所需公式的单元格。
    
2. 按 CTRL + C。
    
    若要替换整个公式，选择要用于替换其公式的单元格。
    
    粘贴到现有公式的公式，双击单元格，然后单击插入点要粘贴公式。
    
3. 按 Ctrl+V
    
4. 按 Enter 接受公式。
