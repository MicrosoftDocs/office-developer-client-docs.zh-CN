# <a name="copy-and-paste-a-formula-from-one-cell-into-another"></a>将公式从一个单元格复制并粘贴到另一个单元格

1. 选择包含所需公式的单元格。
    
2. 按 CTRL + C。
    
    若要替换整个公式, 请选择要替换其公式的单元格。
    
    若要将公式粘贴到现有公式中, 请双击该单元格, 然后单击要在其中粘贴公式的插入点。
    
3. 按 Ctrl+V
    
4. 按 Enter 接受公式。
