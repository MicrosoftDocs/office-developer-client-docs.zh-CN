# <a name="lineto-row-geometry-section"></a>LineTo 行（“Geometry”内容）

包含直线段终顶点的*x*坐标和*y*坐标。 
  
LineTo 行包含以下单元格。
  
|**Cell**|**Description**|
|:-----|:-----|
|[X](x-cell-geometry-section.md) <br/> |直线段终顶点的*x*坐标。  <br/> |
|[Y](y-cell-geometry-section.md) <br/> |直线段终顶点的*y*坐标。  <br/> |
