# <a name="lineto-row-geometry-section"></a>LineTo 行（“Geometry”部分）

包含*x* -和*y* -直线段终顶点的坐标。 
  
LineTo 行包含以下单元格。
  
|**Cell**|**说明**|
|:-----|:-----|
|[X](x-cell-geometry-section.md) <br/> |*X* -直线段终顶点的坐标。  <br/> |
|[Y](y-cell-geometry-section.md) <br/> |*Y* -直线段终顶点的坐标。  <br/> |
