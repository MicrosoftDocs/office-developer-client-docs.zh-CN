# <a name="upper-function"></a>UPPER 函数

返回一个转换为大写的字符串。
  
## <a name="syntax"></a>语法

右上 (* **表达式** *) 
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _expression_ <br/> |必需  <br/> |**而异** <br/> | 可以是字符串、单元格引用或表达式；结果转换为字符串，然后再转换为大写形式。  <br/> |
   
## <a name="remarks"></a>注解

大小写转换取决于当前的用户设置，并且特定于区域设置。 
  
## <a name="example"></a>示例

UPPER("mIxEd CAse") 
  
返回“MIXED CASE”。
