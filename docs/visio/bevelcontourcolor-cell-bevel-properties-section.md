# <a name="bevelcontourcolor-cell-bevel-properties-section"></a>BevelContourColor Cell (Bevel Properties Section) 

确定斜面轮廓线的颜色（以 RGB 值表示或由活动主题确定）。
  
## <a name="remarks"></a>备注

若要从另一个公式 **、Cell** 元素 **的 N** 属性值或使用 **CellsU** 属性从某个程序按名称获取对 **BevelContourColor** 单元格的引用，请使用： 
  
|||
|:-----|:-----|
| 单元格名称：  <br/> | BevelContourColor  <br/> |
   
若要从程序按索引获取对 **BevelContourColor** 单元格的引用，请使用带下列参数的 **CellsSRC** 属性： 
  
|||
|:-----|:-----|
| 内容索引：  <br/> |**visSectionObject** <br/> |
| 行索引：  <br/> |**visRowBevelProperties** <br/> |
| 单元格索引：  <br/> |**vis BevelContourColor** <br/> |
