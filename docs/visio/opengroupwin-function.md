# <a name="opengroupwin-function"></a>OPENGROUPWIN 函数

在组窗口中打开组。 如果该对象不是一个组, 则此函数不执行任何操作。
  
## <a name="syntax"></a>语法

OPENGROUPWIN( )
  
## <a name="remarks"></a>说明

在 Visio 的早期版本中，此函数以 _OPENGROUPWIN 的形式出现。Visio 4.0 版和更高版本接受这两种样式中的任意一种。
