# <a name="reflectiondist-cell-additional-effect-properties-section"></a>ReflectionDist 单元格（“Additional Effect Properties”部分）

确定的距离反射与形状偏移中从 0.0 指向为因 100.0。 
  
## <a name="remarks"></a>说明

要从另一个公式，由**N** **单元格**元素的属性的值或使用**CellsU**属性从某个程序按名称获取对**ReflectionDist**单元格的引用，请使用： 
  
|||
|:-----|:-----|
| 单元格名称：  <br/> | ReflectionDist  <br/> |
   
若要从某个程序按索引获取对**ReflectionDist**单元格的引用，请使用带下列参数的**CellsSRC**属性： 
  
|||
|:-----|:-----|
| 内容索引：  <br/> |**visSectionObject** <br/> |
| 行索引：  <br/> |**visRowOtherEffectProperties** <br/> |
| 单元格索引：  <br/> |**visReflectionDist** <br/> |
