# <a name="reflectiondist-cell-additional-effect-properties-section"></a>ReflectionDist 单元格 ("其他效果属性" 部分)

确定反射相对于形状的偏移距离, 以磅为单位, 从0.0 到100.0。 
  
## <a name="remarks"></a>说明

若要从另一个公式按名称获取对**ReflectionDist**单元格的引用、 **cell**元素的**N**属性值, 或从使用**CellsU**属性的某个程序获取对该单元格的引用, 请使用: 
  
|||
|:-----|:-----|
| 单元格名称：  <br/> | ReflectionDist  <br/> |
   
若要从某个程序按索引获取对**ReflectionDist**单元格的引用, 请使用带下列参数的**CellsSRC**属性: 
  
|||
|:-----|:-----|
| 内容索引：  <br/> |**visSectionObject** <br/> |
| 行索引：  <br/> |**visRowOtherEffectProperties** <br/> |
| 单元格索引：  <br/> |**visReflectionDist** <br/> |
