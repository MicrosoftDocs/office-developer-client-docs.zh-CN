# <a name="sinh-function"></a>SINH 函数

返回一个角度的双曲正弦值。 
  
## <a name="syntax"></a>语法

SINH (* **角度** *) 
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _角度_ <br/> |必需  <br/> |**数字** <br/> |要获取的双曲正弦值的角度。  <br/> |
   
### <a name="return-value"></a>返回值

Number
  
## <a name="remarks"></a>注解

除非在_angle_中指定不同的度量单位，否则 SINH 函数使用弧度单位。
  
## <a name="example"></a>示例

SINH(45 deg) 
  
返回 0.8687。
