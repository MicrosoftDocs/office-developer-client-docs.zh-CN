# <a name="sinh-function"></a>SINH 函数

返回角度的双曲正弦值。 
  
## <a name="syntax"></a>语法

SINH (* * *angle* * *) 
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _angle_ <br/> |必需  <br/> |**数值** <br/> |要获取其双曲正弦值的角度。  <br/> |
   
### <a name="return-value"></a>返回值

数字
  
## <a name="remarks"></a>说明

SINH 函数使用弧度, 除非您以_角度_指定不同的度量单位。
  
## <a name="example"></a>示例

SINH(45 deg) 
  
返回 0.8687。
