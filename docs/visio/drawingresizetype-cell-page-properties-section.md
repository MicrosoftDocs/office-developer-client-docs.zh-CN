# <a name="drawingresizetype-cell-page-properties-section"></a>DrawingResizeType 单元格（“Page Properties”部分）

确定是否在绘图页自动调整大小以适合图表。 
  
## <a name="remarks"></a>说明

若要从另一个公式或使用 **CellsU** 属性从某个程序按名称获取对 DrawingResizeType 单元格的引用，请使用以下内容。 
  
|||
|:-----|:-----|
|单元格名称：  <br/> |DrawingResizeType  <br/> |
   
若要从某个程序按索引获取对 DrawingResizeType 单元格的引用，请使用带下列参数的 **CellsSRC** 属性： 
  
|||
|:-----|:-----|
|内容索引：  <br/> |**visSectionObject** <br/> |
|行索引：  <br/> |**visRowPage** <br/> |
|单元格索引：  <br/> |**visPageDrawResizeType** <br/> |
