# <a name="glowsize-cell-additional-effect-properties-section"></a>GlowSize 单元格 ("其他效果属性" 部分)

确定形状的外部发光的大小 (以磅为单位)。 
  
## <a name="remarks"></a>注解

若要从另一个公式按名称获取对**GlowSize**单元格的引用、 **cell**元素的**N**属性值, 或从使用**CellsU**属性的某个程序获取对该单元格的引用, 请使用: 
  
|||
|:-----|:-----|
| 单元格名称：  <br/> | GlowSize  <br/> |
   
若要从某个程序按索引获取对**GlowSize**单元格的引用, 请使用带下列参数的**CellsSRC**属性: 
  
|||
|:-----|:-----|
| 内容索引：  <br/> |**visSectionObject** <br/> |
| 行索引：  <br/> |**visRowOtherEffectProperties** <br/> |
| 单元格索引：  <br/> |**visGlowSize** <br/> |
