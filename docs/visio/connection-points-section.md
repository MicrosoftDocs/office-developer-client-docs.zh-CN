# <a name="connection-points-section"></a>“Connection Points”内容

包含与为形状定义的各连接点相对应的行和单元格。
  
## <a name="remarks"></a>注解

可以在 ShapeSheet 窗口中的 "**连接点**" 部分中定义连接点, 也可以通过单击 "**开始**" 选项卡上的 "**连接点**" 工具来定义连接点。 
  
如果您的形状没有 "**连接点**" 部分, 则可以在 ShapeSheet 窗口中插入 "**连接点**" 部分。 方法是：在 ShapeSheet 窗口中单击右键，单击 **“插入内容”**，然后单击 **“连接点”**。 若要向现有 "**连接点**" 部分添加连接点, 请右键单击某一行, 然后单击快捷菜单上的 "**插入行**"。 
  
您可以在此内容中更改连接点的行类型以显示可用于草稿计算的其他单元格。为此，请右键单击该行，然后在快捷菜单上单击 **“更改行类型”**。
