# <a name="change-the-name-and-view-the-id-of-a-shape"></a>更改名称并查看形状的 ID

1. 在绘图窗口中选择一个形状。
    
2. 在[“开发工具”](run-in-developer-mode-display-the-developer-tab.md)选项卡上的 **“形状设计”** 组中，单击 **“形状名”**。
    
3. 在**形状名**对话框中，输入一个名称。 
    
    在**形状名**对话框的顶部列出该形状的 ID。 
    
4. 单击“确定”****。
