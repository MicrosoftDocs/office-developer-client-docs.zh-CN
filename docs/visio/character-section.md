# <a name="character-section"></a>“Character”内容

显示形状文本的格式属性，如字体、颜色、文本样式、大小写、相对于基线的位置以及磅值。
  
## <a name="remarks"></a>备注

形状中的每个字符样式在 ShapeSheet 窗口中查看的 **"Character"** 内容中都有对应的行。 行的名称表示应用该字符样式的字符数。 
  
可以在"文本"对话框中设置文本格式 (单击"开始"选项卡) 上"字体"组中对话框启动器，通过应用文本样式或在 **"Character"** 内容单元格中创建条目。
