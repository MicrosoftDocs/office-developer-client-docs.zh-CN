# <a name="character-section"></a>“Character”部分

显示形状文本的格式属性，如字体、颜色、文本样式、大小写、相对于基线的位置以及磅值。
  
## <a name="remarks"></a>注解

每个形状中的字符样式**Character**内容中有对应的行，如在 ShapeSheet 窗口中查看。 行的名称表示字符样式应用于的字符的数。 
  
通过应用文本样式，或进行**字符**部分单元格中的一个条目，您可以在**文本**对话框 （单击**主页**选项卡的**字体**对话框启动器组），设置文本格式。
