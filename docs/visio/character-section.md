# <a name="character-section"></a>“Character”内容

显示形状文本的格式属性，如字体、颜色、文本样式、大小写、相对于基线的位置以及磅值。
  
## <a name="remarks"></a>注解

形状中的每个字符样式在 ShapeSheet 窗口中都有对应的 "**字符**" 部分中的行。 行的名称表示应用该字符样式的字符数。 
  
您可以通过应用文本样式或在 "**字符**" 部分单元格中进行输入, 在 "**文本**" 对话框 (单击 "**主页**" 选项卡上的 "**字体**" 组中的对话框启动器) 中设置文本格式。
