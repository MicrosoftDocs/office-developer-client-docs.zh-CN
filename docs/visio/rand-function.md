# <a name="rand-function"></a>RAND 函数

返回从 0.0 至 1.0 的随机浮点数。每次计算函数时（根据系统时钟，每分钟计算一次），函数都会返回一个不同数值。 
  
## <a name="syntax"></a>语法

RAND( )
  
### <a name="return-value"></a>返回值

Float
  
## <a name="remarks"></a>注解

通过将形状的属性设置为随机值，可使用此函数创建动画效果。
  
## <a name="example"></a>示例

RAND( ) 
  
返回一个小数，如 0.3503。
