# <a name="version-function"></a>VERSION 函数

返回与当前处于活动状态的 Microsoft Visio 应用程序的主版本匹配的整数。
  
## <a name="version-information"></a>版本信息

添加的版本： Visio 2010
 
  
## <a name="syntax"></a>语法

版本 ()
  
### <a name="return-value"></a>返回值

整数
  
## <a name="remarks"></a>说明

您可以使用 VERSION 函数编写 ShapeSheet 公式，这些公式根据计算时使用的 Visio 版本而执行不同的计算。
