# <a name="cosh-function"></a>COSH 函数

返回角度的双曲余弦值。
  
## <a name="syntax"></a>语法

COSH ( *angle*) 
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _angle_ <br/> |必需  <br/> |**数值** <br/> |要获取其双曲余弦值的角度。  <br/> |
   
## <a name="remarks"></a>说明

COSH 函数使用弧度, 除非您以*角度*指定不同的度量单位。 
  
## <a name="example"></a>示例

COSH(45 deg) 
  
返回 1.3246。
