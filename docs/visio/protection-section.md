# <a name="protection-section"></a>“Protection”部分

显示当前锁定设置与**保护**命令 （[开发人员](run-in-developer-mode-display-the-developer-tab.md)选项卡） 以及可以只能在 ShapeSheet 窗口中设置的少量附加锁定。 
  
## <a name="remarks"></a>注解

锁定有助于防止对形状的意外更改，但不能防止值在其他情况下被重置。它也不会防止在 ShapeSheet 窗口中进行更改。
  
若要防止更改公式，请使用**GUARD**函数。
