# <a name="tabs-section"></a>“Tabs”内容

包含控制制表位位置和对齐方式的形状或样式的单元格。
  
## <a name="remarks"></a>注解

您还可以使用 "**文本**" 对话框 (在 "**开始**" 选项卡上的 "**字体**" 组中单击 "对话框启动器") 中的 "**选项**卡" 选项卡或使用**文本标尺**设置这些值。 若要显示 **“文本标尺”**，请在文本编辑模式下右键单击文本，然后在快捷菜单上单击 **“文本标尺”**。 
  
您可以设置特定字符的制表位，方法是先选定该文本，然后设置制表位。当您设置特定字符的制表位时，会在 ShapeSheet 窗口中添加一个新行。该行的名称表示制表位包含的字符数。
