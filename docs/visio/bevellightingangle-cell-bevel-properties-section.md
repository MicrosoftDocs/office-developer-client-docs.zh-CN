# <a name="bevellightingangle-cell-bevel-properties-section"></a>BevelLightingAngle 单元格 （凹凸效果 Properties 内容）

确定闪电相对于中度凹凸效果的角度。 
  
## <a name="remarks"></a>备注

要从另一个公式，由**N** **单元格**元素的属性的值或使用**CellsU**属性从某个程序按名称获取对**BevelLightingAngle**单元格的引用，请使用： 
  
|||
|:-----|:-----|
| 单元格名称：  <br/> | BevelLightingAngle  <br/> |
   
若要从某个程序按索引获取对**BevelLightingAngle**单元格的引用，请使用带下列参数的**CellsSRC**属性： 
  
|||
|:-----|:-----|
| 内容索引：  <br/> |**visSectionObject** <br/> |
| 行索引：  <br/> |**visRowBevelProperties** <br/> |
| 单元格索引：  <br/> |**visBevelLightingAngle** <br/> |
