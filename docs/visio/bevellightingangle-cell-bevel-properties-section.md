# <a name="bevellightingangle-cell-bevel-properties-section"></a>BevelLightingAngle 单元格 ("棱台属性" 部分)

确定闪电相对于斜面的角度 (以度为单位)。 
  
## <a name="remarks"></a>注解

若要从另一个公式按名称获取对**BevelLightingAngle**单元格的引用、 **cell**元素的**N**属性值, 或从使用**CellsU**属性的某个程序获取对该单元格的引用, 请使用: 
  
|||
|:-----|:-----|
| 单元格名称：  <br/> | BevelLightingAngle  <br/> |
   
若要从某个程序按索引获取对**BevelLightingAngle**单元格的引用, 请使用带下列参数的**CellsSRC**属性: 
  
|||
|:-----|:-----|
| 内容索引：  <br/> |**visSectionObject** <br/> |
| 行索引：  <br/> |**visRowBevelProperties** <br/> |
| 单元格索引：  <br/> |**visBevelLightingAngle** <br/> |
