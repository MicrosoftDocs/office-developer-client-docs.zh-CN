# <a name="gradient-properties-section"></a>“Gradient Properties”部分

显示任何渐变填充或线条颜色效果应用于形状的属性。 本节还包括控制是否形状的渐变旋转与形状或形状是否假定应用的渐变时分组与其他形状的单元格。 
  
## <a name="remarks"></a>说明

您可以在**设置形状格式**对话框设置渐变设置 (**主** > **形状样式** > **填充** > **填充选项** > **填充** > **渐变填充**)。
