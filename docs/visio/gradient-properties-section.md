# <a name="gradient-properties-section"></a>"渐变属性" 部分

显示应用于形状的任何渐变填充或线条颜色效果的属性。 此部分还包括一些单元格, 这些单元格控制形状的渐变是与形状一起旋转还是在使用其他形状进行分组时形状是否采用应用的渐变。 
  
## <a name="remarks"></a>说明

您可以在 "设置**形状格式**" 对话框中设置渐变设置 (**主页** > **形状样式** > **填充** > **填充选项** > "**填充** > **渐变填充**")。
