# <a name="gradient-properties-section"></a>Gradient Properties Section

显示应用于形状的任何渐变填充或线条颜色效果的属性。 此部分还包括控制形状的渐变是否随形状旋转的单元格，或者该形状与其他形状组合时是否假定应用渐变的单元格。 
  
## <a name="remarks"></a>备注

You can set gradient settings in the **Format Shape** dialog box (**Home**  >  **Shape Styles**  >  **Fill**  >  **Options**  >  **Fill** Gradient  >  **fill) .**
