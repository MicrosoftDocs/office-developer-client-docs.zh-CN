# <a name="sum-function"></a>SUM 函数

返回数字的列表的总和。
  
## <a name="syntax"></a>语法

SUM (* **数字 1* * *，* **数字 2* * *，...，* * *[numberN]* * *) 
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _数字 1_ <br/> |必需  <br/> |**Numeric** <br/> |第一个数字。  <br/> |
| _数字 2_ <br/> |必需  <br/> |**Numeric** <br/> |第二个数字。  <br/> |
| _numberN_ <br/> |可选  <br/> |**Numeric** <br/> |第 n 个数字。  <br/> |
   
### <a name="return-value"></a>返回值

Numeric
  
## <a name="example"></a>示例

SUM(5,7,12)
  
返回 24。
