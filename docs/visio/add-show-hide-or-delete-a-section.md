# <a name="add-show-hide-or-delete-a-section"></a>添加、显示、隐藏或删除节

您可以在 ShapeSheet 中添加、显示、隐藏和删除节。
  
> [!NOTE]
> 如果要向 ShapeSheet 中添加自己的自定义节，请使用["用户定义的单元格](user-defined-cells-section.md)"或"Scratch"[](scratch-section.md)内容来定义 ShapeSheet 逻辑。 
  
### <a name="to-add-a-section"></a>添加分区

1. 在 ShapeSheet 窗口中单击以激活该窗口。
    
2. 执行以下操作之一：
    
  - 在 **"设计"选项卡上的"ShapeSheet 工具****"** 下的"**内容"** 组中，单击"插入 **"。**
    
  - 或者在 ShapeSheet 窗口中单击右键，然后单击快捷菜单上的 **“插入内容”**。 
    
3. 选取您要添加的内容的复选框，然后单击 **“确定”**。
    
    > [!NOTE]
    >  如果您在 ShapeSheet 窗口中看不到新内容，应确保该内容已设置为显示。在 **“ShapeSheet 工具”** 选项卡上的 **“视图”** 组中，单击 **“内容”**，然后选中您要显示的内容的复选框。 
  
### <a name="to-show-or-hide-sections"></a>显示或隐藏分区

- 在要删除的内容中选择任一单元格，然后在 **“ShapeSheet 工具”** 选项卡上的 **“内容”** 组中单击 **“删除”**。
    
- 也可以右键单击单元格，然后单击快捷菜单上的 **"删除节** "。 
    
    > [!NOTE]
    >  某些内容不能删除。例如，由于 **“Shape Transform”** 内容指定一个形状的位置，因此您不能删除它。如果某一内容不能删除，**“删除内容”** 将不可用。 
  
### <a name="to-collapse-or-expand-a-section"></a>折叠或展开节

1. 在 ShapeSheet 窗口中单击以激活该窗口。
    
2. 指向节标题。
    
    如果指针变为减号 ![如果指针变为减号，折叠部分](media/IC_SSMinus_ZA07645855.gif)，可以折叠分区。
    
    如果指针变为加号 ![如果指针变为加号，请展开 部分](media/IC_SSPlus_ZA07645856.gif)，可以展开 部分。
    
3. 单击部分标题。
    
    显示将更改为隐藏或显示节的单元格。
    
### <a name="to-delete-a-section"></a>删除分区

1. 在 ShapeSheet 窗口中单击以激活该窗口。
    
2. 在 **"设计"选项卡上的"ShapeSheet 工具****"** 下的"**视图"** 部分，单击"内容 **"。** 或者在 ShapeSheet 窗口中单击右键，然后单击 **“查看内容”**。
    
3. 选中您要显示的内容，或者清除您要隐藏的内容。
    
4. 单击“确定”。
