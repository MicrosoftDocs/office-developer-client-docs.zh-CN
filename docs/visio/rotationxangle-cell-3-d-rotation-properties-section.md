# <a name="rotationxangle-cell-3-d-rotation-properties-section"></a>RotationXAngle 单元格 ("三维旋转属性" 部分)

确定沿 X 轴旋转的角度 (以度为单位) (0.0-359.9)。 
  
## <a name="remarks"></a>注解

若要从另一个公式按名称获取对**RotationXAngle**单元格的引用、 **cell**元素的**N**属性值, 或从使用**CellsU**属性的某个程序获取对该单元格的引用, 请使用: 
  
|||
|:-----|:-----|
|单元格名称：  <br/> |RotationXAngle  <br/> |
   
若要从某个程序按索引获取对**RotationXAngle**单元格的引用, 请使用带下列参数的**CellsSRC**属性: 
  
|||
|:-----|:-----|
|内容索引：  <br/> |**visSectionObject** <br/> |
|行索引：  <br/> |**visRow3DRotationProperties** <br/> |
|单元格索引：  <br/> |**visRotationXAngle** <br/> |
