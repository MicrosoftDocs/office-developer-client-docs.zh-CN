# <a name="rotationxangle-cell-3-d-rotation-properties-section"></a>RotationXAngle Cell (3-D Rotation Properties Section) 

确定沿 X 轴旋转的角度，以 0.0 - 359.9 (度) 。 
  
## <a name="remarks"></a>备注

若要从另一个公式 **、Cell** 元素 **的 N** 属性值或使用 **CellsU** 属性从某个程序按名称获取对 **RotationXAngle** 单元格的引用，请使用： 
  
|||
|:-----|:-----|
|单元格名称：  <br/> |RotationXAngle  <br/> |
   
若要从程序按索引获取对 **RotationXAngle** 单元格的引用，请使用带下列参数的 **CellsSRC** 属性： 
  
|||
|:-----|:-----|
|内容索引：  <br/> |**visSectionObject** <br/> |
|行索引：  <br/> |**visRow3DRotationProperties** <br/> |
|单元格索引：  <br/> |**visRotationXAngle** <br/> |
