# <a name="miscellaneous-section"></a>“Miscellaneous”内容

包含形状和组合的各种属性，如控件选择突出显示和可见性。
  
## <a name="remarks"></a>注解

可以在 ShapeSheet 窗口中或绘图窗口的 **“行为”** 对话框（单击[“开发工具”](run-in-developer-mode-display-the-developer-tab.md)选项卡上 **“形状设计”** 组中的 **“行为”**）中的 **“行为”** 选项卡上设置此内容中许多单元格的值。可以使用“Group Properties”内容中的单元格来设置其他组合属性。 
  
在 Visio 版本5.0 产品中, 此部分包括**ObjInteract**和**ObjBehavior**单元格。 **ObjInteract**单元格中的行为现在映射到 "Shape Layout" 部分中的以下单元格: **ShapeFixedCode**、 **ShapePermeableX**、 **ShapePermeableY**和**ShapePermeablePlace**。 ObjBehavior 单元格中的行为现在映射到**ConFixedCode**和**ShapeRouteStyle**单元格。 
  
在 Visio 版本5.0 日文产品中, 此部分包含一个**VerticalText**单元格。 在 Visio 版本2000产品及更高版本中, 该单元格的值存储在 "**文本块格式**" 部分的 " **TextDirection** " 单元格中。
