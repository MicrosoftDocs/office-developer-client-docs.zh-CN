# <a name="miscellaneous-section"></a>“Miscellaneous”部分

包含形状和组合的各种属性，如控件选择突出显示和可见性。
  
## <a name="remarks"></a>注解

可以在 ShapeSheet 窗口中或绘图窗口的 **“行为”** 对话框（单击[“开发工具”](run-in-developer-mode-display-the-developer-tab.md)选项卡上 **“形状设计”** 组中的 **“行为”**）中的 **“行为”** 选项卡上设置此内容中许多单元格的值。可以使用“Group Properties”内容中的单元格来设置其他组合属性。 
  
在 Visio 5.0 版产品中，此部分包含**ObjInteract**和**ObjBehavior**单元格。 Shape Layout 内容中的以下单元格现在映射中的**ObjInteract**单元格的行为： **ShapeFixedCode**、 **ShapePermeableX**、 **ShapePermeableY**和**ShapePermeablePlace**。 现在，对**ConFixedCode**和**ShapeRouteStyle**单元格映射中的 ObjBehavior 单元格的行为。 
  
Visio 5.0 日语版产品，在此部分包含的**内容的 VerticalText**单元格。 Visio 版本 2000年产品中及更高版本，此单元格的值存储在**TextDirection**单元格**Text Block Format**内容中。
