# <a name="line-format-section"></a>“Line Format”内容

包含那些控制某形状的线条属性（如图案、粗细和颜色）的单元格。它们确定线端形状是否已进行了格式设置（例如，带有箭头）、线端大小格式、应用于线条的外圆半径以及线端样式（圆形或正方形）。
  
## <a name="remarks"></a>备注

您可以通过使用"开始"选项卡上的"设置形状格式"窗格 (、在"形状样式"组中单击"线条"、单击"线条选项 **) "、** 应用线条样式或在"线条格式"单元格中输入公式来设置线条格式。
