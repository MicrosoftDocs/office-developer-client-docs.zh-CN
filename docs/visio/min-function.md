# <a name="min-function"></a>MIN 函数

返回列表中的最小值。最小表示最接近负无穷大。
  
## <a name="syntax"></a>语法

最小值 (* **数字 1* * *，* **数字 2* * *，...，* * *numberN* * *) 
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _数字 1_ <br/> |必需  <br/> |**因情况而异** <br/> |列表中的第一个数字。  <br/> |
| _数字 2_ <br/> |可选  <br/> |**因情况而异** <br/> | 列表中的第二个数字。  <br/> |
| _numberN_ <br/> |可选  <br/> |**因情况而异** <br/> |列表中的第 n 个数字。  <br/> |
   
### <a name="return-value"></a>返回值

因情况而异
  
## <a name="example"></a>示例

MIN(13 in,1 ft, 20 cm) 
  
返回 20 厘米。
