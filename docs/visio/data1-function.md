# <a name="data1-function"></a>DATA1 函数

返回"开发工具"选项卡上"形状名称"对话框中"数据 **1"** 框中的文本 (，单击"形状名称") 字符串。 
  
## <a name="syntax"></a>语法

DATA1 () 
  
## <a name="remarks"></a>备注

如果该框为空，则函数返回空字符串。
