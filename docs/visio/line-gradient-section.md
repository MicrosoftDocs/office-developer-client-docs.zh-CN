# <a name="line-gradient-section"></a>“Line Gradient”部分

显示应用于形状的线条渐变颜色值。 
  
## <a name="remarks"></a>说明

部分中的各[渐变停止行 （Line 渐变内容）](gradient-stop-row-line-gradient-section.md)行代表行中的渐变光圈。 此部分可以为空。 
  
仅包含一节中的前 10 行使用渐变设置。 10 之外的所有**渐变停止**行将被忽略。 
  
在**设置形状格式**窗格的**行**部分，可以在渐变属性设置行 （**主页**选项卡，在**形状样式**组中，单击**行**中，，然后单击**行选项**）。 
  
您可以通过启用或禁用渐变线条颜色形状更改[LineGradientEnabled 单元格 （渐变 Properties 内容）](linegradientenabled-cell-gradient-properties-section.md)单元格的值。
