# <a name="paragraph-section"></a>“Paragraph”内容

显示形状文本的段落格式属性，如段落的缩进、行间距、项目符号和水平对齐方式。
  
## <a name="remarks"></a>说明

可以通过应用文本样式, 或在 "**文本**" 对话框 (在 "**开始**" 选项卡上, 单击 "**字体**" 组中的对话框启动器) 中通过在 "**段落**" 内容单元格中输入值来设置段落格式。 
  
**段落**部分不包含选项卡设置的单元格。 若要为形状文本设置制表符，请使用 **“文本”** 对话框。
