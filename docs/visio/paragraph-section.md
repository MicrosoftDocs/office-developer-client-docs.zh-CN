# <a name="paragraph-section"></a>“Paragraph”部分

显示形状文本的段落格式属性，如段落的缩进、行间距、项目符号和水平对齐方式。
  
## <a name="remarks"></a>注解

可以通过在**段落**部分单元格进行输入，通过应用文本样式，或在**文本**对话框中输入值来设置段落格式 （在**主页**选项卡上，单击**字体**组中的对话框启动器）。 
  
**Paragraph**内容中不包含选项卡设置单元格。 若要设置形状的文本的选项卡，请使用**文本**对话框。
