# <a name="paragraph-section"></a>“Paragraph”内容

显示形状文本的段落格式属性，如段落的缩进、行间距、项目符号和水平对齐方式。
  
## <a name="remarks"></a>备注

通过 **应用文本** 样式，或在"开始"选项卡上的"文本"对话框 (中输入值，单击"字体"组中) 中的对话框启动器，可以设置段落格式。  
  
**"Paragraph"** 内容不包括选项卡设置的单元格。 若要为形状文本设置制表符，请使用 **“文本”** 对话框。
