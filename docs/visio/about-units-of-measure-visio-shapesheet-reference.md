<span data-ttu-id="25966-p115">当 Visio 分析和存储数字单位对时，它可以使用显式单位或隐式单位。用显式单位表示的数字总是按最初输入的度量单位显示。用隐式单位表示的数字总是转换为以适合于单元格的绘图、页面或角度单位表示的等价值。</span><span class="sxs-lookup"><span data-stu-id="25966-p115">When Visio parses and stores a number-unit pair, it can use explicit units or implicit units.
