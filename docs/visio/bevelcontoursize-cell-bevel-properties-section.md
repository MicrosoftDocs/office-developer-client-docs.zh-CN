# <a name="bevelcontoursize-cell-bevel-properties-section"></a>BevelContourSize 单元格（“Bevel Properties”部分）

确定凹凸效果的分布，以磅为单位的大小。 
  
## <a name="remarks"></a>说明

要从另一个公式，由**N** **单元格**元素的属性的值或使用**CellsU**属性从某个程序按名称获取对**BevelContourSize**单元格的引用，请使用： 
  
|||
|:-----|:-----|
| 单元格名称：  <br/> | BevelContourSize  <br/> |
   
若要从某个程序按索引获取对**BevelContourSize**单元格的引用，请使用带下列参数的**CellsSRC**属性： 
  
|||
|:-----|:-----|
| 内容索引：  <br/> |**visSectionObject** <br/> |
| 行索引：  <br/> |**visRowBevelProperties** <br/> |
| 单元格索引：  <br/> |**visBevelContourSize** <br/> |
