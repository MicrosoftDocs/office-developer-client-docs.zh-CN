# <a name="blob-function"></a>BLOB 函数

对 base64 字符串进行解码，并将其存储为二进制数据。 
  
## <a name="syntax"></a>语法

BLOB (* * *bintext* * *) 
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _bintext_ <br/> |必需  <br/> |**String** <br/> | 一个以 base64 编码的字符串。  <br/> |
