# <a name="docmd-function"></a>DOCMD 函数

执行已确定的命令。
  
## <a name="syntax"></a>语法

 **DOCMD**( _commandID_)
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _commandID_ <br/> |必需  <br/> |**Number** <br/> | 要执行的命令。  <br/> |
   
## <a name="remarks"></a>注解

有关 DOCMD 函数支持的命令的列表, 请参阅 Microsoft Visio 2013 Automation Reference 中的 "DOCMD/DOCMD 命令" 主题。 
  
## <a name="example"></a>示例

 `DOCMD (1312)`
  
导致在用户界面中显示 **“形状数据”** 对话框。
