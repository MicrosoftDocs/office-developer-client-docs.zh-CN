# <a name="docmd-function"></a>DOCMD 函数

执行标识的命令。
  
## <a name="syntax"></a>语法

 **DOCMD**( _commandID_)
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _commandID_ <br/> |必需  <br/> |**编号** <br/> | 要执行的命令。  <br/> |
   
## <a name="remarks"></a>注解

DOCMD 函数所支持的命令的列表，请参阅主题的 Microsoft Visio 2013 Automation 参考中的"DoCmd/DOCMD 命令"。 
  
## <a name="example"></a>示例

 `DOCMD (1312)`
  
导致在用户界面中显示 **“形状数据”** 对话框。
