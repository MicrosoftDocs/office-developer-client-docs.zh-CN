# <a name="docmd-function"></a>DOCMD 函数

执行标识的命令。
  
## <a name="syntax"></a>语法

 **DOCMD** ( _commandID_) 
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _commandID_ <br/> |必需  <br/> |**Number** <br/> | 要执行的命令。  <br/> |
   
## <a name="remarks"></a>备注

有关 DOCMD 函数支持的命令列表，请参阅 Microsoft Visio 2013 Automation Reference中的主题"DoCmd/DOCMD 命令"。 
  
## <a name="example"></a>示例

 `DOCMD (1312)`
  
导致在用户界面中显示 **“形状数据”** 对话框。
