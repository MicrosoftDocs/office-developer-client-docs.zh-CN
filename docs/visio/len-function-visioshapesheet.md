# <a name="len-function-visioshapesheet"></a>LEN 函数 (VisioShapeSheet)

文本字符串中返回字符的数。
  
## <a name="syntax"></a>语法

LEN (* **文本** *) 
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _text_ <br/> |必需  <br/> |**字符串** <br/> |要查找其长度的字符串。  <br/> |
   
### <a name="return-value"></a>返回值

Numeric
  
## <a name="example"></a>示例

LEN ("Seattle, WA") 
  
返回 11。
