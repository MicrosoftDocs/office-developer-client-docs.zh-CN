# <a name="len-function-visioshapesheet"></a>LEN 函数 (VisioShapeSheet)

返回文本字符串中的字符数。
  
## <a name="syntax"></a>语法

LEN (* * *text* * *) 
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _text_ <br/> |必需  <br/> |**String** <br/> |要查找其长度的字符串。  <br/> |
   
### <a name="return-value"></a>返回值

数值
  
## <a name="example"></a>示例

LEN ("Seattle, WA") 
  
返回 11。
