# <a name="perspective-cell-3-d-rotation-properties-section"></a>Perspective 单元格（“3-D Rotation Properties”部分）

确定透视角度旋转的角度来看中度 (0 到 359.9)
  
## <a name="remarks"></a>说明

要从另一个公式，由**N** **单元格**元素的属性的值或使用**CellsU**属性从某个程序按名称获取对**角度**单元格的引用，请使用： 
  
|||
|:-----|:-----|
|单元格名称：  <br/> |透视  <br/> |
   
若要从某个程序按索引获取对**角度**单元格的引用，请使用带下列参数的**CellsSRC**属性： 
  
|||
|:-----|:-----|
|内容索引：  <br/> |**visSectionObject** <br/> |
|行索引：  <br/> |**visRow3DRotationProperties** <br/> |
|单元格索引：  <br/> |**visPerspective** <br/> |
