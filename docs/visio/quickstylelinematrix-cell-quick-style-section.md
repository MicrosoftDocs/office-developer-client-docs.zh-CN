# <a name="quickstylelinematrix-cell-quick-style-section"></a>QuickStyleLineMatrix 单元格 ("快速样式" 部分)

确定形状继承的快速样式线样式, 如0-6 中的整数。 
  
## <a name="remarks"></a>注解

若要从另一个公式按名称获取对**QuickStyleLineMatrix**单元格的引用、 **cell**元素的**N**属性值, 或从使用**CellsU**属性的某个程序获取对该单元格的引用, 请使用: 
  
|||
|:-----|:-----|
| 单元格名称：  <br/> | QuickStyleLineMatrix  <br/> |
   
若要从某个程序按索引获取对**QuickStyleLineMatrix**单元格的引用, 请使用带下列参数的**CellsSRC**属性: 
  
|||
|:-----|:-----|
| 内容索引：  <br/> |**visSectionObject** <br/> |
| 行索引：  <br/> |**visRowQuickStyleProperties** <br/> |
| 单元格索引：  <br/> |**visQuickStyleLineMatrix** <br/> |
