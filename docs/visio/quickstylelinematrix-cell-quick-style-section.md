# <a name="quickstylelinematrix-cell-quick-style-section"></a>QuickStyleLineMatrix Cell (Quick Style Section) 

确定形状继承的快速样式样式，作为 0-6 的整数。 
  
## <a name="remarks"></a>备注

若要从另一个公式 **、Cell** 元素 **的 N** 属性值或使用 **CellsU** 属性从某个程序按名称获取对 **QuickStyleLineMatrix** 单元格的引用，请使用： 
  
|||
|:-----|:-----|
| 单元格名称：  <br/> | QuickStyleLineMatrix  <br/> |
   
若要从程序按索引获取对 **QuickStyleLineMatrix** 单元格的引用，请使用带下列参数的 **CellsSRC** 属性： 
  
|||
|:-----|:-----|
| 内容索引：  <br/> |**visSectionObject** <br/> |
| 行索引：  <br/> |**visRowQuickStyleProperties** <br/> |
| 单元格索引：  <br/> |**visQuickStyleLineMatrix** <br/> |
