# <a name="action-tag-section"></a>"Action Tag"内容

包含与为形状或页定义的每个动作标记相对应的行。
  
> [!NOTE]
> 在以前版本的 Microsoft Visio 中，动作标记称为“智能标记”。 
  
## <a name="remarks"></a>备注

" **动作标记** "部分控制动作标记在绘图中的外观和位置。 使用 **TagName** 单元格，您可以将每个动作标记与"Actions"内容中的一行或多行关联，其中定义了动作标记菜单上出现的菜单项。 
  
 一个形状或页面可以包含多个动作标记。 
  
如果某个动作标记没有匹配动作，则该动作标记按钮仍会按照定义方式显示和消失，但其菜单为空白。
  
只能在 ShapeSheet 窗口中插入"**操作** 标记" (ShapeSheet 窗口中右键单击，单击"插入内容"，然后选中"操作标记"复选框) 。  若要向现有的"Action **Tag"** 内容添加动作标记，请右键单击一行，然后单击快捷菜单上的"插入行"。
