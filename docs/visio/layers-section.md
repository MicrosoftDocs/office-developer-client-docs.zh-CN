# <a name="layers-section"></a>“Layers”部分

显示为某一页定义的所有图层和为各图层设置的属性。 
  
## <a name="remarks"></a>注解

此内容为页属性，只能在页的 ShapeSheet 窗口中查看。其单元格对应于 **“图层属性”** 对话框（在 **“开始”** 选项卡上的 **“编辑”** 组中，单击 **“图层”**，然后单击 **“图层属性”**）中的选项。
