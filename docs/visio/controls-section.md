# <a name="controls-section"></a>“Controls”内容

包含与为形状定义的每个控制手柄相对应的行和单元格。
  
## <a name="remarks"></a>备注

可以在 ShapeSheet 窗口的 **"控件"** 部分定义控制手柄。 如果形状没有 **"Controls"** 内容，可以在 ShapeSheet 窗口中插入一个" Controls"内容。 方法是：在 ShapeSheet 窗口中单击右键，单击 **“插入内容”**，然后选中 **“Controls”** 复选框。 若要向现有 Controls 内容添加 **控件** 句柄，请右键单击一行，然后单击 **快捷菜单上的** "插入行"。
