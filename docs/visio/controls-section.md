# <a name="controls-section"></a>“Controls”内容

包含与为形状定义的每个控制手柄相对应的行和单元格。
  
## <a name="remarks"></a>注解

您可以在 ShapeSheet 窗口中的 "**控件**" 部分定义控制手柄。 如果形状没有 "**控件**" 部分, 则可以在 ShapeSheet 窗口中插入一个。 方法是：在 ShapeSheet 窗口中单击右键，单击 **“插入内容”**，然后选中 **“Controls”** 复选框。 若要向现有的 "**控件**" 部分添加控制手柄, 请右键单击某一行, 然后单击快捷菜单上的 "**插入行**"。
