# <a name="controls-section"></a>“Controls”内容

包含与为形状定义的每个控制手柄相对应的行和单元格。
  
## <a name="remarks"></a>注解

您可以定义在 ShapeSheet 窗口中的**Controls**内容中的控制手柄。 如果形状没有**Controls**内容，则可以插入一个在 ShapeSheet 窗口中。 在 ShapeSheet 窗口中右键单击，单击**插入节**，，然后选择**控件**复选框。 若要添加到现有的**Controls**内容的控制手柄，右击某一行，然后单击快捷菜单上的**插入行**。
