# <a name="doclastsave-function"></a>DOCLASTSAVE 函数

返回上次保存文档时的日期和时间, 以序列值的形式返回。
  
## <a name="syntax"></a>语法

DOCLASTSAVE ()
  
## <a name="remarks"></a>注解

使用 FORMAT 函数设置该值的格式。
