# <a name="doclastsave-function"></a>DOCLASTSAVE 函数

返回日期和时间上次保存文档，作为序列值。
  
## <a name="syntax"></a>语法

DOCLASTSAVE ()
  
## <a name="remarks"></a>注解

使用 FORMAT 函数设置该值的格式。
