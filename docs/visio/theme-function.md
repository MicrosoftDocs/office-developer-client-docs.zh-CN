# <a name="theme-function"></a>THEME 函数

获取当前主题的格式设置。
  
## <a name="syntax"></a>语法

主题 ()
  
## <a name="example"></a>示例

```vb
Shape.FillForegnd = THEME("FillColor")
```

向形状分配当前主题的填充颜色。
