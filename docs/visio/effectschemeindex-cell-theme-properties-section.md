# <a name="effectschemeindex-cell-theme-properties-section"></a>EffectSchemeIndex 单元格（“Theme Properties”部分）

确定应用于一个形状，作为整数的主题的效果方案。
  
## <a name="remarks"></a>说明

要从另一个公式，由**N** **单元格**元素的属性的值或使用**CellsU**属性从某个程序按名称获取对**EffectSchemeIndex**单元格的引用，请使用： 
  
|||
|:-----|:-----|
| 单元格名称：  <br/> | EffectSchemeIndex  <br/> |
   
若要从某个程序按索引获取对**EffectSchemeIndex**单元格的引用，请使用带下列参数的**CellsSRC**属性： 
  
|||
|:-----|:-----|
| 内容索引：  <br/> |**visSectionObject** <br/> |
| 行索引：  <br/> |**visRowThemeProperties** <br/> |
| 单元格索引：  <br/> |* * visEffectSchemeIndex * * <br/> |
