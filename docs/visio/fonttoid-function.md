# <a name="fonttoid-function"></a>FONTTOID 函数

返回指定字体的标识符 (ID)。
  
> [!NOTE]
> 建议使用 FONT [函数，](font-function.md) 该函数类似于 **FONTTOID** 函数。 在 Microsoft Visio 2013 中，**引入了 FONT** 函数来替换 **FONTTOID**。 
  
## <a name="syntax"></a>语法

FONTTOID (** *font_name* ** )  
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _font_name_ <br/> |必需  <br/> |**String** <br/> |字体的名称。  <br/> |
