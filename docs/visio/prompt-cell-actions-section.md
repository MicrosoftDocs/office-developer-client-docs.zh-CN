# <a name="prompt-cell-actions-section"></a>Prompt 单元格（“Actions”内容）

从 Visio 2002 开始，不再使用这个单元格。
  
## <a name="remarks"></a>说明

在 Visio 的早期版本中，在选择某一快捷命令后，该单元格指定的相应的说明性提示便出现在状态栏中。
