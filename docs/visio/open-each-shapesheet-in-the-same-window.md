# <a name="open-each-shapesheet-in-the-same-window"></a>在同一窗口中打开每个 ShapeSheet

1. 单击**文件**选项卡，单击**选项**，然后单击**高级**。
    
2. 在 **“常规”** 下，选择 **“在同一窗口中打开每一 ShapeSheet”**。
    
3. 单击“确定”****。
