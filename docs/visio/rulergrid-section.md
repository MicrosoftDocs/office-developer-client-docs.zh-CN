# <a name="ruler-amp-grid-section"></a>标尺&amp;网格部分

显示页的标尺和网格的当前设置。
  
## <a name="remarks"></a>注解

此内容为页属性，只能在页的 ShapeSheet 窗口中查看。 它所包含的单元格对应于 " ** &amp;标尺网格**" 对话框 (在 "**视图**" 选项卡上的 "**显示/隐藏**" 组中, 单击对话框启动器) 中的设置。
