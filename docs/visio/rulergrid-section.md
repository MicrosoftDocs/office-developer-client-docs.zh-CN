# <a name="ruler-amp-grid-section"></a>"标 &amp; 尺网格"部分

显示页的标尺和网格的当前设置。
  
## <a name="remarks"></a>备注

此内容为页属性，只能在页的 ShapeSheet 窗口中查看。 它包含的单元格对应于"视图"选项卡上"标尺网格" (中的设置，在"显示 **/** 隐藏"组中，单击对话框启动器) 。 **&amp;**
