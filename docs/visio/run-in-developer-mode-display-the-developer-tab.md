# <a name="run-in-developer-mode-display-the-developer-tab"></a>以开发人员模式运行（显示“开发工具”选项卡）

- 单击 "**文件**" 选项卡, 然后单击 "**选项**"。
    
- 在 " **Visio 选项**" 对话框中, 单击 "**高级**", 然后在 "**常规**" 下, 选择 "**以开发人员模式运行**"。
    
然后，您可以选择一个对象，并单击 **“显示 ShapeSheet”**，以便更快地访问 ShapeSheet 窗口。
