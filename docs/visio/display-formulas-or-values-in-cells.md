# <a name="display-formulas-or-values-in-cells"></a>显示单元格中的公式或值

1. 在 ShapeSheet 窗口中单击以激活该窗口。
    
2. 在 **“ShapeSheet 工具”** 选项卡上的 **“视图”** 组中，单击 **“公式”** 以查看公式，或者单击 **“值”** 以查看值。 
    
    > [!NOTE]
    > 您也可以在 ShapeSheet 窗口中单击右键，然后单击快捷菜单上的 **“公式”** 或 **“值”**。
