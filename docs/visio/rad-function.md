# <a name="rad-function"></a>RAD 函数

将角度值从度转换为弧度。
  
## <a name="syntax"></a>语法

RAD (* * *angle* * *) 
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _angle_ <br/> |必需  <br/> |**Number** <br/> |要转换的角度。  <br/> |
   
## <a name="example"></a>示例

RAD (45) 
  
返回 0.7854。
