# <a name="rad-function"></a>RAD 函数

将从度的角度值转换为弧度。
  
## <a name="syntax"></a>语法

RAD (* **角度** *) 
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _角度_ <br/> |必需  <br/> |**编号** <br/> |要转换的角度。  <br/> |
   
## <a name="example"></a>示例

RAD(45) 
  
返回 0.7854。
