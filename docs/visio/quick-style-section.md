# <a name="quick-style-section"></a>“Quick Style”部分

显示当前快速样式的格式应用于该形状，包括线条颜色和样式、 填充颜色和样式、 阴影颜色、 效果、 字体和样式类型的值。 
  
## <a name="remarks"></a>说明

您可以选择形状，然后选择新的快速样式的快速样式 （在**主页**选项卡的**形状样式**组中，单击**详细**，，，然后从库中选择另一个快速样式）。
