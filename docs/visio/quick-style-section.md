# <a name="quick-style-section"></a>"快速样式" 部分

显示应用于形状的当前快速样式格式设置值, 包括线条颜色和样式、填充颜色和样式、阴影颜色、效果、字体和样式类型。 
  
## <a name="remarks"></a>注解

您可以快速样式选择形状, 然后选择新的快速样式 (在 "**开始**" 选项卡上的 "**形状样式**" 组中, 单击 "**更多**", 然后从库中选择另一个快速样式)。
