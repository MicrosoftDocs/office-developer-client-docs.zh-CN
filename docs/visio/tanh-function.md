# <a name="tanh-function"></a>TANH 函数

返回一个角度的双曲正切值。
    
 
  
## <a name="syntax"></a>语法

TANH (* **角度** *) 
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _角度_ <br/> |必需  <br/> |**Numeric** <br/> |要获取双曲正切值的角度。  <br/> |
   
### <a name="return-value"></a>返回值

Numeric
  
## <a name="remarks"></a>注解

除非在*angle*中指定不同的度量单位，否则 TANH 函数使用弧度单位。 
  
## <a name="example"></a>示例

TANH(-45) 
  
返回 -0.6558。
