# <a name="lockthemeeffects-cell-protection-section"></a>LockThemeEffects 单元格（“Protection”部分）

防止对形状应用主题效果。 
  
与**保护**对话框中的**阻止应用主题效果**复选框设置相对应。 
  
若要从另一个公式或从使用 **CellsU** 属性的某个程序按名称引用 LockThemeColors 单元格，请使用：

 
  
|||
|:-----|:-----|
|单元格名称：  <br/> |LockThemeEffects  <br/> |
   
若要从某个程序按索引引用 LockThemeEffects 单元格，请使用带下列参数的 **CellsSRC** 属性：

 
  
|||
|:-----|:-----|
|内容索引：  <br/> |**visSectionObject** <br/> |
|行索引：  <br/> |**visRowLock** <br/> |
|单元格索引：  <br/> |**visLockThemeEffects** <br/> |
