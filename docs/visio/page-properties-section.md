# <a name="page-properties-section"></a>“Page Properties”内容

包含控制页属性（如页面宽度、高度和比例）的单元格。
  
## <a name="remarks"></a>注解

您可以通过在 " **page properties** " 内容单元格中进行输入, 或使用 "**页面设置**" 对话框 (在 "**设计**" 选项卡上, 单击 "**页面设置**" 组中的对话框启动器) 设置页面属性的值。
