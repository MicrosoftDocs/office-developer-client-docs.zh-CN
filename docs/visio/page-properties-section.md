# <a name="page-properties-section"></a>“Page Properties”部分

包含控制页属性（如页面宽度、高度和比例）的单元格。
  
## <a name="remarks"></a>注解

您可以设置页面属性的值进行的单元格中**网页属性**部分中，输入或通过使用**页面设置**对话框 （在**设计**选项卡上，单击**页面设置**组中的对话框启动器）。
