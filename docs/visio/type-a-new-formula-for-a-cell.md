# <a name="type-a-new-formula-for-a-cell"></a>为单元格键入新公式

1. 选择单元格。
    
    单元格的公式出现在单元格和公式栏中。
    
2. 键入新的公式。
    
    键入该公式将替换单元格的上一的公式。
    
    如果您键入一个数字，但不提供的度量单位，Microsoft Visio 将使用从单元格的以前的值的单位，如果出现一个。 否则，如果单元格需要单位，Visio 将单元格使用默认的单位。
    
3. 若要接受公式，请按 ENTER。
    
    如果数字或公式包含错误，Visio 将显示一条消息，然后突出显示单元格的公式栏中的错误。 更正此错误，然后按 ENTER 接受更正。
