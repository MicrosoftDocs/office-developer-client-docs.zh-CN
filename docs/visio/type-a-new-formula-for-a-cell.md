# <a name="type-a-new-formula-for-a-cell"></a>为单元格键入新公式

1. 选择单元格。
    
    单元格的公式显示在单元格和编辑栏中。
    
2. 键入新的公式。
    
    键入的公式将替换单元格的上一个公式。
    
    如果键入数字但不提供度量单位, Microsoft Visio 将使用单元格的上一个值中的单位 (如果有)。 否则, 如果单元格需要单位, Visio 将使用单元格的默认单位。
    
3. 若要接受公式, 请按 enter。
    
    如果数字或公式包含错误, Visio 将显示一条消息, 然后在编辑栏或单元格中突出显示错误。 更正错误, 然后按 enter 接受更正。
