# <a name="log10-function"></a>LOG10 函数

返回数字的对数。 该数字必须是正数或 LOG10 返回错误值 # num。
  
## <a name="syntax"></a>语法

LOG10 (* **数量** *) 
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _number_ <br/> |必需  <br/> |**Numeric** <br/> | 要查找其以 10 为底的对数的数。  <br/> |
   
### <a name="return-value"></a>返回值

Numeric
  
## <a name="example"></a>示例

LOG10(42) 
  
返回 1.6232。
