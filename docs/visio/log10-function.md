# <a name="log10-function"></a>LOG10 函数

返回一个数以10为底的对数。 该数字必须为正数或 LOG10 返回 #NUM 的错误值。
  
## <a name="syntax"></a>语法

LOG10 (* **数字** *) 
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _number_ <br/> |必需  <br/> |**数值** <br/> | 要查找其以 10 为底的对数的数。  <br/> |
   
### <a name="return-value"></a>返回值

数值
  
## <a name="example"></a>示例

LOG10 (42) 
  
返回 1.6232。
