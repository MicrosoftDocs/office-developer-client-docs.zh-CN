# <a name="log10-function"></a>LOG10 函数

返回数字的底数 10 对数。 数字必须为正数，否则 LOG10 将返回错误#NUM。
  
## <a name="syntax"></a>语法

LOG10 (** *number* ** )  
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _number_ <br/> |必需  <br/> |**Numeric** <br/> | 要查找其以 10 为底的对数的数。  <br/> |
   
### <a name="return-value"></a>返回值

Numeric
  
## <a name="example"></a>示例

LOG10 (42)  
  
返回 1.6232。
