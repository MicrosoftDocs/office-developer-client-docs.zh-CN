# <a name="containermembercount-function"></a>CONTAINERMEMBERCOUNT 函数

返回容器中形状的总数。
  
## <a name="version-information"></a>版本信息

添加的版本： Visio 2010
 
  
## <a name="syntax"></a>语法

CONTAINERMEMBERCOUNT()
  
### <a name="return-value"></a>返回值

 **Integer**
  
## <a name="remarks"></a>说明

如果形状不是容器，则 CONTAINERMEMBERCOUNT 返回 -1。
