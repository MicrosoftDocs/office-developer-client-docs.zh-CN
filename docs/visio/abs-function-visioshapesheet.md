# <a name="abs-function-visioshapesheet"></a>ABS 函数 (VisioShapeSheet)

返回数字的绝对值。
  
## <a name="syntax"></a>语法

ABS (*数字*) 
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _Number_ <br/> |必需  <br/> |**Decimal** <br/> |要查找其绝对值的数字。  <br/> |
   
## <a name="example"></a>示例

ABS (-1.24) 
  
返回 1.24。
