# <a name="reference-a-cell-in-a-formula"></a>引用公式中的单元格

1. 双击您要放置引用的单元格，然后将插入点放在公式中所需的位置上。
    
2. 单击要引用的单元格。
    
3. 按 Enter 以接受公式，或者按 Esc 取消更改。
