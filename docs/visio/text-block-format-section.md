# <a name="text-block-format-section"></a>“Text Block Format”内容

包含控制形状文本块中文本的对齐方式、边距以及默认制表位位置的单元格。
  
## <a name="remarks"></a>注解

您还可以通过使用**文本**对话框来设置这些文本属性 （在**主页**选项卡上，单击**字体**组中的对话框启动器）。
