# <a name="updatealignbox-cell-miscellaneous-section"></a>UpdateAlignBox 单元格（“Miscellaneous”部分）

只要移动控制手柄，就重新计算选择矩形。
  
## <a name="remarks"></a>注释

要从另一个公式或从使用 **CellsU** 属性的某个程序按名称获取对 UpdateAlignBox 单元格的引用，请使用： 
  
|||
|:-----|:-----|
| 单元格名称：  <br/> | UpdateAlignBox  <br/> |
   
要从某个程序按索引获取对 UpdateAlignBox 单元格的引用，请使用带下列参数的 **CellsSRC** 属性： 
  
|||
|:-----|:-----|
| 内容索引：  <br/> |**visSectionObject** <br/> |
| 行索引：  <br/> |**visRowMisc** <br/> |
| 单元格索引：  <br/> |**visUpdateAlignBox** <br/> |
