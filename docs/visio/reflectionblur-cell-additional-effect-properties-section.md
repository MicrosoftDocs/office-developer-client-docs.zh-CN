# <a name="reflectionblur-cell-additional-effect-properties-section"></a>ReflectionBlur 单元格（“Additional Effect Properties”部分）

确定一个形状，以磅为单位 0.0 和 100.0 之间的反射的模糊量。
  
## <a name="remarks"></a>说明

要从另一个公式，由**N** **单元格**元素的属性的值或使用**CellsU**属性从某个程序按名称获取对**ReflectionBlur**单元格的引用，请使用： 
  
|||
|:-----|:-----|
| 单元格名称：  <br/> | ReflectionBlur  <br/> |
   
若要从某个程序按索引获取对**ReflectionBlur**单元格的引用，请使用带下列参数的**CellsSRC**属性： 
  
|||
|:-----|:-----|
| 内容索引：  <br/> |**visSectionObject** <br/> |
| 行索引：  <br/> |**visRowOtherEffectProperties** <br/> |
| 单元格索引：  <br/> |**visReflectionBlur** <br/> |
