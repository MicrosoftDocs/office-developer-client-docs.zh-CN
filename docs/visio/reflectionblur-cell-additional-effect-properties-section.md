# <a name="reflectionblur-cell-additional-effect-properties-section"></a>ReflectionBlur 单元格 ("其他效果属性" 部分)

确定形状上的反射的模糊量, 以磅为单位, 介于0.0 和100.0 之间。
  
## <a name="remarks"></a>注解

若要从另一个公式按名称获取对**ReflectionBlur**单元格的引用、 **cell**元素的**N**属性值, 或从使用**CellsU**属性的某个程序获取对该单元格的引用, 请使用: 
  
|||
|:-----|:-----|
| 单元格名称：  <br/> | ReflectionBlur  <br/> |
   
若要从某个程序按索引获取对**ReflectionBlur**单元格的引用, 请使用带下列参数的**CellsSRC**属性: 
  
|||
|:-----|:-----|
| 内容索引：  <br/> |**visSectionObject** <br/> |
| 行索引：  <br/> |**visRowOtherEffectProperties** <br/> |
| 单元格索引：  <br/> |**visReflectionBlur** <br/> |
