# <a name="3-d-rotation-properties-section"></a>"三维旋转属性"部分

显示形状的旋转角度沿 x 轴、y 轴和 z 轴的当前值。
  
> [!NOTE]
> "三维旋转属性"部分不包括控制形状斜面效果的单元格。 若要设置斜面效果，请更改"斜面属性" [部分单元格](bevel-properties-section.md) 的值。 
  
## <a name="remarks"></a>备注

可以在"设置形状格式"对话框中更改三维旋转值 ("主形状样式效果  >    >    >  **""三** 维旋转"，或者通过更改"三维旋转属性"部分中的单元格之一来更改旋转值。
