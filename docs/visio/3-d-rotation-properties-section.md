# <a name="3-d-rotation-properties-section"></a>"三维旋转属性" 部分

显示形状的当前值沿 x 轴、y 轴和 z 轴的旋转角度。
  
> [!NOTE]
> "三维旋转属性" 部分不包含控制形状棱台效果的单元格。 若要设置凹凸效果, 请更改 "[棱台属性](bevel-properties-section.md)" 部分中的单元格的值。 
  
## <a name="remarks"></a>注解

您可以更改 "**设置形状格式**" 对话框中的3-d 旋转值 (**主页** > **形状样式** > **效果** > 三维**旋转**或通过对三维**旋转属性中的一个单元格进行更改**部分。
