# <a name="text-fields-section"></a>“Text Fields”内容

显示使用**字段**对话框在形状的文本中插入的公式和函数。 
  
## <a name="remarks"></a>备注

添加此内容的唯一方法是使用**字段**对话框 （在**插入**选项卡） 将域插入到形状。 
  
文本块中的每个文本域均在 ShapeSheet 窗口中具有对应的行。行的名称表示该域包含的字符数。
