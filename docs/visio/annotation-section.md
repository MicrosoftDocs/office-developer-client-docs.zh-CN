# <a name="annotation-section"></a>“Annotation”部分

包含与插入到文档页中的注释有关的信息。 
  
> [!NOTE]
> 本节用于跟踪注释，仅当打开 Microsoft Visio 2013 中的.vsd 文件或.vsd 文件格式保存.vsdx 文件时。 它不用于跟踪在 Visio 2013 中的新.vsdx 文档中的注释。 
  
## <a name="remarks"></a>说明

**Annotation**内容的页面的 ShapeSheet 包含一个行添加到页面在绘图中，每个批注是否在原始绘图页或标记贴页面 （包含标记的审阅者添加到绘图页）。 在**Annotation**内容中不包含有关使用墨迹或其他形状，而不是注释已添加的标记的信息。 
  
若要查看页面的 ShapeSheet，请确保未选择该页面上的任何内容，右键单击该页面，然后单击 **“显示 ShapeSheet”**。
  
有关特定审阅者的信息，请参阅文档的 ShapeSheet 中的**Reviewer**内容。
