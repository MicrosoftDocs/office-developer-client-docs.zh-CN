# <a name="intup-function"></a>INTUP 函数

将数字向上舍入到下一个整数。
  
## <a name="syntax"></a>语法

INTUP (* **数字** *) 
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _number_ <br/> |必需  <br/> |**Number** <br/> |要向上舍入的数。  <br/> |
   
## <a name="example-1"></a>示例 1

INTUP (3.2)
  
返回 4。
  
## <a name="example-2"></a>示例 2

INTUP (-3.2)
  
返回 -3。
  
## <a name="example-3"></a>示例 3

INTUP (3)
  
返回 3。
