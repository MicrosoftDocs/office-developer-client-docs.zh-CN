# <a name="is1d-function"></a>IS1D 函数

如果形状是 1-D（一维）形状，则返回 TRUE；如果形状是 2-D（二维）形状，则返回 FALSE。
  
## <a name="version-information"></a>版本信息

添加的版本： Visio 2010
 
  
## <a name="syntax"></a>语法

IS1D ()
  
### <a name="return-value"></a>返回值

 **Boolean**
  
## <a name="remarks"></a>备注

如果工作表（形状）是页面或文档，则 IS1D 函数返回 #REF!。
