# <a name="atan-function"></a>ATAN 函数

返回一个数字，例如，角度的正切值_数_的反正切。 生成的角度位于范围-PI/2 < = 角度 < = PI/2 弧度 (-90 < = 角度 < = 90 度)。 
  
## <a name="syntax"></a>语法

ATAN (* **数量** *) 
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _number_ <br/> |必需  <br/> |**数字** <br/> |角度的正切值。  <br/> |
   
## <a name="example"></a>示例

ATAN(1) 
  
返回 45 度
