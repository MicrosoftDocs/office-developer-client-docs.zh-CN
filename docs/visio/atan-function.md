# <a name="atan-function"></a>ATAN 函数

返回一个数的反正切值, 例如, 其正切值为_数字_的角度。 计算得出的角度的范围为 -PI/2 <= 角度 <= PI/2 弧度（-90 <= 角度 <= 90 度）。 
  
## <a name="syntax"></a>语法

ATAN (* **数字** *) 
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _number_ <br/> |必需  <br/> |**数值** <br/> |角度的正切值。  <br/> |
   
## <a name="example"></a>示例

ATAN (1) 
  
返回 45 度
