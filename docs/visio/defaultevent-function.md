# <a name="defaultevent-function"></a>DEFAULTEVENT 函数

执行与对象关联的默认事件。
  
## <a name="syntax"></a>语法

DEFAULTEVENT( )
  
## <a name="remarks"></a>注解

下表列出与几个对象相关联的默认事件。
  
|**对象**|**默认事件**|
|:-----|:-----|
|Shape  <br/> |编辑文本。  <br/> |
|Group  <br/> |不执行任何操作。  <br/> |
|OLE  <br/> |执行主要动作。  <br/> |
|其他  <br/> |不执行任何操作。  <br/> |
   
在 Visio 的早期版本中，此函数以 _DEFAULTEVENT 的形式出现，而组合的默认事件是打开一个组合窗口。
