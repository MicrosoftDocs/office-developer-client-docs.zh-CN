# <a name="listmembercount-function"></a>LISTMEMBERCOUNT 函数

返回列表容器形状中成员形状的数目。
  
## <a name="version-information"></a>版本信息

添加的版本： Visio 2010
 
  
## <a name="syntax"></a>语法

LISTMEMBERCOUNT ()
  
### <a name="return-value"></a>返回值

整数
  
## <a name="remarks"></a>说明

如果形状不是列表容器，则 LISTMEMBERCOUNT 函数返回 -1。
