# <a name="data3-function"></a>DATA3 函数

返回形状在**形状名**对话框的**数据 3**框中的文本 （在**开发人员**选项卡，单击**形状名**），为 string。 
  
## <a name="syntax"></a>语法

DATA3 ()
  
## <a name="remarks"></a>注解

如果该框为空，则函数返回空字符串。
