# <a name="data3-function"></a>DATA3 函数

返回形状的 "**形状名称**" 对话框 (在 "**开发工具**" 选项卡上, 单击 "**形状名称**") 中的 "**数据 3** " 框中的文本, 作为字符串。 
  
## <a name="syntax"></a>语法

DATA3 ()
  
## <a name="remarks"></a>说明

如果该框为空，则函数返回空字符串。
