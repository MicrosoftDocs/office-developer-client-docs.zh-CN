# <a name="containercount-function"></a>CONTAINERCOUNT 函数

返回包含成员形式的形状（包括嵌套关系）的容器的总数。
  
## <a name="version-information"></a>版本信息

添加的版本： Visio 2010
 
  
## <a name="syntax"></a>语法

CONTAINERCOUNT()
  
### <a name="return-value"></a>返回值

 **Integer**
