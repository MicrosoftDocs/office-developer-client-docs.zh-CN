# <a name="infiniteline-row-geometry-section"></a>InfiniteLine 行（“Geometry”部分）

包含*x*和*y* -无限长线上两个点的坐标。 
  
InfiniteLine 行包含以下单元格。
  
|**Cell**|**说明**|
|:-----|:-----|
|[X](x-cell-geometry-section.md) <br/> |*X* -无限长线; 上某个点的坐标与*y*配对-Y 单元格所表示的坐标。  <br/> |
|[Y](y-cell-geometry-section.md) <br/> |*Y* -无限长线; 上某个点的坐标与*x*配对-X 单元格所表示的坐标。  <br/> |
|[A](a-cell-geometry-section.md) <br/> |*X* -无限长线; 上某个点的坐标与*y*配对-B 单元格所表示的坐标。  <br/> |
|[B](b-cell-geometry-section.md) <br/> |*Y* -; 无限长线上某个点的坐标与*x*配对-A 单元格所表示的坐标。  <br/> |
   
## <a name="remarks"></a>注解

包含 Ellipse 或 InfiniteLine 行的“Geometry”内容不应包含任何其他行。
