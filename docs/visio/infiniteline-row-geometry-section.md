# <a name="infiniteline-row-geometry-section"></a>InfiniteLine 行（“Geometry”内容）

包含  *无限长*  线上两个点的 x 坐标和  *y*  坐标。 
  
InfiniteLine 行包含以下单元格。
  
|**Cell**|**说明**|
|:-----|:-----|
|[X](x-cell-geometry-section.md) <br/> |无限  *长*  线上点的 x 坐标;与由  *Y 单元格*  表示的 y 坐标配对。  <br/> |
|[Y](y-cell-geometry-section.md) <br/> |无限长线上点的  *y*  坐标;与 X 单元格表示的  *x*  坐标配对。  <br/> |
|[A](a-cell-geometry-section.md) <br/> |无限  *长*  线上点的 x 坐标;与 B 单元格表示的  *y*  坐标配对。  <br/> |
|[B](b-cell-geometry-section.md) <br/> |无限长线上点的  *y*  坐标;与由 A 单元格表示的  *x*  坐标配对。  <br/> |
   
## <a name="remarks"></a>备注

包含 Ellipse 或 InfiniteLine 行的“Geometry”内容不应包含任何其他行。
