# <a name="theme-properties-section"></a>"主题属性" 部分

显示当前应用于文档的主题的索引值, 包括颜色、字体、连接符和效果值。 
  
## <a name="remarks"></a>注解

您可以在 "**设计**" 选项卡上设置主题值, 也可以在 ShapeSheet 的 "**主题属性**" 部分中的单元格中进行更改。
