# <a name="theme-properties-section"></a>主题属性部分

显示当前应用于文档，包括颜色、 字体、 连接器和效果值的主题的索引值。 
  
## <a name="remarks"></a>备注

在**设计**选项卡上或通过在 ShapeSheet 的**主题属性**部分中的单元格进行更改，您可以设置主题值。
