# <a name="theme-properties-section"></a>Theme Properties Section

显示当前应用于文档的主题的索引值，包括颜色、字体、连接线和效果值。 
  
## <a name="remarks"></a>备注

可以在"设计" **选项卡上设置** 主题值，或者通过更改 ShapeSheet 的" **主题属性** "部分中的单元格来设置主题值。
