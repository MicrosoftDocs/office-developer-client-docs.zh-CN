# <a name="tan-function-visioshapesheet"></a>TAN Function (VisioShapeSheet)

返回一个角度的正切值。
  
## <a name="syntax"></a>语法

TAN (* **角度** *) 
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _角度_ <br/> |必需  <br/> |**Numeric** <br/> |要获取的正切值的角度。  <br/> |
   
### <a name="return-value"></a>返回值

Numeric
  
## <a name="remarks"></a>注解

TAN 函数使用弧度，除非在*angle*中指定不同的度量单位。 
  
## <a name="example"></a>示例

TAN(45 deg) 
  
返回 1。
