# <a name="reflectiontrans-cell-additional-effect-properties-section"></a>ReflectionTrans 单元格（“Additional Effect Properties”部分）

确定反射，从 0 到 100%的百分比的透明度。 
  
## <a name="remarks"></a>说明

要从另一个公式，由**N** **单元格**元素的属性的值或使用**CellsU**属性从某个程序按名称获取对**ReflectionTrans**单元格的引用，请使用： 
  
|||
|:-----|:-----|
| 单元格名称：  <br/> | ReflectionTrans  <br/> |
   
若要从某个程序按索引获取对**ReflectionTrans**单元格的引用，请使用带下列参数的**CellsSRC**属性： 
  
|||
|:-----|:-----|
| 内容索引：  <br/> |**visSectionObject** <br/> |
| 行索引：  <br/> |**visRowOtherEffectProperties** <br/> |
| 单元格索引：  <br/> |**visReflectionTrans** <br/> |
