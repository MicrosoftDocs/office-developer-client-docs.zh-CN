# <a name="ln-function"></a>LN 函数

返回数字的自然对数。 该数字必须为正数，否则 LN 将返回错误值 #NUM。
  
## <a name="syntax"></a>语法

LN (* **数字** *) 
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _number_ <br/> |必需  <br/> |**数值** <br/> | 要查找其自然对数的数。  <br/> |
   
### <a name="return-value"></a>返回值

数值
  
## <a name="example"></a>示例

LN (42) 
  
返回 3.7377。
