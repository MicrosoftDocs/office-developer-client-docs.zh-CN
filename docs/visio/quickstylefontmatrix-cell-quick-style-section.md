# <a name="quickstylefontmatrix-cell-quick-style-section"></a>QuickStyleFontMatrix 单元格（“Quick Style”部分）

为从 1 到 6 的整数确定的每个快速样式的字体的样式。
  
## <a name="remarks"></a>说明

要从另一个公式，由**N** **单元格**元素的属性的值或使用**CellsU**属性从某个程序按名称获取对**QuickStyleFontMatrix**单元格的引用，请使用： 
  
|||
|:-----|:-----|
| 单元格名称：  <br/> | QuickStyleFontMatrix  <br/> |
   
若要从某个程序按索引获取对**QuickStyleFontMatrix**单元格的引用，请使用带下列参数的**CellsSRC**属性： 
  
|||
|:-----|:-----|
| 内容索引：  <br/> |**visSectionObject** <br/> |
| 行索引：  <br/> |**visRowQuickStyleProperties** <br/> |
| 单元格索引：  <br/> |**visQuickStyleFontMatrix** <br/> |
