# <a name="embellishmentindex-cell-theme-properties-section"></a>EmbellishmentIndex 单元格 ("主题属性" 部分)

更改标注、容器、时间线和组织结构图形状的外观 (embellishment)。
  
## <a name="remarks"></a>说明

若要从另一个公式按名称获取对**EmbellishmentIndex**单元格的引用、 **cell**元素的**N**属性值, 或从使用**CellsU**属性的某个程序获取对该单元格的引用, 请使用: 
  
|||
|:-----|:-----|
| 单元格名称：  <br/> | EmbellishmentIndex  <br/> |
   
若要从某个程序按索引获取对**EmbellishmentIndex**单元格的引用, 请使用带下列参数的**CellsSRC**属性: 
  
|||
|:-----|:-----|
| 内容索引：  <br/> |**visSectionObject** <br/> |
| 行索引：  <br/> |**visRowThemeProperties** <br/> |
| 单元格索引：  <br/> |**visEmbellishmentIndex** <br/> |
