# <a name="variationstyleindex-cell-theme-properties-section"></a>VariationStyleIndex Cell (Theme Properties Section) 

确定页面上活动主题变体的样式索引，作为整数。
  
## <a name="remarks"></a>备注

若要从另一个公式、Cell 元素 **的 N** 属性值或使用 **CellsU** 属性从某个程序按名称获取对 **VariationStyleIndex** 单元格的引用，请使用： 
  
|||
|:-----|:-----|
| 单元格名称：  <br/> | VariationStyleIndex  <br/> |
   
若要从程序按索引获取对 **VariationStyleIndex** 单元格的引用，请使用带下列参数的 **CellsSRC** 属性： 
  
|||
|:-----|:-----|
| 内容索引：  <br/> |**visSectionObject** <br/> |
| 行索引：  <br/> |**visRowThemeProperties** <br/> |
| 单元格索引：  <br/> |**visVariationStyleIndex** <br/> |
