# <a name="listorder-function"></a>LISTORDER 函数

返回形状在列表中的位置 (从1开始)。
  
## <a name="version-information"></a>版本信息

添加的版本： Visio 2010
 
  
## <a name="syntax"></a>语法

LISTORDER ()
  
### <a name="return-value"></a>返回值

整数
  
## <a name="remarks"></a>注解

如果形状不是列表成员，则 LISTORDER 函数返回 -1。
