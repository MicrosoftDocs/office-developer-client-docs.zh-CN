# <a name="relcubbezto-row-geometry-section"></a>RelCubBezTo Row (Geometry Section) 

包含三次方贝塞尔曲线终点相对于形状的宽度和高度的  *x*  坐标和  *y*  坐标，曲线相对形状的宽度和高度的起点的控制点的  *x*  坐标和  *y*  坐标，以及曲线相对形状的宽度和高度终点的控制点的  *x*  坐标和  *y*  坐标。 
  
> [!NOTE]
> **RelCubBezTo** 行只能保留为 .vsdx、.vsdm、.vstx、.vstm、.vssx 和 .vssm 文件格式。 将文件保存为 Visio 2003-2010 格式时 **，RelCubBezTo** 行将转换为 [NURBSTo](nurbsto-row-geometry-section.md)行。 
  
**RelCubBezTo** 行包含以下单元格。 
  
|**Cell**|**说明**|
|:-----|:-----|
|[X](x-cell-geometry-section.md) <br/> |三次方贝塞尔曲线的终顶点相对于形状宽度的  *x*  坐标。  <br/> |
|[Y](y-cell-geometry-section.md) <br/> |三次方贝塞尔曲线终顶点相对于形状高度的  *y*  坐标。  <br/> |
|[A](a-cell-geometry-section.md) <br/> |曲线的起点控制点相对于形状宽度的  *x*  坐标;弧形上的点。控制点最好位于弧形的开始顶点和结束顶点之间。  <br/> |
|[B](b-cell-geometry-section.md) <br/> |曲线的起点控制点相对于形状高度的  *y*  坐标。  <br/> |
|[C](c-cell-geometry-section.md) <br/> |曲线的结束控制点相对于形状宽度的  *x*  坐标;弧形上的点。控制点最好位于弧形的开始控制点和结束顶点之间。  <br/> |
|[D](d-cell-geometry-section.md) <br/> |曲线的结束控制点相对于形状高度的  *y*  坐标。  <br/> |
