# <a name="user-defined-cells-section"></a>“User-defined Cells”内容

在特定于用户的单元格中输入公式（可由其他单元格和加载项工具引用）的工作区。
  
## <a name="remarks"></a>备注

与["Scratch"](scratch-section.md)部分一样，解决方案可以通过许多不同的方式使用"用户定义的单元格"部分。 **"User-defined Cells"内容** 的优点就是单元格可以命名;**"Scratch"内容中的** 单元格无法命名。 
  
如果您的解决方案具有明确的目的，建议您使用 **User-defined Cells** 部分中的单元格，以便清楚起见，因为可以命名 User 单元格。 **Scratch** 部分通常用于隔离重复的复杂计算。 
  
若要添加此内容，请使用 **“插入内容”** 对话框（在 ShapeSheet 窗口中单击右键，然后单击 **“插入内容”**）。
