# <a name="fill-gradient-section"></a>“Fill Gradient”部分

显示应用于形状的填充渐变颜色值。 
  
## <a name="remarks"></a>说明

部分中的各[渐变停止行 （Fill 渐变内容）](gradient-stop-row-fill-gradient-section.md)行代表中填充的渐变光圈。 此部分可以为空。 
  
仅包含一节中的前 10 行使用渐变设置。 10 之外的所有**渐变停止**行将被忽略。 
  
在**设置形状格式**窗格中，可以在渐变属性设置填充 （在**主页**选项卡，**形状样式**组中，单击**填充**，然后单击**填充选项**）。 
  
您可以通过启用或禁用渐变填充形状更改[FillGradientEnabled 单元格 （渐变 Properties 内容）](fillgradientenabled-cell-gradient-properties-section.md)单元格的值。
