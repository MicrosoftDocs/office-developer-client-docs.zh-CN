# <a name="height-cell-shape-transform-section"></a>Height 单元格（“Shape Transform”内容）

确定以绘图单位表示的形状的高度。
  
## <a name="remarks"></a>注释

要从另一个公式或使用**CellsU**属性从某个程序按名称获取对 Height 单元格的引用，请使用： 
  
|||
|:-----|:-----|
| 单元格名称：  <br/> | Height  <br/> |
   
若要从某个程序按索引获取对 Height 单元格的引用，请使用带下列参数的**CellsSRC**属性： 
  
|||
|:-----|:-----|
| 内容索引：  <br/> |**visSectionObject** <br/> |
| 行索引：  <br/> |**visRowXFormOut** <br/> |
| 单元格索引：  <br/> |**visXFormHeight** <br/> |
