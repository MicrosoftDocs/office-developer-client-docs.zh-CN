# <a name="text-transform-section"></a>“Text Transform”内容

包含与形状的文本块有关的位置信息。
  
## <a name="remarks"></a>备注

若要添加此内容，请使用 **“插入内容”** 对话框（在 ShapeSheet 窗口中单击右键，然后单击 **“插入内容”**）。 在"**文本转换**"部分设置值与使用文本块工具处理文本块 **("主页**"选项卡上的"工具") 。
