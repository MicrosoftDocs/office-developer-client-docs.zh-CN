# <a name="open-the-shapesheet-for-a-shape-page-or-document"></a>打开形状、页面或文档的 ShapeSheet

默认情况下，Microsoft Visio 中，在 ShapeSheet 窗口处于隐藏状态。 若要访问 shapesheet （英文），您需要运行 Visio 开发人员模式运行 （显示开发人员选项卡） 中。
  
### <a name="to-run-visio-in-developer-mode"></a>若要以开发人员模式运行 Visio

1. 单击**文件**选项卡，然后单击**选项**。
    
2. **Visio 选项**对话框中，单击**高级**，然后选择在**常规**下的**开发工具模式运行**。
    
### <a name="to-view-the-shapesheet-for-a-shape"></a>若要查看形状 ShapeSheet

- 右键单击形状在绘图画布上的，单击**显示 ShapeSheet**。
    
- 选择的形状在绘图画布，然后在**开发人员**选项卡的**形状设计**组中，单击**显示 ShapeSheet**，然后单击**形状**。
    
### <a name="to-view-the-sheet-for-a-page"></a>若要查看页面的工作表

- 在**开发人员**选项卡的**形状设计**组中，单击**显示 ShapeSheet**，，然后单击**页面**。
    
### <a name="to-view-the-sheet-for-a-document"></a>若要查看文档的工作表

- 在**开发人员**选项卡的**形状设计**组中，单击**显示 ShapeSheet**，，然后单击**文档**。
