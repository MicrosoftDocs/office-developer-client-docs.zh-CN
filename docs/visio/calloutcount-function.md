# <a name="calloutcount-function"></a>CALLOUTCOUNT 函数

返回与形状相关的标注形状总数。
  
## <a name="version-information"></a>版本信息

添加的版本： Visio 2010
 
  
## <a name="syntax"></a>语法

CALLOUTCOUNT ()
  
### <a name="return-value"></a>返回值

 **Integer**
