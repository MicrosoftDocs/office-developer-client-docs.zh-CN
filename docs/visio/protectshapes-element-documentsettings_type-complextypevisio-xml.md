# <a name="protectshapes-element-documentsettings_type-complextype-visio-xml"></a>ProtectShapes 元素 (DocumentSettings_Type complexType)  (Visio XML)

|**元素类型** <br/> |[ProtectShapes_Type](protectshapes_type-complextypevisio-xml.md) <br/> |
|**命名空间** <br/> |http://schemas.microsoft.com/office/visio/2012/main  <br/> |
|**架构文件** <br/> |VisioSchema15.xsd  <br/> |
|**文档部件** <br/> |document.xml  <br/> |
   
## <a name="definition"></a>定义

```XML
< xs:element name="ProtectShapes" type="ProtectShapes_Type" minOccurs="0" maxOccurs="1" >
</xs:element >
```
