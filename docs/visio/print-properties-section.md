# <a name="print-properties-section"></a>“Print Properties”内容

包含用于控制如何设置绘图页在打印纸上显示的格式的单元格。 
  
## <a name="remarks"></a>备注

使用 **"打印属性** "部分可控制页面级别的打印。 这些设置基于每页并且独立于绘图页设置，例如，可以采用纵向方向打印采用横向方向设置的绘图页。 还可以使用组合方向打印文档。 
  
"打印属性"部分中的设置对应于"设计"选项卡上 ("页面设置"对话框中"打印设置"选项卡上设置的值，单击"页面设置"组中对话框启动器) 。   有关 **适用于绘图** 页的设置，请参阅"页面属性"部分。
