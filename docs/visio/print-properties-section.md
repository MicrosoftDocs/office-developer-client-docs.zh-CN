# <a name="print-properties-section"></a>“Print Properties”部分

包含用于控制如何设置绘图页在打印纸上显示的格式的单元格。 
  
## <a name="remarks"></a>注解

**打印属性**部分用于控制打印页级别。 这些设置为每个页，因此独立的绘图页设置，例如，可以使用纵向打印使用横向打印绘图页。 此外，可以使用组合方向打印文档。 
  
**打印属性**部分中的设置对应于**页面设置**对话框中**打印设置**选项卡上设置的值 （**设计**选项卡中，单击**页面设置**组中的对话框启动器）。 请参阅适用于在绘图页的设置**页上属性**部分。
