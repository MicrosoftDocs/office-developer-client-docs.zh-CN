# <a name="directory-function"></a>DIRECTORY 函数

返回在其中保存文档，为 string 的目录 （文件夹） 的完整路径。 如果文档尚未保存，则返回空字符串。
  
## <a name="syntax"></a>语法

DIRECTORY ()
