# <a name="directory-function"></a>DIRECTORY 函数

以字符串的形式返回保存文档的目录 (文件夹) 的完整路径。 如果尚未保存文档, 则返回一个空字符串。
  
## <a name="syntax"></a>语法

DIRECTORY ()
