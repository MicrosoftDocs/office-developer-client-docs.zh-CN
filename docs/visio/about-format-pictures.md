# <a name="about-format-pictures"></a>关于设置图片格式

格式图片用来确定如何显示一个值。例如，您可以控制小数点右侧或左侧显示的位数，或者控制一个文本字符串显示为大写还是小写。
  
> [!NOTE]
> 使用 Microsoft Office system 格式定义的日期或时间的格式图片，括起格式图片中双大括号，例如，"{{d/yy}}"。 如果您使用的预定义的格式，例如，201，将其括大括号和尖括号，如下所示:"{\<201\>}" 
  
以下各节显示符号，可用来设置不同类型的值的显示格式。
  
## <a name="string-and-numeric-values"></a>字符串和数值

|**字符**|**说明**|
|:-----|:-----|
|#  <br/> |数字占位符。显示一位数字或不显示任何数字。不显示前导或末尾的零。如果小数点左侧的位数多于占位符的位数，则所有位都会显示出来。如果小数点右侧的位数多于占位符的位数，则小数部分将被舍入为占位符的位数。对于一个尺寸，如果占位符是最左侧的位，则不显示值为零的次级单位。  <br/> 例如，FORMAT(0ft 11.25in,"#.##u") 的结果是 11.25in。  <br/> |
|0  <br/> |数字占位符（零）。显示一位数字或不显示任何数字。显示前导或末尾的零。如果小数点左侧的位数多于占位符的位数，则所有位都会显示出来。如果小数点右侧的位数多于占位符的位数，则小数部分将被舍入为占位符的位数。对于一个尺寸，显示值为零的次级单位。  <br/> 例如，FORMAT(2ft 11.33in,"0.## u") 的结果是 2 ft. 11.33 in。  <br/> |
|.  <br/> |小数占位符。 确定左侧和右侧的十进制位置显示多少位数。 在多单位，小数点用在最小 （最右边） 子单位。 显示定义的系统的**区域和语言**的十进制字符设置 （控制面板）。  <br/> 例如，FORMAT(250 cm,"0.000 u") 的结果是 250.000 cm。  <br/> |
|,  <br/> |千位分隔符。 如果外面数字占位符 （# 或 0），分隔符分隔千具有小数点左侧的四个或多个数字的号码。 显示千位分隔符为系统的**区域和语言**定义设置 （控制面板）。  <br/> |
|E- E+ e- e+  <br/> |科学计数格式。如果该格式中至少有一个数字占位符位于这些符号的右侧，数字将以科学计数格式显示。在数字和其指数间插入 E 或 e。对于 E+ 或 e+，在正指数前显示正号 (+)，在负指数前显示负号 (-)。对于 E- 或 e-，只有在指数为负数时显示负号 (-)。  <br/> 例如，FORMAT(12345.67,"###.#e+#") 的结果是 .123.5e+2。  <br/> |
|u 或 U  <br/> |短标签占位符。 插入每个子单位后缩写的单位标签。 例如： 英寸、 英尺度。 U 占位符插入 u 占位符插入小写标签大小写混合的标签。 占位符前插入标签前的空格数相同。  <br/> 例如，FORMAT(12 c 13 d,"#u") 的结果是 13c1。  <br/> |
|uu 或 UU  <br/> |长标签占位符。 每个子单位后插入单位标签。 例如： 英寸、 英尺度 U 占位符插入大小写混合的标签时 u 占位符插入小写标签。 占位符前插入标签前的空格数相同。  <br/> 例如，FORMAT(12.43in,"# #/4 UU") 的结果是 12 2/4 INCHES。  <br/> |
|uuu 或 UUU  <br/> |通用标签占位符。 每个子单位后插入单位标签的通用 （到 Visio 内部） 窗体。 U 占位符插入 u 占位符插入小写标签大小写混合的标签。 占位符前插入标签前的空格数相同。  <br/> |
|/  <br/> |分数占位符。如果存在前导数字占位符，它将表达式显示为一个代分数。否则显示为一个假分数。如果数字占位符在分母中，将该分数舍为分子为 1 的最接近的分数并化简。如果分母中没有数字占位符，则舍入为最接近的分数但不要化简。  <br/> 例如，FORMAT(12.43,"# #/4") 显示为 12 2/4。  <br/> |
|空格  <br/> |格式化输出中显示一个空格字符。 若要显示另一个字符，请使用反斜杠 (\)字符。  <br/> |
   
## <a name="currency-values"></a>货币值

|**字符**|**说明**|
|:-----|:-----|
|$  <br/> |货币符号。 显示为系统的**区域和语言**定义的货币符号设置 （控制面板）  <br/> |
|u 或 U  <br/> |短标签占位符。 插入的标准本地货币符号或非本地货币的三个字符货币缩写。 例如，$99.00，42.70 FRF。 U 占位符插入小写和 U 插入大小写混合的标签。  <br/> |
|uu 或 UU  <br/> |长标签占位符。 插入每个子单位后长货币标签。 例如： 美元、 法国法郎。 U 占位符插入小写和 U 插入大小写混合的标签。  <br/> |
|uuu 或 UUU  <br/> |通用标签占位符。 每个子单位后插入所有货币的通用、 三个字符的货币缩写。 例如，99.00 美元，42.70 FRF。 U 占位符插入小写和 U 插入大小写混合的标签。 占位符前插入标签前的空格数相同。  <br/> |
   
## <a name="text-values"></a>文本值

|**字符**|**说明**|
|:-----|:-----|
|\  <br/> |显示下一个字符原样。 若要显示的反斜杠字符，请键入\\。 请参阅"text"。  <br/> |
|"text" 或 'text'  <br/> |按原样显示引号内的文本。另请参阅 \（反斜杠）。  <br/> |
|@  <br/> |文本占位符。在表达式的值为字符串时替代该字符串。  <br/> 例如，FORMAT("Hello", "'You entered ('@')'" ) 结果为“You entered (Hello)”。  <br/> |
|@+  <br/> |大写文本占位符。对于字符串值，将输入替换为大写。  <br/> 例如，FORMAT("Hello", "@ @+ @-" ) 的结果是“Hello HELLO hello)”。  <br/> |
|@-  <br/> |文本占位符。 对于字符串值，将输入替换为小写。  <br/> 例如，FORMAT("Hello", "@ @+ @-" ) 的结果是“Hello HELLO hello)”。  <br/> |

|HH  <br/> |小时占位符。以有一个前导零的 24 小时制 (00-24) 显示小时。  <br/> |

|s  <br/> |秒钟占位符。以无前导零的形式显示秒钟 (0-59)。  <br/> |
|ss  <br/> |秒钟占位符。以有一个前导零的形式显示秒钟 (00-59)。  <br/> |
|t  <br/> |AM/PM 缩写。 显示定义的系统的**区域和语言**的缩写设置 （控制面板）。  <br/> |
|tt  <br/> |AM/PM 指示符。 显示完整的指示符定义系统的**区域和语言**设置 （控制面板）。  <br/> |
|t_c 或 tt_c  <br/> |繁体中文 AM/PM 指示符。显示指示符。与用户的区域设置无关。  <br/> |

|tt_e  <br/> |英文 AM/PM 指示符。显示完整的指示符。与用户的区域设置无关。  <br/> |
|t_s 或 tt_s  <br/> |简体中文 AM/PM 指示符。显示指示符。与用户的区域设置无关。  <br/> |
|T  <br/> |标准时间格式。  <br/> |
