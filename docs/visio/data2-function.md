# <a name="data2-function"></a>DATA2 函数

返回形状**名称**对话框 (在 "**开发工具**" 选项卡上, 单击 "**形状名称**") 中的 "**数据 2** " 框中的文本, 作为字符串。 
  
## <a name="syntax"></a>语法

DATA2 ()
  
## <a name="remarks"></a>说明

如果该框为空，则函数返回空字符串。
