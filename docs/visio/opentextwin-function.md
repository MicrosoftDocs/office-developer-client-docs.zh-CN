# <a name="opentextwin-function"></a>OPENTEXTWIN 函数

打开形状的文本块, 以便可以对文本进行编辑。
  
## <a name="syntax"></a>语法

OPENTEXTWIN( )
  
## <a name="remarks"></a>说明

在 Visio 的早期版本中，此函数以 _OPENTEXTWIN 的形式出现。Visio 4.0 版和更高版本接受这两种样式中的任意一种。
