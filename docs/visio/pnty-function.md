# <a name="pnty-function"></a>PNTY 函数

返回点的_y_坐标。
  
## <a name="syntax"></a>语法

PNTX (* **点** *) 
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _鼠标_ <br/> |必需  <br/> |**Number** <br/> |点的_y_坐标。  <br/> |
   
### <a name="return-value"></a>返回值

帐号
  
## <a name="example"></a>示例

PNTY (PNT (7, 12)) 
  
返回 12。
