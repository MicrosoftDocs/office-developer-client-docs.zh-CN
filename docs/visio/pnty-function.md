# <a name="pnty-function"></a>PNTY 函数

返回_y_-点的坐标。
  
## <a name="syntax"></a>语法

PNTX (* **指向** *) 
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _磅_ <br/> |必需  <br/> |**编号** <br/> |_Y_-点的坐标。  <br/> |
   
### <a name="return-value"></a>返回值

Number
  
## <a name="example"></a>示例

PNTY(PNT(7,12)) 
  
返回 12。
