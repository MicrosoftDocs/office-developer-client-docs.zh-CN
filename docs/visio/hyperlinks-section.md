# <a name="hyperlinks-section"></a>“Hyperlinks”部分

包含用于创建多个跳跃形状或绘图页和其他绘图页、 另一个文件或网站的单元格。
  
## <a name="remarks"></a>说明

若要定义超链接，请在此内容的单元格中输入值，或者使用 **“超链接”** 对话框（单击 **“插入”** 选项卡上的 **“超链接”**）。 
  
可以将“Hyperlinks”内容添加到 ShapeSheet 窗口中的形状、组合、页或外来对象中。
