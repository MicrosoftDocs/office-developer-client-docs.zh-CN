# <a name="cos-function-visioshapesheet"></a>COS 函数 (VisioShapeSheet)

返回一个角度的余弦值。 
  
## <a name="syntax"></a>语法

COS （*角度*） 
  
### <a name="parameters"></a>参数

|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _角度_ <br/> |必需  <br/> |**数字** <br/> |要获取其余弦值的角度。  <br/> |
   
## <a name="remarks"></a>注解

COS 函数使用弧度，除非在*angle*中指定不同的度量单位。 
  
## <a name="example"></a>示例

COS(45 deg) 
  
返回 0.7071。
