# <a name="doccreation-function"></a>DOCCREATION 函数

以序列值的形式返回创建文档的日期和时间。
  
## <a name="syntax"></a>语法

DOCCREATION ()
  
## <a name="remarks"></a>注解

使用 FORMAT 函数设置该值的格式。
