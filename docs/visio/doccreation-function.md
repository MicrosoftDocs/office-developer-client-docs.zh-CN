# <a name="doccreation-function"></a>DOCCREATION 函数

返回日期和时间创建了文档，作为序列值。
  
## <a name="syntax"></a>语法

DOCCREATION ()
  
## <a name="remarks"></a>注解

使用 FORMAT 函数设置该值的格式。
