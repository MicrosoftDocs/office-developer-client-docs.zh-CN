# <a name="sheetref-function"></a>SHEETREF 函数

返回一个对在_sheetref_中指定的工作表 (形状) 的引用, 如果没有_sheetref_限定符, 则返回到当前工作表。 
  
## <a name="version-information"></a>版本信息

添加的版本： Visio 2010
 
  
## <a name="syntax"></a>语法

 *\<sheetref\>!*  SHEETREF () 
  
### <a name="return-value"></a>返回值

ShapeSheet 参考
  
## <a name="remarks"></a>注解

您可以在使用工作表引用标记的其他函数中使用此函数。
