# <a name="sheetref-function"></a>SHEETREF 函数

返回对 sheetref 中 (工作表) ，或者，如果没有 _sheetref_ 限定符，则返回对当前工作表的引用。  
  
## <a name="version-information"></a>版本信息

添加的版本： Visio 2010
 
  
## <a name="syntax"></a>语法

 *\<sheetref \> ！*  SHEETREF ()  
  
### <a name="return-value"></a>返回值

ShapeSheet 参考
  
## <a name="remarks"></a>备注

您可以在使用工作表引用标记的其他函数中使用此函数。
