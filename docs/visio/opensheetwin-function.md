# <a name="opensheetwin-function"></a>OPENSHEETWIN 函数

在新窗口中打开 ShapeSheet 窗口。
  
## <a name="syntax"></a>语法

OPENSHEETWIN( )
  
## <a name="remarks"></a>注解

在 Visio 的早期版本中，此函数以 _OPENSHEETWIN 的形式出现。Visio 4.0 版和更高版本接受这两种样式中的任意一种。
