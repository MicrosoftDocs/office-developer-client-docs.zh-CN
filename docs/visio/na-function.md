# <a name="na-function"></a>NA 函数

返回错误值 #NA！。 
  
在信息缺失的 ShapeSheet 单元格中使用 NA 函数，以便在计算中不使用单元格的默认值。
  
## <a name="syntax"></a>语法

NA( )
  
### <a name="return-value"></a>返回值

String
  
## <a name="remarks"></a>备注

NA 是表示“没有可用值”的错误值。
