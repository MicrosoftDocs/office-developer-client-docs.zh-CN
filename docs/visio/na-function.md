# <a name="na-function"></a>NA 函数

将返回错误值 #NA ！。 
  
在信息缺失的 ShapeSheet 单元格中使用 NA 函数，以便在计算中不使用单元格的默认值。
  
## <a name="syntax"></a>语法

NA( )
  
### <a name="return-value"></a>返回值

字符串
  
## <a name="remarks"></a>注解

NA 是表示“没有可用值”的错误值。
