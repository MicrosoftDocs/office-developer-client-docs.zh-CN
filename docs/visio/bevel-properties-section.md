# <a name="bevel-properties-section"></a>"棱台属性" 部分

显示应用于形状的任何斜面效果的属性。 
  
## <a name="remarks"></a>注解

您可以在 "**设置形状格式**" 窗格中为形状应用斜角 ("**主页** > **形状样式** > "**棱台** > **选项**, 或对 "**棱台属性**" 部分中的单元格进行更改表.
