# <a name="bevel-properties-section"></a>“Bevel Properties”部分

显示任何各种棱台效果应用于形状的属性。 
  
## <a name="remarks"></a>说明

您可以将凹凸效果应用于**设置形状格式**窗格中的形状 (**主** > **形状样式** > **Bevel** > **三维选项**，或通过的**Bevel 属性**部分中的单元格进行更改ShapeSheet。
