# <a name="third-party-applications-integrating-with-office-mobile-products-on-windows-android-and-ios"></a>与 Office、Android 和 iOS Windows移动产品集成的第三方应用程序
 
作为我公司的授权代表，我同意将每个文件所有权类型的正确信息（无论文件被视为企业文件还是个人/使用者文件）通过 Windows 10 上的 Expando 属性、Android 上的存储访问框架和 iOS 上的文档选取器来传递给 Microsoft。 企业文件类型是指，存储在专门面向企业的存储产品上的文件。 个人/使用者文件类型是指，存储在专门面向使用者的存储产品上的文件。
  
我进一步 (，) 无法将文档正确识别为使用者或企业可能会导致我的公司无法与 Microsoft Office 产品集成;和 (b) Microsoft 可能会时时更新这些条款，并且我或其他代表以我的能力行动，在传递后续文件信息之前检查并查看新条款是否适用。 如果 Microsoft 更改条款，则在此内容页顶部找到的生效日期将指示该更改，例如"自月/年更新"。
