# <a name="third-party-applications-integrating-with-office-mobile-products-on-windows-android-and-ios"></a>与 Windows、Android 和 iOS 上的 Office 移动产品集成的第三方应用程序
 
作为我公司的授权代表，我同意将每个文件所有权类型的正确信息（无论文件被视为企业文件还是个人/使用者文件）通过 Windows 10 上的 Expando 属性、Android 上的存储访问框架和 iOS 上的文档选取器来传递给 Microsoft。 企业文件类型是指，存储在专门面向企业的存储产品上的文件。 个人/使用者文件类型是指，存储在专门面向使用者的存储产品上的文件。
  
我进一步确认 (a) 未能正确地将文档标识为消费者或 business 可能导致我的公司能够与 Microsoft Office 产品集成;(b) Microsoft 可能会在一段时间内更新这些条款, 而 onus 位于我的容量中, 或另一个代表在传递后续文件信息之前, 查看并查看新条款是否适用。 如果 Microsoft 更改了术语, 则在此内容页顶部找到的有效日期将指示所做的更改, 例如, "根据月/年进行更新"。
