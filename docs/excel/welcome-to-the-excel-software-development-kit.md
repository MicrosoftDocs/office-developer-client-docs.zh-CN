# <a name="welcome-to-the-excel-software-development-kit"></a>欢迎使用 Excel 软件开发工具包

 **适用于** Excel 2013 | Office 2013 | Visual Studio 
  
欢迎使用 Excel 2013 XLL 软件开发工具包 (SDK) 文档。此参考包含帮助你开发 Microsoft Excel 2013 XLL 的概念概述、编程任务和示例。
  
修订日期：2012 年 11 月
  
下载 [Excel 2013 XLL SDK](https://go.microsoft.com/fwlink/?LinkID=251082&amp;clcid=0x409)。
  
Excel 2013 XLL SDK 包括以下方面：
  
- **C 应用程序编程接口 (API)**—包括使 DLL 能够访问 Excel 2013 功能的标头和源文件，以及接口说明，DLL 应将其公开以使用 Excel 加载项管理器。
    
- **Microsoft Visual Studio 项目**—包括 C/C++ 源代码并介绍了如何使用 C API。这些示例项目提供了示例，它们是你开发自己的加载项的起点。
    
SDK 文档包含以下部分：
  
- [Getting Started with the Excel XLL SDK](getting-started-with-the-excel-xll-sdk.md)
    
- [Developing Excel XLLs](developing-excel-xlls.md)
    
- [Developing Excel Cluster Connectors](developing-excel-cluster-connectors.md)
    
- [Excel XLL SDK API Function Reference](excel-xll-sdk-api-function-reference.md)
    
## <a name="functionality-not-covered"></a>未涵盖的功能

不涵盖以下主题：
  
- 在 Excel 宏 (XLM) 工作表中开发用户定义的函数和命令。
    
- 在 DLL 中创建用户定义的函数以控制 XLM 宏的执行流。
    
    此类函数通过返回一种特殊的流控制数据类型工作，本文档中不作介绍。
