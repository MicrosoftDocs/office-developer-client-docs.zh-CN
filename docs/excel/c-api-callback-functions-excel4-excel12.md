# <a name="c-api-callback-functions-excel4-excel12"></a>C API 回调函数 Excel4、Excel12

**适用于** Excel 2013 | Office 2013 | Visual Studio 
  
**Excel4**和**Excel12**函数可用于启用 Dll 调用内部 Microsoft Excel 工作表函数、 宏工作表函数或命令或仅 XLL 特殊函数或命令。 所有的最新版本的 Excel 支持**Excel4**函数。 支持在 Excel 2007 中启动**Excel12**函数。 两个窗体中提供了两个函数： 
  
- 可变长度参数列表表单 (**Excel4/Excel12**)
    
- 数组参数窗体 (**Excel4v/Excel12v**)
    
除其中参数传递给这些回调的方式，这两种形式的作用相同。 [Excel4/Excel12](excel4-excel12.md)完全介绍这两种形式的基本概念。 [Excel4v/Excel12v](excel4v-excel12v.md)介绍有关此窗体的其他问题。
