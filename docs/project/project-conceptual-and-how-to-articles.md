# <a name="project-conceptual-and-how-to-articles"></a>项目概念性和操作说明文章

通过使用 JavaScript 和 .net Framework、project Professional 2013 中的任务窗格外接程序、project Server 接口和 CSOM 开发, 在本节中查找有关项目的信息的链接。OData 报告服务。
  
project server 2013 基于 Office project server 2007 中引入的 project server Interface (PSI) web 服务框架。 Project Server 2013 平台是使用 .net Framework 4 生成的。 Project Web App 需要 SharePoint Server 2013 作为新功能的功能的基础, 例如, 用于访问云中的 Project Server 的客户端对象模型 (CSOM)、Windows Workflow Foundation (WF4) 的版本4和适用于的客户端的 OData 服务报告数据库表。 Project Professional 2013 可以使用新的任务窗格 Office 外接程序。
