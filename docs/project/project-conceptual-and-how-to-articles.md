# <a name="project-conceptual-and-how-to-articles"></a>项目概念性和操作说明文章

查找本节中有关 Project 2013 平台的主要部分、使用 JavaScript 和 .NET Framework 进行 CSOM 开发入门、Project Professional 2013 中的任务窗格加载项、Project 服务器接口和 OData 报告服务的信息的链接。
  
ProjectServer 2013 基于 Office Project Project Server 2007 中引入的 Project Server Interface (PSI) Web 服务框架。 the Project Server 2013 platform is built with the .NET Framework 4. Project Web App 需要 SharePoint Server 2013 作为新功能的基础，例如客户端对象模型 (CSOM) 以访问云中的 Project 服务器、Windows Workflow Foundation (WF4) 版本 4 和报告数据库表的 OData 服务。 Project Professional 2013 可以使用新的任务窗格Office外接程序。

- [Project Server 2013](project-server-2013-architecture-and-programmability.md)体系结构和可编程性介绍了 Project 2013 平台的主要部分，包括 Project 客户端和 Project Server。 
  
- [Project 2013](client-side-object-model-csom-for-project-2013.md)的客户端对象模型 (CSOM) 包括有关使用 JavaScript 和 .NET Framework 进行 CSOM 开发入门的主题。 
  
- [Project编程](project-client-programming.md)包括 2013 年 10 月任务窗格Project Professional简介。 单独的 SDK 包含有关 Project 和其他 Office 2013 客户端中的外接程序的文章和参考。 有关详细信息，请参阅Office[加载项](https://docs.microsoft.com/office/dev/add-ins/overview/office-add-ins)开发人员文档。
