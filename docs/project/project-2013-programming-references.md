# <a name="project-2013-programming-references"></a>Project 2013 编程参考文档

查找 Project Server Interface (PSI) 参考的概述、如何使用 ASMX 接口和 PSI 的 WCF 接口、有关 Project Server 错误代码的信息以及 **ProjectData OData** 服务的引用。 
  
> [!NOTE]
> 有关 Project Server 2013 中 PSI 和客户端对象模型 (CSOM) 的主要引用，请参阅以下部分：类库和[Web](https://msdn.microsoft.com/library/ef1830e0-3c9a-4f98-aa0a-5556c298e7d1%28Office.15%29.aspx)服务引用和 Project [Server 2013](javascript-library-and-rest-reference-for-project-server-2013.md)的 JavaScript API 引用。 > 有关 Office 外接程序引用（包括 Project 2013 任务窗格应用程序的 JavaScript 引用）的信息，请参阅适用于 Office 的[JavaScript API。](https://msdn.microsoft.com/library/fp142185.aspx) > 有关 Project Standard 2013 和 Project Professional 2013 的 VBA 参考，请参阅[Project 2013 VBA 开发人员参考](https://msdn.microsoft.com/library/jj235035.aspx)。 >本地 Project Server 2013 数据库中的报告表和视图的引用尚不可用。 
  
## <a name="in-this-section"></a>本节内容

[Project PSI 参考概述](project-psi-reference-overview.md)介绍 Project Server 2013 的 PSI 引用中的程序集和命名空间。 
  
[基于 ASMX 的代码示例在 Project](prerequisites-for-asmx-based-code-samples-in-project.md)介绍如何使用 PSI 引用中的代码示例，这些代码示例是使用测试应用程序中的 ASMX Web 服务构建的。 
  
[基于 WCF 的代码示例的先决条件Project](prerequisites-for-wcf-based-code-samples-in-project.md)介绍如何使用 PSI 引用中的代码示例，这些代码示例在测试应用程序中使用 Windows Communication Foundation (WCF) 服务构建。 
  
[Project 服务器错误代码](project-server-error-codes.md)按功能区域列出 Project Server 错误的代码和说明。 
  
[ProjectData - Project OData 服务引用](https://msdn.microsoft.com/library/office/jj163015.aspx)包括适用于 Project Server 报告数据的 OData 服务的概述、介绍将 OData 源与 LINQ 查询一同使用，以及 **引用 ProjectData** 服务中的 XML 元数据。
