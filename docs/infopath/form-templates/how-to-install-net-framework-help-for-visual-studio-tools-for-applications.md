# <a name="install-local-net-framework-help-for-visual-studio-tools-for-applications"></a>Visual Studio Tools for Applications 安装本地.NET Framework 帮助

当运行 Microsoft InfoPath 和 Visual Studio 2008 的计算机连接到 Internet 时，Visual Studio 2008 帮助查看器可以访问在线来源中的 .NET Framework 文档以及 Visual Basic 和 Visual C# 语言参考。但是，如果您使用的计算机未连接到 Internet，则您将无法访问这些文档源，直到您下载并安装了 .NET Framework 2.0 SDK 文档的本地副本，并执行了将该文档与 Visual Studio 2008 开发环境相集成所需的步骤。
  
要查看说明如何执行此操作的帮助主题，请打开帮助中的“目录”**** 窗格，展开“Visual Studio for Applications”****，再展开“集成开发环境入门”****，然后单击“如何：将 .NET Framework SDK 文档与集成开发环境集成”****。
  
> [!IMPORTANT]
> 如果您运行在 Windows Vista 的用户帐户控制，您必须执行上面引用的过程之前以管理员身份启动 InfoPath。 为此，导航到 C:\Program Files\Microsoft Office\Office14，右键单击 INFOPATH。EXE，，然后单击**以管理员身份运行**。 如果您不以管理员身份运行 InfoPath，您将无法完成此过程。
