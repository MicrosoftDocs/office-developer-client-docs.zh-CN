# <a name="guidelines-for-improving-the-performance-of-infopath-forms"></a>提高 InfoPath 表单性能的指南

设计具有更高性能的 InfoPath 表单模板需要您仔细规划视图大小和数量、XML 的大小和结构以及所用文件和图片控件的数量。还需要仔细组织表单中数字签名及数据连接的大小和数量。通过遵循提高 InfoPath 2007 表单的性能一文中的指导原则和最佳做法，您可以避免在设计表单模板时出现性能问题，并排查现有表单模板中的这些相同问题。
