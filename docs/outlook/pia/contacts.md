# <a name="contacts"></a>联系人

本节收录了与联系人相关的示例主题。 联系人表示要与之通信的人员和企业。 “联系人”文件夹中包含电子邮件通讯簿和联系人信息存储。 [ContactItem](https://msdn.microsoft.com/library/bb644956\(v=office.15\)) 对象是 Outlook 内置对象，表示“联系人”文件夹中的唯一联系人。 **ContactItem** 对象有超过 100 个属性（如 [FirstName](https://msdn.microsoft.com/library/bb652965\(v=office.15\))、[CompanyName](https://msdn.microsoft.com/library/bb610212\(v=office.15\)) 和 [OfficeLocation](https://msdn.microsoft.com/library/bb647145\(v=office.15\))），以及在内置属性不足时创建的自定义属性。这些属性可用于存储每个联系人的相关信息。
