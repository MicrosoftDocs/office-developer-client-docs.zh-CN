# <a name="ifreebusysupportplaceholder2"></a>IFreeBusySupport::Placeholder2

此成员是占位符，不受支持。
  
## <a name="quick-info"></a>快速信息

请参阅 [IFreeBusySupport](ifreebusysupport.md)。
  
```cpp
HRESULT Placeholder2();
```

## <a name="return-values"></a>返回值

始终S_OK，因为此成员是占位符且不受支持。
  
## <a name="remarks"></a>备注

此成员必须返回S_OK。
