# <a name="functions-outlook-exported-apis"></a>函数 (Outlook 导出的 api)

本节介绍了 Outlook 导出的函数。
  
|**函数**|**说明**|
|:-----|:-----|
|[HrCreateApptRebaser](hrcreateapptrebaser.md) <br/> |初始化[IOlkApptRebaser](iolkapptrebaser.md)对象, 以便在 Outlook 日历中的重定约会中使用。  <br/> |
|[HrProcessConvActionForSentItem](hrprocessconvactionforsentitem.md) <br/> |根据邮件项目的[PidTagConversationId](https://msdn.microsoft.com/library/f8e4a5fa-cb73-4eca-b174-72e1fda821a6%28Office.15%29.aspx)对邮件项目执行发送后分类。  <br/> |
|[RebaseTaskComplete](rebasetaskcomplete.md) <br/> |报告重定约会的完成情况。  <br/> |
|[RebaseTaskProgress](rebasetaskprogress.md) <br/> |报告约会的枚举和重定的进度。  <br/> |
