# <a name="data-types-outlook-exported-apis"></a>数据类型 (Outlook 导出的 api)

本节介绍了 Outlook 导出的数据类型。
  
|**数据类型**|**说明**|
|:-----|:-----|
|[TZDEFINITION](tzdefinition.md) <br/> |表示包含所有历史、当前和未来时区班次规则的时区的完整时区 (夏时制)。  <br/> |
|[TZREG](tzreg.md) <br/> |定义夏时制的开始时间、何时返回标准时间以及夏时制转换的小时数。  <br/> |
|[TZRULE](tzrule.md) <br/> |指定有关夏时制开始时间的时区规则的信息, 以及该时区规则优先的生效年份。  <br/> |
