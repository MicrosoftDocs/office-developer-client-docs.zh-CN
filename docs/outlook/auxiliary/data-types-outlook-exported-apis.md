# <a name="data-types-outlook-exported-apis"></a>数据类型 （Outlook 导出 Api）

本节介绍 Outlook 将导出的数据类型。
  
|**数据类型**|**说明**|
|:-----|:-----|
|[TZDEFINITION](tzdefinition.md) <br/> |代表包括夏时制的所有历史、 当前和将来的时区 shift 规则整个所在的时区。  <br/> |
|[TZREG](tzreg.md) <br/> |定义夏时制时间启动时，返回到标准时间发生时，和多少个小时夏 shift 是。  <br/> |
|[TZRULE](tzrule.md) <br/> |指定的时区规则时夏时制开始，以及该所在的时区规则首先会生效的年份的信息。  <br/> |
