# <a name="ifreebusysupportplaceholder10"></a>IFreeBusySupport::Placeholder10

## <a name="quick-info"></a>快速信息

请参阅[IFreeBusySupport](ifreebusysupport.md)。
  

此成员必须返回 S_OK。
