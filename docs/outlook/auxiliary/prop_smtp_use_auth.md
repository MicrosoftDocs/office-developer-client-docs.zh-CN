# <a name="propsmtpuseauth"></a>PROP_SMTP_USE_AUTH

指定是否使用的 SMTP 帐户身份验证。
  
## <a name="quick-info"></a>快速信息

|||
|:-----|:-----|
|标识符:  <br/> |0x0203  <br/> |
|属性类型  <br/> |PT_DWORD  <br/> |
|属性标记：  <br/> |0x02030003  <br/> |
|访问权限  <br/> |只读  <br/> |
   
## <a name="remarks"></a>备注

0 值意味着不使用身份验证，否则使用身份验证。
