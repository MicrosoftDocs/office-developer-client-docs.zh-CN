# <a name="prop_smtp_ssl"></a>PROP_SMTP_SSL

指定是否对 SMTP 帐户 (SSL) 安全套接字层协议。
  
## <a name="quick-info"></a>快速信息

|||
|:-----|:-----|
|标识符:  <br/> |0x0202  <br/> |
|属性类型  <br/> |PT_DWORD  <br/> |
|属性标记：  <br/> |0x02020003  <br/> |
|访问权限  <br/> |只读  <br/> |
   
## <a name="remarks"></a>备注

零值表示不使用 SSL 加密，否则使用 SSL 加密。
