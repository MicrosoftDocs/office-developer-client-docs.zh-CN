# <a name="propsmtpssl"></a>PROP_SMTP_SSL

指定是否使用安全套接字层 (SSL) 协议进行 SMTP 帐户。
  
## <a name="quick-info"></a>快速信息

|||
|:-----|:-----|
|标识符：  <br/> |0x0202  <br/> |
|属性类型  <br/> |PT_DWORD  <br/> |
|属性标记：  <br/> |0x02020003  <br/> |
|访问权限  <br/> |只读  <br/> |
   
## <a name="remarks"></a>说明

0 值意味着不使用 SSL 加密，否则使用 SSL 加密。
