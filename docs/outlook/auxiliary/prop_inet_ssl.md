# <a name="propinetssl"></a>PROP_INET_SSL

指定是否为常规 Internet 邮箱使用安全套接字层 (SSL)。
  
## <a name="quick-info"></a>快速信息

|||
|:-----|:-----|
|标识符：  <br/> |0x0105  <br/> |
|属性类型  <br/> |PT_DWORD  <br/> |
|属性标记：  <br/> |0x01050003  <br/> |
|访问权限  <br/> |只读  <br/> |
   
## <a name="remarks"></a>说明

0 值意味着不使用 SSL，否则使用 SSL。
