# <a name="propsmtpusespa"></a>PROP_SMTP_USE_SPA

指定是否对 SMTP 帐户使用安全密码身份验证 (SPA)。
  
## <a name="quick-info"></a>快速信息

|||
|:-----|:-----|
|标识符:  <br/> |0x0207  <br/> |
|属性类型  <br/> |PT_DWORD  <br/> |
|属性标记：  <br/> |0x02070003  <br/> |
|访问权限  <br/> |只读  <br/> |
   
## <a name="remarks"></a>说明

零值表示不使用 spa, 否则使用 spa。
