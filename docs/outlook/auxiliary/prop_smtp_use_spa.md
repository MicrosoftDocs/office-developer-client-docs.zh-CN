# <a name="propsmtpusespa"></a>PROP_SMTP_USE_SPA

指定是否为 SMTP 帐户使用安全密码身份验证 (SPA)。
  
## <a name="quick-info"></a>快速信息

|||
|:-----|:-----|
|标识符：  <br/> |0x0207  <br/> |
|属性类型  <br/> |PT_DWORD  <br/> |
|属性标记：  <br/> |0x02070003  <br/> |
|访问权限  <br/> |只读  <br/> |
   
## <a name="remarks"></a>说明

0 值意味着不使用 SPA，否则使用 SPA。
