# <a name="ifreebusysupportplaceholder12"></a>IFreeBusySupport::Placeholder12

此成员是占位符，不受支持。
  
## <a name="quick-info"></a>快速信息

请参阅 [IFreeBusySupport](ifreebusysupport.md)。
  

此成员必须返回S_OK。
