# <a name="ienumfbblockreset"></a>IEnumFBBlock::Reset

通过将游标设置为开头来重置枚举数。
  
## <a name="quick-info"></a>快速信息

请参阅[IEnumFBBlock](ienumfbblock.md)。
  
```cpp
HRESULT Reset();
```

## <a name="return-values"></a>返回值

如果该调用成功，则返回 S_OK否则为一个错误代码。
