# <a name="interfaces-account-management-api"></a>帐户 (API) 

本部分介绍帐户管理 API 中的接口。
  
|**接口**|**说明**|
|:-----|:-----|
|[IOlkAccount](iolkaccount.md) <br/> |支持获取和设置属性以及帐户的其他信息。  <br/> |
|[IOlkAccountHelper](iolkaccounthelper.md) <br/> |在当前 MAPI 会话中提供用于管理帐户的帮助程序功能。  <br/> |
|[IOlkAccountManager](iolkaccountmanager.md) <br/> |管理对帐户的访问权限并设置有关帐户更改的通知。  <br/> |
|[IOlkAccountNotify](iolkaccountnotify.md) <br/> |为客户端提供对帐户更改的回调。  <br/> |
|[IOlkEnum](iolkenum.md) <br/> |支持将帐户枚举为 [IUnknown](https://docs.microsoft.com/windows/desktop/api/unknwn/nn-unknwn-iunknown) 对象。  <br/> |
|[IOlkErrorUnknown](iolkerrorunknown.md) <br/> |提供有关最后一个错误的额外信息。  <br/> |
