# <a name="about-the-data-degradation-layer-api"></a>关于数据降级层 API

数据降级层 API 提供了隐藏 Outlook 对象的基础本机字符格式的功能, 允许客户端以首选字符格式 (而不是对象的本机字符格式) 访问对象。 它提供[常量 (数据降级层 API)](constants-data-degradation-layer-api.md)中显示的常量定义, 以及返回以指定格式包装的 Outlook 对象的函数[HrCreateNewWrappedObject](hrcreatenewwrappedobject.md) 。
