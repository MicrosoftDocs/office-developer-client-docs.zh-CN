# <a name="about-the-data-degradation-layer-api"></a>关于数据降级层 API

数据降级层 API 提供能够隐藏基础本机字符格式的 Outlook 对象，允许客户端来访问首选的字符格式而不是对象的本机字符格式中的对象。 它提供了常量定义[常量 （数据降级层 API）](constants-data-degradation-layer-api.md)和函数返回指定的格式中自动换行的 Outlook 对象的[HrCreateNewWrappedObject](hrcreatenewwrappedobject.md)中所示。
