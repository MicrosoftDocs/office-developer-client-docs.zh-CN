# <a name="propsmtpport"></a>PROP_SMTP_PORT

代表 SMTP 帐户的端口号。
  
## <a name="quick-info"></a>快速信息

|||
|:-----|:-----|
|标识符：  <br/> |0x0201  <br/> |
|属性类型  <br/> |PT_DWORD  <br/> |
|属性标记：  <br/> |0x02010003  <br/> |
|访问权限  <br/> |只读  <br/> |
