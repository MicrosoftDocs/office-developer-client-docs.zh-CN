# <a name="iolkenumreset"></a>IOlkEnum::Reset

将枚举器重置到开头。
  
## <a name="quick-info"></a>快速信息

请参阅[IOlkEnum](iolkenum.md)。
  
```cpp
HRESULT IOlkEnum::Reset ();
```

## <a name="return-values"></a>返回值

如果该调用成功，则返回 S_OK否则为一个错误代码。
