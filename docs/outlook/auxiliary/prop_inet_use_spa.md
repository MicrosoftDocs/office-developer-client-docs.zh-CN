# <a name="prop_inet_use_spa"></a>PROP_INET_USE_SPA

指定是否对常规 Internet 邮箱 (SPA) 安全密码身份验证。
  
## <a name="quick-info"></a>快速信息

|||
|:-----|:-----|
|标识符:  <br/> |0x0108  <br/> |
|属性类型  <br/> |PT_DWORD  <br/> |
|属性标记：  <br/> |0x01080003  <br/> |
|访问权限  <br/> |只读  <br/> |
   
## <a name="remarks"></a>备注

零值表示不使用 SPA，否则使用 SPA。
