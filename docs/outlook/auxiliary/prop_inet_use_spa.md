# <a name="propinetusespa"></a>PROP_INET_USE_SPA

指定是否对常规 Internet 邮箱使用安全密码身份验证 (SPA)。
  
## <a name="quick-info"></a>快速信息

|||
|:-----|:-----|
|标识符:  <br/> |0x0108  <br/> |
|属性类型  <br/> |PT_DWORD  <br/> |
|属性标记：  <br/> |0x01080003  <br/> |
|访问权限  <br/> |只读  <br/> |
   
## <a name="remarks"></a>注解

零值表示不使用 spa, 否则使用 spa。
