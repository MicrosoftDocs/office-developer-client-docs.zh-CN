# <a name="interfaces-freebusy-api"></a>接口 (忙/闲 API)

本节介绍忙/闲 API 中的接口。
  
|**接口**|**说明**|
|:-----|:-----|
|[IEnumFBBlock](ienumfbblock.md) <br/> |支持访问和枚举忙/闲时间范围内的用户数据块。  <br/> |
|[IFreeBusyData](ifreebusydata.md) <br/> |为给定的用户、 获取和设置一个时间范围并返回枚举忙/闲此时间范围内的数据块界面。  <br/> |
|[IFreeBusySupport](ifreebusysupport.md) <br/> |支持接口访问指定的用户的忙/闲数据的规范。  <br/> |
