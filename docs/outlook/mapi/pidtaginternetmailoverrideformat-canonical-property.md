# <a name="pidtaginternetmailoverrideformat-canonical-property"></a>PidTagInternetMailOverrideFormat 规范属性

  
  
**适用于**： Outlook 2013 |Outlook 2016 
  
指示的编码方法和 HTML 包含应该设置传出邮件上的附件。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_INETMAIL_OVERRIDE_FORMAT  <br/> |
|标识符：  <br/> |0x5902  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |MIME  <br/> |
