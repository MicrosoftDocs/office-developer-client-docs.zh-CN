# <a name="pidtagchildrensnames-canonical-property"></a>PidTagChildrensNames 规范属性

  
  
**适用于**： Outlook 
  
包含的子级名称的列表
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_CHILDRENS_NAMES，PR_CHILDRENS_NAMES_A，PR_CHILDRENS_NAMES_W  <br/> |
|标识符：  <br/> |0x3A58  <br/> |
|数据类型：  <br/> |PT_MV_UNICODE PT_MV_STRING8  <br/> |
|区域：  <br/> |联系人  <br/> |
