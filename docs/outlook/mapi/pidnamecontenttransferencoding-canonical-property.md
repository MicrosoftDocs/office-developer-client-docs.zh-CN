# <a name="pidnamecontenttransferencoding-canonical-property"></a>PidNameContentTransferEncoding 规范属性

  
  
**适用于**： Outlook 2013 |Outlook 2016 
  
包含 [RFC3282] 内容传输编码标头字段值。
  
|||
|:-----|:-----|
|友好名称：  <br/> |无  <br/> |
|属性进行设置：  <br/> |PS_INTERNET_HEADERS  <br/> |
|属性名称：  <br/> |内容传送编码  <br/> |
|数据类型：  <br/> |PT_UNICODE  <br/> |
|区域：  <br/> |电子邮件  <br/> |
