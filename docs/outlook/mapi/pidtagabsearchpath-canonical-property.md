# <a name="pidtagabsearchpath-canonical-property"></a>PidTagAbSearchPath 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
包含要搜索以解析名称的通讯簿容器的条目标识符列表。 
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_AB_SEARCH_PATH  <br/> |
|标识符:  <br/> |0x3D05  <br/> |
|数据类型：  <br/> |PT_MV_BINARY  <br/> |
|区域：  <br/> |通讯簿  <br/> |
   
## <a name="remarks"></a>说明

不要使用此属性。 它保留供 MAPI 使用。
