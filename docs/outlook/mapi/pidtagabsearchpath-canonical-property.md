# <a name="pidtagabsearchpath-canonical-property"></a>PidTagAbSearchPath 规范属性

  
  
**适用于**： Outlook 
  
包含要搜索来解析名称的通讯簿容器的项标识符的列表。 
  
|||
|:-----|:-----|
|关联的属性：  <br/> |PR_AB_SEARCH_PATH  <br/> |
|标识符:  <br/> |0x3D05  <br/> |
|数据类型：  <br/> |PT_MV_BINARY  <br/> |
|区域：  <br/> |通讯簿  <br/> |
   
## <a name="remarks"></a>备注

不要使用此属性。 它是通过 MAPI 供使用。
