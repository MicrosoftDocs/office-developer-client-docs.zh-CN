# <a name="pidtagbusiness2telephonenumber-canonical-property"></a>PidTagBusiness2TelephoneNumber 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
在收件人的业务位置包含辅助电话号码。 
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_BUSINESS2_TELEPHONE_NUMBER、PR_BUSINESS2_TELEPHONE_NUMBER_A、PR_BUSINESS2_TELEPHONE_NUMBER_W、PR_OFFICE2_TELEPHONE_NUMBER、PR_OFFICE2_TELEPHONE_NUMBER_A、PR_OFFICE2_TELEPHONE_NUMBER_W  <br/> |
|标识符:  <br/> |0x3A1B  <br/> |
|数据类型：  <br/> |PT_UNICODE、PT_STRING8  <br/> |
|区域：  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>注解

这些属性是提供有关收件人的标识和访问信息的属性示例。 这些属性由收件人和收件人的组织定义。
