# <a name="pidtagmessagedeliveryid-canonical-property"></a>PidTagMessageDeliveryId 规范属性

  
  
**适用于**： Outlook 
  
包含传送到客户端应用程序的消息的邮件传输系统 (MTS) 标识符。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_MESSAGE_DELIVERY_ID  <br/> |
|标识符：  <br/> |0x001B  <br/> |
|数据类型：  <br/> |PT_BINARY  <br/> |
|区域：  <br/> |Server  <br/> |
