# <a name="pidtagregisteredmailtype-canonical-property"></a>PidTagRegisteredMailType 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
此属性包含用于实际传递邮件的注册类型。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_REGISTERED_MAIL_TYPE  <br/> |
|标识符:  <br/> |0x0C16  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |MAPI 收件人  <br/> |
