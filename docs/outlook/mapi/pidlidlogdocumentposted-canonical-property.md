# <a name="pidlidlogdocumentposted-canonical-property"></a>PidLidLogDocumentPosted 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
指示文档是在记录期间通过电子邮件发送的还是发送到服务器文件夹的。
  
|||
|:-----|:-----|

|LONG ID (的一) ：  <br/> |0x00008711  <br/> |
|数据类型：  <br/> |PT_BOOLEAN  <br/> |
|区域：  <br/> |日志  <br/> |
