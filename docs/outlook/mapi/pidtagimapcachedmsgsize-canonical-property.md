# <a name="pidtagimapcachedmsgsize-canonical-property"></a>PidTagImapCachedMsgsize 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
包含缓存的 Internet 邮件访问协议邮件的大小。
  
|||
|:-----|:-----|
|相关属性：  <br/> |无  <br/> |
|标识符:  <br/> |0x10F0  <br/> |
|数据类型：  <br/> |PT_BINARY  <br/> |
|区域：  <br/> |重命名邮件文件夹  <br/> |
