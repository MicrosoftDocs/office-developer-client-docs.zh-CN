# <a name="pidlidsharingconfigurationurl-canonical-property"></a>PidLidSharingConfigurationUrl 规范属性

  
  
**适用于**： Outlook 
  
将指定的共享邮件的属性。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidSharingConfigUrl  <br/> |
|属性进行设置：  <br/> |PSETID_Sharing  <br/> |
|长 ID （盖）：  <br/> |0x00008A24  <br/> |
|数据类型：  <br/> |PT_UNICODE  <br/> |
|区域：  <br/> |共享  <br/> |
   
## <a name="remarks"></a>说明

此属性的值必须是零长度字符串。
