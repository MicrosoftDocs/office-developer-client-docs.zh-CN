# <a name="pidtagcontactversion-canonical-property"></a>PidTagContactVersion 规范属性

  
  
**适用于**： Outlook 
  
包含收件人的联系人信息的版本。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_CONTACT_VERSION  <br/> |
|标识符：  <br/> |0x3A52  <br/> |
|数据类型：  <br/> |PT_CLSID  <br/> |
|区域：  <br/> |联系人  <br/> |
   
## <a name="remarks"></a>说明

此属性是提供标识和访问有关收件人信息的属性之一。 由收件人和收件人的组织定义这些属性。
