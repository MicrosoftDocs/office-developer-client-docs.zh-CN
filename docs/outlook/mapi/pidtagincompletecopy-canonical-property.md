# <a name="pidtagincompletecopy-canonical-property"></a>PidTagIncompleteCopy 规范属性

  
  
**适用于**： Outlook 2013 |Outlook 2016 
  
如果此消息的另一条消息不完整副本，包含 TRUE。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_INCOMPLETE_COPY  <br/> |
|标识符：  <br/> |0x0035  <br/> |
|数据类型：  <br/> |PT_BOOLEAN  <br/> |
|区域：  <br/> |常规消息属性  <br/> |
