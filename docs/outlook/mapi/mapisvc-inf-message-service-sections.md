# <a name="mapisvcinf-message-service-sections"></a>MapiSvc.inf 邮件服务部分

  
  
**适用于**： Outlook 
  
mapisvc.inf **[服务]** 节中列出的项的每个包括消息服务的一部分。 以下各节中的条目的两种类型： 一个用于设置某些属性，另一个用于列出与要配置的消息服务相关的节的名称。
