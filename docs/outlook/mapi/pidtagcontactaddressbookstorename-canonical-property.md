# <a name="pidtagcontactaddressbookstorename-canonical-property"></a>PidTagContactAddressBookStoreName 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
包含与通讯簿提供程序关联的存储区的显示名称。 
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_CONTAB_STORE_NAME、PR_CONTAB_STORE_NAME_A、PR_CONTAB_STORE_NAME_W  <br/> |
|标识符:  <br/> |0x6612  <br/> |
|数据类型：  <br/> |PT_UNICODE、PT_STRING8  <br/> |
|区域：  <br/> |联系人通讯簿  <br/> |
   
## <a name="remarks"></a>注解

这些属性位于联系人通讯簿容器中, 并且在联系人通讯簿容器表中的列。
