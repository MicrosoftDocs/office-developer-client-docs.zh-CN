# <a name="custom-processing-with-tnef"></a>使用 TNEF 进行自定义处理

**适用于**：Outlook 2013 | Outlook 2016 
  
传输提供程序可以使用自定义处理本身处理附件本身的属性, 单独传输附件, 或通过邮件系统的附件模型传送附件。 TNEF 使用一种机制, 使传输提供商能够从邮件中发送附件, 并将其重新连接到接收端。
