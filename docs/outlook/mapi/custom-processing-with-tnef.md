# <a name="custom-processing-with-tnef"></a>自定义处理与 TNEF

**适用于**： Outlook 2013 |Outlook 2016 
  
传输提供程序可以使用自定义处理处理附件本身上的属性、 单独传输附件或传输这些邮件系统的附件模型通过。 TNEF 使用允许传输提供程序以发送除了邮件附件并将其重新连接接收端的机制。
