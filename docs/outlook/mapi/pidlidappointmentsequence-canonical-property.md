# <a name="pidlidappointmentsequence-canonical-property"></a>PidLidAppointmentSequence 规范属性

  
  
**适用于**： Outlook 2013 |Outlook 2016 
  
指定会议对象的序列号。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidApptSequence  <br/> |
|属性进行设置：  <br/> |PSETID_Appointment  <br/> |
|长 ID （盖）：  <br/> |0x00008201  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |会议  <br/> |
