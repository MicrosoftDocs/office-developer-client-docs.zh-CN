# <a name="supporting-multivalued-properties-in-message-stores"></a>在消息存储中支持多值的属性

  
  
**适用于**： Outlook 
  
支持具有多个值的属性的要求均编档在[MAPI 属性概述](mapi-property-overview.md)。 多值属性支持不是必需的消息存储提供程序，但建议，特别是对于默认消息存储提供程序。 其他某个功能，如文件夹表单库，请需要多值属性支持。
