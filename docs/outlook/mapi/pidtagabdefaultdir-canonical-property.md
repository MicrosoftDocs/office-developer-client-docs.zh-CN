# <a name="pidtagabdefaultdir-canonical-property"></a>PidTagAbDefaultDir 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
包含要首先显示的通讯簿容器的条目标识符。 
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_AB_DEFAULT_DIR  <br/> |
|标识符:  <br/> |0x3D06  <br/> |
|数据类型：  <br/> |PT_BINARY  <br/> |
|区域：  <br/> |通讯簿  <br/> |
   
## <a name="remarks"></a>说明

不要使用此属性。 它保留供 MAPI 使用。
