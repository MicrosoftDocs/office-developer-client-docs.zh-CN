# <a name="pidtagcompanymaintelephonenumber-canonical-property"></a>PidTagCompanyMainTelephoneNumber 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
包含公司主要电话号码
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_COMPANY_MAIN_PHONE_NUMBER，PR_COMPANY_MAIN_PHONE_NUMBER_A，PR_COMPANY_MAIN_PHONE_NUMBER_W  <br/> |
|标识符：  <br/> |0x3A57  <br/> |
|数据类型：  <br/> |PT_UNICODE PT_STRING8  <br/> |
|区域：  <br/> |联系人  <br/> |
   
## <a name="remarks"></a>说明

由用户或用户的组织定义这些属性。
