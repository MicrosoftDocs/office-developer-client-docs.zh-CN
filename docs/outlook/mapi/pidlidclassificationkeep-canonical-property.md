# <a name="pidlidclassificationkeep-canonical-property"></a>PidLidClassificationKeep 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
指示邮件分类是否应在转发或答复邮件时与邮件一起保留。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidClassKeep  <br/> |
|属性集:  <br/> |PSETID_Common  <br/> |
|长 ID (盖子):  <br/> |0x000085BA  <br/> |
|数据类型：  <br/> |PT_BOOLEAN  <br/> |
|区域：  <br/> |常规邮件  <br/> |
   
## <a name="remarks"></a>注解

有关详细信息, 请参阅[如何为 Outlook 2007 部署邮件分类](https://msdn.microsoft.com/library/5a220424-edd5-4a21-b7fd-8106c23c3b39.aspx)。
