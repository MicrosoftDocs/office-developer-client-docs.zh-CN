# <a name="pidtagstoreproviders-canonical-property"></a>PidTagStoreProviders 规范属性

  
  
**适用于**： Outlook 
  
包含当前配置文件中的消息存储提供程序的标识符的列表。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_STORE_PROVIDERS  <br/> |
|标识符：  <br/> |0x3D00  <br/> |
|数据类型：  <br/> |PT_BINARY  <br/> |
|区域：  <br/> |MAPI 配置文件  <br/> |
   
## <a name="remarks"></a>说明

不要使用此属性。 它是通过 MAPI 供使用。
