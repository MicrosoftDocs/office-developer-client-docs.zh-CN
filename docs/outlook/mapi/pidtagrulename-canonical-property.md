# <a name="pidtagrulename-canonical-property"></a>PidTagRuleName 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
指定规则的用户可读名称。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_RULE_NAME  <br/> |
|标识符:  <br/> |0x6682  <br/> |
|数据类型：  <br/> |PT_STRING8  <br/> |
|区域：  <br/> |服务器端规则  <br/> |
