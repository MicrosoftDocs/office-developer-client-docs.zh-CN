# <a name="pidlidhomeaddress-canonical-property"></a>PidLidHomeAddress 规范属性

  
  
**适用于**： Outlook 
  
指定联系人的住宅地址的完整地址。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidHomeAddress  <br/> |
|属性进行设置：  <br/> |PSETID_Address  <br/> |
|长 ID （盖）：  <br/> |0x0000801A  <br/> |
|数据类型：  <br/> |PT_UNICODE  <br/> |
|区域：  <br/> |联系人  <br/> |
   
## <a name="remarks"></a>说明

此属性应为其他物理地址属性的组合，并基于客户端区域设置。
