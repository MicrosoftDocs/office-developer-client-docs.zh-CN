# <a name="pidtagabsearchpathupdate-canonical-property"></a>PidTagAbSearchPathUpdate 规范属性

  
  
**适用于**： Outlook 
  
包含由用户明确配置的地址簿容器的项标识符的列表。 
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_AB_SEARCH_PATH_UPDATE  <br/> |
|标识符：  <br/> |0x3D11  <br/> |
|数据类型：  <br/> |PT_BINARY  <br/> |
|区域：  <br/> |通讯簿  <br/> |
   
## <a name="remarks"></a>说明

不要使用此属性。 它是通过 MAPI 供使用。
