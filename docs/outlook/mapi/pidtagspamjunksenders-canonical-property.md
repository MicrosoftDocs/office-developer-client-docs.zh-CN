# <a name="pidtagspamjunksenders-canonical-property"></a>PidTagSpamJunkSenders 规范属性

  
  
**适用于**： Outlook 2013 |Outlook 2016 
  
包含以分号分隔的电子邮件地址和域组成阻止发件人列表。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_SPAM_JUNK_SENDERS_W  <br/> |
|长 ID （盖）：  <br/> |0x041A  <br/> |
|数据类型：  <br/> |PT_UNICODE  <br/> |
|区域：  <br/> |垃圾邮件  <br/> |
