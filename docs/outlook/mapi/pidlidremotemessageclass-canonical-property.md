# <a name="pidlidremotemessageclass-canonical-property"></a>PidLidRemoteMessageClass 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
指示远程项目的邮件类。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidRemoteMsgClass  <br/> |
|属性集：  <br/> |PSETID_Remote  <br/> |
|LONG ID (的一) ：  <br/> |0x00008F02  <br/> |
|数据类型：  <br/> |无  <br/> |
|区域：  <br/> |远程邮件  <br/> |
