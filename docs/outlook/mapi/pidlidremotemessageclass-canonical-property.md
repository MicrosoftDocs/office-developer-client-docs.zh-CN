# <a name="pidlidremotemessageclass-canonical-property"></a>PidLidRemoteMessageClass 规范属性

  
  
**适用于**： Outlook 2013 |Outlook 2016 
  
指示的远程项目的消息类。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidRemoteMsgClass  <br/> |
|属性进行设置：  <br/> |PSETID_Remote  <br/> |
|长 ID （盖）：  <br/> |0x00008F02  <br/> |
|数据类型：  <br/> |无  <br/> |
|区域：  <br/> |远程邮件  <br/> |
