# <a name="pidlidfax3richtextformat-canonical-property"></a>PidLidFax3RichTextFormat 规范属性

  
  
**适用于**： Outlook 
  
指定联系人的富文本格式的其他传真地址。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidFax3RTF  <br/> |
|属性进行设置：  <br/> |PSETID_Address  <br/> |
|长 ID （盖）：  <br/> |0x000080D6  <br/> |
|数据类型：  <br/> |PT_BOOLEAN  <br/> |
|区域：  <br/> |联系人  <br/> |
