# <a name="pidtagnonipmsubtreeentryid-canonical-property"></a>PidTagNonIpmSubtreeEntryId 规范属性

  
  
**适用于**： Outlook 
  
在 Exchange 公用文件夹存储区中显示的**NON_IPM_SUBTREE**文件夹的条目 ID。 
  
|||
|:-----|:-----|
|关联的属性：  <br/> |PR_NON_IPM_SUBTREE_ENTRYID  <br/> |
|标识符:  <br/> |0x6620  <br/> |
|数据类型：  <br/> |PT_BINARY  <br/> |
|区域：  <br/> |消息存储库  <br/> |
