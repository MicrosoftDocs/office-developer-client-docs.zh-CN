# <a name="pidtagrecipientstatus-canonical-property"></a>PidTagRecipientStatus 规范属性

  
  
**适用于**： Outlook 2013 |Outlook 2016 
  
包含一个值，它使用 MAPI 后台处理程序中分配传递责任之间传输提供程序。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_RECIPIENT_STATUS  <br/> |
|标识符：  <br/> |0x0E15  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |MAPI 非可传送  <br/> |
   
## <a name="remarks"></a>注解

不要使用此属性。 它是通过 MAPI 供使用。
