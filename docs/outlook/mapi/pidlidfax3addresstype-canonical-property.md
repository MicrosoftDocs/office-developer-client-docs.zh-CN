# <a name="pidlidfax3addresstype-canonical-property"></a>PidLidFax3AddressType 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
指定另一个联系人的传真地址的地址类型。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidFax3AddrType  <br/> |
|属性集：  <br/> |PSETID_Address  <br/> |
|LONG ID (的一) ：  <br/> |0x000080D2  <br/> |
|数据类型：  <br/> |PT_UNICODE  <br/> |
|区域：  <br/> |联系人  <br/> |
   
## <a name="remarks"></a>备注

此属性（如果存在）必须设置为"FAX"。
