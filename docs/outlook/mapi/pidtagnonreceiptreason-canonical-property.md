# <a name="pidtagnonreceiptreason-canonical-property"></a>PidTagNonReceiptReason 规范属性

  
  
**适用于**： Outlook 2013 |Outlook 2016 
  
包含未收到一条消息的原因的窗体未送达报告的一部分的原因。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_NON_RECEIPT_REASON  <br/> |
|标识符：  <br/> |0x003E  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |Exchange  <br/> |
   
## <a name="remarks"></a>注解

原因可以包括，例如，消息已被丢弃。
