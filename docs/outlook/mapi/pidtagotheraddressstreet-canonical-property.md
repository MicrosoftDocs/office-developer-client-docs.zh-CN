# <a name="pidtagotheraddressstreet-canonical-property"></a>PidTagOtherAddressStreet 规范属性

  
  
**适用于**： Outlook 
  
包含邮件用户的其他街道地址。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_OTHER_ADDRESS_STREET，PR_OTHER_ADDRESS_STREET_A，PR_OTHER_ADDRESS_STREET_W  <br/> |
|标识符：  <br/> |0x3A63  <br/> |
|数据类型：  <br/> |PT_UNICODE PT_STRING8  <br/> |
|区域：  <br/> |Address  <br/> |
   
## <a name="remarks"></a>说明

由用户或用户的组织定义这些属性。
