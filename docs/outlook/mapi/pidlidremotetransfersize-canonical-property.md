# <a name="pidlidremotetransfersize-canonical-property"></a>PidLidRemoteTransferSize 规范属性

  
  
**适用于**： Outlook 2013 |Outlook 2016 
  
指示的传输大小，以字节为单位的远程项目。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidRemoteXferSize  <br/> |
|属性进行设置：  <br/> |PSETID_Remote  <br/> |
|长 ID （盖）：  <br/> |0x00008F05  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |远程邮件  <br/> |
