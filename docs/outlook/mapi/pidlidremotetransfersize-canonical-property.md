# <a name="pidlidremotetransfersize-canonical-property"></a>PidLidRemoteTransferSize 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
指示远程项目的传输大小 (以字节为单位)。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidRemoteXferSize  <br/> |
|属性集:  <br/> |PSETID_Remote  <br/> |
|长 ID (盖子):  <br/> |0x00008F05  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |远程邮件  <br/> |
