# <a name="pidtagbirthday-canonical-property"></a>PidTagBirthday 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
包含联系人的生日。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_BIRTHDAY  <br/> |
|标识符：  <br/> |0x3A42  <br/> |
|数据类型：  <br/> |PT_SYSTIME  <br/> |
|区域：  <br/> |联系人  <br/> |
   
## <a name="remarks"></a>说明

此属性是提供标识和有关联系人的访问信息的属性之一。 由用户和用户的组织定义这些属性。
