# <a name="operating-environment-issues"></a>操作环境问题

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
当 MAPI 开发人员选择开发环境和 Windows SDK 安装环境时, 操作系统和操作环境的问题非常重要。
  
MAPI 开发人员必须知道, 这些选择会影响其组件将在其上运行的操作系统以及它们将能够定位的组件范围。
