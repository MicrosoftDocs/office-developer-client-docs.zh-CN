# <a name="cbnewssortorderset"></a>CbNewSSortOrderSet

  
  
**适用于**： Outlook 2013 |Outlook 2016 
  
计算要为新的[SizedSSortOrderSet](sizedssortorderset.md)结构，其中包含指定的数目的排序次序由[SSortOrder](ssortorder.md)结构分配的字节数。 
  
|||
|:-----|:-----|
|头文件：  <br/> |Mapidefs.h  <br/> |
|相关的结构：  <br/> |**SSortOrderSet** <br/> |
   
```cpp
CbNewSSortOrderSet (_csort)
```

## <a name="parameters"></a>参数

 __csort_
  
> 若要包括在**SSortOrderSet**结构**SSortOrder**结构的计数。
