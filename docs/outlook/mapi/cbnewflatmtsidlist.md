# <a name="cbnewflatmtsidlist"></a>CbNewFLATMTSIDLIST

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
计算应为包含多个指定大小的[MTSID](mtsid.md)结构的新[FLATMTSIDLIST](flatmtsidlist.md)结构分配的字节数。 
  
|||
|:-----|:-----|
|标头文件：  <br/> |mapidefs。h  <br/> |
|相关结构:  <br/> |**FLATMTSIDLIST** <br/> |
   
```cpp
CbNewFLATMTSIDIDLIST (_cb)
```

## <a name="parameters"></a>参数

 __cb_
  
> 要包含在新**FLATMTSIDLIST**结构中的**MTSID**结构中的字节数。
