# <a name="mapi-form-libraries"></a>MAPI 表单库

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
为了帮助组织服务器, 并使其易于提供给客户端应用程序, MAPI 表单体系结构包括支持安装、管理、复制和使用表单服务器的表单库。
  
表单库是表单服务器的存储库以及表单管理器在加载表单服务器时所使用的配置信息。 
  
> [!NOTE]
> 在 MAPI 历史记录的早期, 表单库称为 "注册表"。 因此, 术语 "注册表" 在一些与窗体相关的 MAPI 接口、属性、配置文件条目等中保持不存在。 如果应用程序的接口需要使用其中一个术语, 则应以独占方式使用 "library" 一词。
