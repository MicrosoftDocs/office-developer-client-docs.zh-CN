# <a name="mapi-form-libraries"></a>MAPI 表单库

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
为了帮助组织表单服务器并使它们易于客户端应用程序使用，MAPI 表单体系结构包括表单库，这些库支持表单服务器的安装、管理、复制和使用。
  
表单库是表单服务器的存储库以及表单管理器在加载表单服务器时使用的有关这些服务器的配置信息。 
  
> [!NOTE]
> 在 MAPI 的早期历史记录中，表单库称为"注册表"。 因此，术语"注册表"一直保留在某些与表单相关的 MAPI 接口、属性、配置文件条目等中。 如果应用程序的接口需要使用其中一个术语，您应该专门使用术语"library"。
