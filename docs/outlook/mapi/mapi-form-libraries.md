# <a name="mapi-form-libraries"></a>MAPI 表单库

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
为了帮助组织窗体服务器并使其更易于使用客户端应用程序，MAPI 表单体系结构包括支持的安装、 管理、 复制和使用表单服务器表单库。
  
表单库是存储库的窗体服务器和有关它们在加载窗体服务器时所使用的窗体管理器的配置信息。 
  
> [!NOTE]
> MAPI 的历史记录，在早期表单库被称为"注册表"。 因此，术语"注册表"具有保持在某些表单相关 MAPI 接口、 属性、 配置文件条目等。 如果应用程序的接口要求使用这些词之一，则应该以独占方式使用术语"库"。
