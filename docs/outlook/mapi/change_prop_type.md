# <a name="changeproptype"></a>CHANGE_PROP_TYPE

  
  
**适用于**： Outlook 
  
更新属性类型属性标记为指定的值。 属性标识符不变。 
  
|||
|:-----|:-----|
|头文件：  <br/> |Mapidefs.h  <br/> |
|相关的结构：  <br/> |[SPropValue](spropvalue.md) <br/> |
   
```cpp
CHANGE_PROP_TYPE (ulPropTag, ulPropType)
```

## <a name="parameters"></a>参数

 _ulPropTag_
  
> 要修改的属性标记。
    
 _ulPropType_
  
> 属性类型的新值。
