# <a name="expanding-distribution-lists"></a>展开通讯组列表

  
  
**适用于**： Outlook 
  
 **提示 MAPI 以展开通讯组列表**
  
- 设置为 MAPIPDL 其**PR_ADDRTYPE** ([PidTagAddressType](pidtagaddresstype-canonical-property.md)) 属性。
    
    MAPI 将消息发送到传输提供程序之前展开与此类型的地址。
