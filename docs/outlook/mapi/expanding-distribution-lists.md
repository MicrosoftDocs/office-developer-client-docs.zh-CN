# <a name="expanding-distribution-lists"></a>展开通讯组列表

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
 **提示 MAPI 展开通讯组列表**
  
- 将其**PR_ADDRTYPE** ([PidTagAddressType](pidtagaddresstype-canonical-property.md)) 属性设置为 MAPIPDL。
    
    MAPI 在将邮件发送到传输提供程序之前, 使用此类型展开地址。
