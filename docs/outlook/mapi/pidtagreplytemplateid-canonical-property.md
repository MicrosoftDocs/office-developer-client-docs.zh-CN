# <a name="pidtagreplytemplateid-canonical-property"></a>PidTagReplyTemplateId 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
包含答复模板的全局唯一标识符 (GUID)。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_REPLY_TEMPLATE_ID  <br/> |
|标识符：  <br/> |0x65C2  <br/> |
|数据类型：  <br/> |PT_BINARY  <br/> |
|区域：  <br/> |Rules  <br/> |
