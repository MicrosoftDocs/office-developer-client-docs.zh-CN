# <a name="pidtaglastverbexecutiontime-canonical-property"></a>PidTagLastVerbExecutionTime 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
包含执行的最后一个谓词的时间的时间。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_LAST_VERB_EXECUTION_TIME  <br/> |
|标识符：  <br/> |0x1082  <br/> |
|数据类型：  <br/> |PT_SYSTIME  <br/> |
|区域：  <br/> |历史记录  <br/> |
