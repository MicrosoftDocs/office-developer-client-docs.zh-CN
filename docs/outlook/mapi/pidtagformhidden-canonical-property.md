# <a name="pidtagformhidden-canonical-property"></a>PidTagFormHidden 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
如果要通过撰写菜单和对话框禁止显示窗体, 则该参数为 TRUE。 
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_FORM_HIDDEN  <br/> |
|标识符:  <br/> |0x3307  <br/> |
|数据类型：  <br/> |PT_BOOLEAN  <br/> |
|区域：  <br/> |MAPI 通用  <br/> |
   
## <a name="remarks"></a>注解

与窗体相关的属性是只读的。
