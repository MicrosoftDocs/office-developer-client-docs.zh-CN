# <a name="pidlidheaderitem-canonical-property"></a>PidLidHeaderItem 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
指定在缓存 Exchange 模式下配置 Outlook 时, Microsoft outlook 是否仅将邮件头或邮件的头和正文下载到客户端计算机。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidHeaderItem  <br/> |
|属性集:  <br/> |PSETID_Common  <br/> |
|长 ID (盖子):  <br/> |0x00008578  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |常规邮件  <br/> |
