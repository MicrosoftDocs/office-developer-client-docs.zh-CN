# <a name="pidlidnotex-canonical-property"></a>PidLidNoteX 规范属性

  
  
**适用于**： Outlook 2013 |Outlook 2016 
  
指定以像素为单位，从用户界面显示一条注释的位置屏幕左边缘的距离。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidNoteX  <br/> |
|属性进行设置：  <br/> |PSETID_Note  <br/> |
|长 ID （盖）：  <br/> |0x00008B04  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |粘滞便笺  <br/> |
