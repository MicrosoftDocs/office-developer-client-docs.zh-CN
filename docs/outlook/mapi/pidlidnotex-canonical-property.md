# <a name="pidlidnotex-canonical-property"></a>PidLidNoteX 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
指定从屏幕左边缘到用户界面显示注释的距离（以像素为单位）。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidNoteX  <br/> |
|属性集：  <br/> |PSETID_Note  <br/> |
|LONG ID (的一) ：  <br/> |0x00008B04  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |便笺  <br/> |
