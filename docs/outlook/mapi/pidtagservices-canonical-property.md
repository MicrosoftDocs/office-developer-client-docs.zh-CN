# <a name="pidtagservices-canonical-property"></a>PidTagServices 规范属性

  
  
**适用于**： Outlook 
  
包含当前配置文件中的消息服务的标识符列表。
  
|||
|:-----|:-----|
|关联的属性：  <br/> |PR_SERVICES  <br/> |
|标识符:  <br/> |0x3D0E  <br/> |
|数据类型：  <br/> |PT_BINARY  <br/> |
|区域：  <br/> |MAPI 配置文件  <br/> |
   
## <a name="remarks"></a>备注

不要使用此属性。 它是通过 MAPI 供使用。
