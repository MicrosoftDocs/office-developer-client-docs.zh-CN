# <a name="pidtagdelegatedbyrule-canonical-property"></a>PidTagDelegatedByRule 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
指示文件夹的邮件是否由规则委派。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_DELEGATED_BY_RULE  <br/> |
|标识符:  <br/> |0x3FE3  <br/> |
|数据类型：  <br/> |PT_BOOLEAN  <br/> |
|区域：  <br/> |MAPI 状态  <br/> |
