# <a name="pidtagjunkpermanentlydelete-canonical-property"></a>PidTagJunkPermanentlyDelete 规范属性

  
  
**适用于**： Outlook 2013 |Outlook 2016 
  
表示，如果设为 1，邮件标识为垃圾邮件可能被永久删除。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_JUNK_PERMANENTLY_DELETE  <br/> |
|标识符：  <br/> |0x6102  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |垃圾邮件  <br/> |
