# <a name="pidtagformdesignername-canonical-property"></a>PidTagFormDesignerName 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
包含用于设计窗体的对象的显示名称。 
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_FORM_DESIGNER_NAME、PR_FORM_DESIGNER_NAME_A、PR_FORM_DESIGNER_NAME_W  <br/> |
|标识符:  <br/> |0x3308  <br/> |
|数据类型：  <br/> |PT_STRING8、PT_UNICODE  <br/> |
|区域：  <br/> |MAPI 通用  <br/> |
   
## <a name="remarks"></a>注解

这些属性包含表单设计器对象的唯一标识符。
