# <a name="mapi-properties"></a>MAPI 属性

 
  
**适用于**： Outlook 2013 |Outlook 2016 
  
此部分包含规范支持通过 Microsoft Outlook 消息处理 API (MAPI) 的属性。 每个规范属性对应于一个或多个相关的 MAPI 属性。 规范属性的详细信息，请参阅[MAPI 规范属性](mapi-canonical-properties.md)。 MAPI 属性以及每个对象可用的属性列表中的工作方式的一般说明，请参阅[MAPI 属性概述](mapi-property-overview.md)。 有关的属性相关的宏的信息，请参阅[SPropValue](spropvalue.md)结构参考页。 MAPI 属性和其映射到规范属性名称的完整列表，请参阅[规范属性名称映射 MAPI 名称](mapping-mapi-names-to-canonical-property-names.md)和[MAPI 名称映射规范属性名称](mapping-canonical-property-names-to-mapi-names.md)。 
  
注意： 与 OLE variant 数组的成员，不同 MAPI 多值的属性数组中每个成员都是相同的类型。
