# <a name="pidtaglocaleid-canonical-property"></a>PidTagLocaleId 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
建立用于转换系统生成的邮件的语言区域设置, 例如传递报告。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_LOCALE_ID  <br/> |
|标识符:  <br/> |0x66A1  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |其他  <br/> |
   
## <a name="remarks"></a>注解

此属性在专用邮箱登录上可用。
