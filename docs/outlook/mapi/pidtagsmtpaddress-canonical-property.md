# <a name="pidtagsmtpaddress-canonical-property"></a>PidTagSmtpAddress 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
包含通讯簿对象的 SMTP 地址。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_SMTP_ADDRESS、PR_SMTP_ADDRESS_A、PR_SMTP_ADDRESS_W  <br/> |
|标识符:  <br/> |0x39FE  <br/> |
|数据类型：  <br/> |PT_UNICODE、PT_STRING8  <br/> |
|区域：  <br/> |地址  <br/> |
