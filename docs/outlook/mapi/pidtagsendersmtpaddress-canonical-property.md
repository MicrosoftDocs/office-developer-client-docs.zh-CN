# <a name="pidtagsendersmtpaddress-canonical-property"></a>PidTagSenderSmtpAddress 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
包含发送邮箱所有者的简单邮件传输协议 (SMTP) 电子邮件地址的格式。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_SENDER_SMTP_ADDRESS，PR_SENDER_SMTP_ADDRESS_A，PR_SENDER_SMTP_ADDRESS_W  <br/> |
|标识符：  <br/> |0x5D01  <br/> |
|数据类型：  <br/> |PT_UNICODE PT_STRING8  <br/> |
|区域：  <br/> |Address  <br/> |
   
## <a name="remarks"></a>说明

此属性是个发件人的地址属性的示例。 它必须由传出的传输提供程序，应永远不会传播任何以前现有值设置。
  
如果没有传输提供程序具有提供任何发件人地址属性，MAPI 后台处理程序尝试填写通过调用[IMAPISession::QueryIdentity](imapisession-queryidentity.md)方法的项标识符。 如果不提供了任何条目标识符，MAPI 后台处理程序中所有发件人地址类型的属性的 PT_UNICODE 或 PT_STRING8 存储"Unknown"。
