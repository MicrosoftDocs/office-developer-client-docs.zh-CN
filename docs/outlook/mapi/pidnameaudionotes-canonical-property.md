# <a name="pidnameaudionotes-canonical-property"></a>PidNameAudioNotes 规范属性

  
  
**适用于**： Outlook 
  
指定附加到语音邮件的文本说明。
  
|||
|:-----|:-----|
|友好名称：  <br/> |UMAudioNotes  <br/> |
|属性进行设置：  <br/> |PSETID_UnifiedMessaging  <br/> |
|属性名称：  <br/> |UMAudioNotes  <br/> |
|数据类型：  <br/> |PT_UNICODE  <br/> |
|区域：  <br/> |统一消息  <br/> |
   
## <a name="remarks"></a>说明

若要启用最终用户可以读取和编辑直接在语音邮件上的音频笔记，客户端提供用户可以在其中键入注释添加到语音邮件对象的此属性的一组编辑框。
