# <a name="pidtagrecipientnumber-canonical-property"></a>PidTagRecipientNumber 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
代表邮件上的收件人的收件人号码。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_RECIPIENT_NUMBER  <br/> |
|标识符:  <br/> |0x6662  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |Exchange 邮件读/写  <br/> |
