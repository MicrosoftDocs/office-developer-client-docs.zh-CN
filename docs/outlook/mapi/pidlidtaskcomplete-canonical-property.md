# <a name="pidlidtaskcomplete-canonical-property"></a>PidLidTaskComplete 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
指示任务已完成。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidTaskComplete  <br/> |
