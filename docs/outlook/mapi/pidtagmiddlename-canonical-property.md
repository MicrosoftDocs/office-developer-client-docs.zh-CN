# <a name="pidtagmiddlename-canonical-property"></a>PidTagMiddleName 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
包含联系人的中间名。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_MIDDLE_NAME、PR_MIDDLE_NAME_A、PR_MIDDLE_NAME_W  <br/> |
|标识符:  <br/> |0x3A44  <br/> |
|数据类型：  <br/> |PT_UNICODE、PT_STRING8  <br/> |
|区域：  <br/> |Address  <br/> |
   
## <a name="remarks"></a>注解

这些属性由用户定义。
