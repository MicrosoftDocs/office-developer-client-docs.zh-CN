# <a name="pidtagipmreturnrequested-canonical-property"></a>PidTagIpmReturnRequested 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
如果此消息应随报告一起返回, 则包含 TRUE。
