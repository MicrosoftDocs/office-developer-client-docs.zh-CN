# <a name="pidlidrecalltime-canonical-property"></a>PidLidRecallTime 规范属性

  
  
**适用于**： Outlook 2013 |Outlook 2016 
  
表示邮件撤回启动频率的时间。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidRecallTime  <br/> |
|属性进行设置：  <br/> |PSETID_Common  <br/> |
|长 ID （盖）：  <br/> |0x00008549  <br/> |
|数据类型：  <br/> |PT_SYSTIME  <br/> |
|区域：  <br/> |常规消息  <br/> |
