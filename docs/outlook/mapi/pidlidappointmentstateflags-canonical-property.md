# <a name="pidlidappointmentstateflags-canonical-property"></a>PidLidAppointmentStateFlags 规范属性

  
  
**适用于**： Outlook 
  
指定的描述对象的状态位字段。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidApptStateFlags  <br/> |
|属性进行设置：  <br/> |PSETID_Appointment  <br/> |
|长 ID （盖）：  <br/> |0x00008217  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |会议  <br/> |
   
## <a name="remarks"></a>说明

此属性不是必需的。 下面是可以设置的单个标记。
  
M (asfMeeting，0x00000001)
  
> 此标志指示对象是会议对象或会议相关对象。
    
R (asfReceived，0x00000002:uc)
  
> 此标志指示指定的对象已收到来自其他人。
    
C (asfCanceled，0x00000004)
  
> 此标志指示对象所代表的会议对象已被取消。
