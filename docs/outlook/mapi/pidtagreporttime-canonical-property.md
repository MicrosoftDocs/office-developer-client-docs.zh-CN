# <a name="pidtagreporttime-canonical-property"></a>PidTagReportTime 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
包含邮件系统生成报告的日期和时间。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_REPORT_TIME  <br/> |
|标识符:  <br/> |0x0032  <br/> |
|数据类型：  <br/> |PT_SYSTIME  <br/> |
|区域：  <br/> |MAPI 信封  <br/> |
   
## <a name="remarks"></a>注解

此属性表示传递和 nondelivery 报告上的每个收件人的属性, 以及 read 和未读报告上的每个邮件的属性。
