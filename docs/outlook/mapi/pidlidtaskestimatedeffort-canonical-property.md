# <a name="pidlidtaskestimatedeffort-canonical-property"></a>PidLidTaskEstimatedEffort 规范属性

  
  
**适用于**： Outlook 
  
指示的时间，以分钟为单位，用户需要执行的任务。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidTaskEstimatedEffort  <br/> |
|属性进行设置：  <br/> |PSETID_Task  <br/> |
|长 ID （盖）：  <br/> |0x00008111  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |Task  <br/> |
   
## <a name="remarks"></a>说明

值必须是大于或等于 0 和小于 0x5AE980DF (1,525,252,319)，其中 480 分钟等于一个天和 2400 分钟等于一周 （在工作日和工作周中的五天 8 小时）。
