# <a name="cbnewentryid"></a>CbNewENTRYID

  
  
**适用于**： Outlook 
  
计算指定大小的新[ENTRYID](entryid.md)结构应分配的字节数。 
  
|||
|:-----|:-----|
|头文件：  <br/> |Mapidefs.h  <br/> |
|相关的结构：  <br/> |**ENTRYID** <br/> |
   
```cpp
CbNewENTRYID (_cb)
```

## <a name="parameters"></a>参数

 __cb_
  
> 存储中的新的**ENTRYID**结构中的项标识符的字节数。
