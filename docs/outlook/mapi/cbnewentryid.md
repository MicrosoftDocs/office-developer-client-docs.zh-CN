# <a name="cbnewentryid"></a>CbNewENTRYID

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
计算应为指定大小的新 [ENTRYID](entryid.md) 结构分配的字节数。 
  
|||
|:-----|:-----|
|标头文件：  <br/> |Mapidefs.h  <br/> |
|相关结构：  <br/> |**ENTRYID** <br/> |
   
```cpp
CbNewENTRYID (_cb)
```

## <a name="parameters"></a>参数

 _ _cb_
  
> 要存储在新 **ENTRYID** 结构的条目标识符中的字节数。
