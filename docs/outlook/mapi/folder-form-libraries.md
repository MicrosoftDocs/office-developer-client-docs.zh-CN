# <a name="folder-form-libraries"></a>文件夹表单库

  
  
**适用于**： Outlook 
  
在某些情况下，您可能想要将一个或多个窗体与特定文件夹相关联。 例如，您的组织中的员工无法所有具有进度报告文件夹创建和存储进度报告其个人消息存储中。 进度报告是特定于每个用户的进度报告文件夹，因为它可能不适合在系统范围的窗体库中存储的进度报告窗体。 但是，可以关联内容表中的每个用户的进度报告文件夹中保留的进度报告窗体副本。 此限制将用户指定的文件夹之外使用进度报告表单。
  
从概念上讲，没有一个文件夹表单库的每个文件夹中的消息存储，即使没有窗体服务器安装中。 文件夹表单库实现像其他窗体库 — 它们存储为文件夹的备用部件中的关联内容表。 因为文件夹表单库包含的文件夹中，将其复制以及其父文件夹中复制操作。
