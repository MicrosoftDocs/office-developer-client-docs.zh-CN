# <a name="pidnamexsharingremotename-canonical-property"></a>PidNameXSharingRemoteName 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
表示 **dispidSharingRemoteName** ([PidLidSharingRemoteName](pidlidsharingremotename-canonical-property.md)) 属性的值。
  
|||
|:-----|:-----|
|友好名称：  <br/> |无  <br/> |
|属性集：  <br/> |PS_INTERNET_HEADERS  <br/> |
|属性名称：  <br/> |X-Sharing-Remote-Name  <br/> |
|数据类型：  <br/> |PT_UNICODE  <br/> |
|区域：  <br/> |共享  <br/> |
