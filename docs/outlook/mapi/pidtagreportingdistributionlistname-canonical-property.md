# <a name="pidtagreportingdistributionlistname-canonical-property"></a>PidTagReportingDistributionListName 规范属性

  
  
**适用于**： Outlook 
  
包含其中消息系统提供报表的通讯组列表的显示名称。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_REPORTING_DL_NAME  <br/> |
|标识符：  <br/> |0x1003  <br/> |
|数据类型：  <br/> |PT_BINARY  <br/> |
|区域：  <br/> |MAPI 邮件  <br/> |
