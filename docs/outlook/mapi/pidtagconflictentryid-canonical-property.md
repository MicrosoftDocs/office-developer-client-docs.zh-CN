# <a name="pidtagconflictentryid-canonical-property"></a>PidTagConflictEntryId 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
包含冲突解决消息的 EntryID。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_CONFLICT_ENTRYID  <br/> |
|标识符：  <br/> |0x3FF0  <br/> |
|数据类型：  <br/> |PT_BINARY  <br/> |
|区域：  <br/> |增量更改同步 (ICS)  <br/> |
