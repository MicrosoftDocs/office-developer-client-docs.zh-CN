# <a name="pidtagattachmenthidden-canonical-property"></a>PidTagAttachmentHidden 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
指示最终用户是否隐藏附件。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_ATTACHMENT_HIDDEN  <br/> |
|标识符:  <br/> |0x7FFE  <br/> |
|数据类型：  <br/> |PT_BOOLEAN  <br/> |
|区域：  <br/> |邮件附件  <br/> |
