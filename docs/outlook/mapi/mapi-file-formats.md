# <a name="mapi-file-formats"></a>MAPI 文件格式

  
  
**适用于**： Outlook 2013 |Outlook 2016 
  
MAPI 发布各种客户端和服务提供商使用描述消息服务及其服务提供商和窗体的配置文件的格式。
  
下表介绍了这些文件按名称或类型：
  
|**配置文件**|**说明**|
|:-----|:-----|
|MapiSvc.inf  <br/> |包含 MAPI 子系统、 消息服务和服务提供商的配置信息。  <br/> |
|具有.cfg 扩展名的文件  <br/> |包含窗体的配置信息。  <br/> |
   
有关的 MapiSvc.inf 文件格式的详细信息，请参阅[的 MapiSvc.inf 文件格式](file-format-of-mapisvc-inf.md)。 
  
窗体配置文件的文件格式的详细信息，请参阅[窗体配置文件的文件格式](file-format-of-form-configuration-files.md)。
