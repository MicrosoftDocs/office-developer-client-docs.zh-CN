# <a name="pidtagexceptionstarttime-canonical-property"></a>PidTagExceptionStartTime 规范属性

  
  
**适用于**： Outlook 
  
指示的开始日期和时间的计算机的本地时区异常时创建例外。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_EXCEPTION_STARTTIME  <br/> |

|数据类型：  <br/> |PT_SYSTIME  <br/> |
