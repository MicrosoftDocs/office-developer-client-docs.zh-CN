# <a name="pidtaginitialdetailspane-canonical-property"></a>PidTagInitialDetailsPane 规范属性

  
  
**适用于**： Outlook 
  
指示显示模板以显示第一页。
  
|||
|:-----|:-----|
|关联的属性：  <br/> |PR_INITIAL_DETAILS_PANE  <br/> |
|标识符:  <br/> |0x3F08  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |MAPI 显示表  <br/> |
   
## <a name="remarks"></a>备注

它必须存在所有通讯簿对象的名称服务提供程序界面 (NSPI) 服务器上，并且必须具有的值为零 (0)。 它必须未定义脱机通讯簿中的任何对象。
