# <a name="pidlidclassification-canonical-property"></a>PidLidClassification 规范属性

  
  
**适用于**： Outlook 2013 |Outlook 2016 
  
指示邮件分类的名称。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidClassification  <br/> |
|属性进行设置：  <br/> |PSETID_Common  <br/> |

|数据类型：  <br/> |PT_UNICODE  <br/> |
