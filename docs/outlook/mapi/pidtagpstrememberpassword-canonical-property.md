# <a name="pidtagpstrememberpassword-canonical-property"></a>PidTagPstRememberPassword 规范属性

  
  
**适用于**： Outlook 
  
确定是否要记住的配置文件中的密码。
  
|||
|:-----|:-----|
|关联的属性：  <br/> |PR_PST_REMEMBER_PW  <br/> |
|标识符:  <br/> |0x6701  <br/> |
|数据类型：  <br/> |PT_BOOLEAN  <br/> |
|区域：  <br/> |个人存储表 (.pst) 内部  <br/> |
