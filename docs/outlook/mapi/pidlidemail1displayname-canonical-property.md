# <a name="pidlidemail1displayname-canonical-property"></a>PidLidEmail1DisplayName 规范属性

  
  
**适用于**： Outlook 
  
指定用户可读的显示名称的第一个电子邮件地址。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidEmail1DisplayName  <br/> |
|属性进行设置：  <br/> |PSETID_Address  <br/> |
|长 ID （盖）：  <br/> |0x00008080  <br/> |
|数据类型：  <br/> |PT_UNICODE  <br/> |
|区域：  <br/> |联系人  <br/> |
