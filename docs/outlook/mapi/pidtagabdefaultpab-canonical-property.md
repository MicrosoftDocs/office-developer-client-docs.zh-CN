# <a name="pidtagabdefaultpab-canonical-property"></a>PidTagAbDefaultPab 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
包含要用作个人通讯簿 (PAB) 的通讯簿容器的条目标识符。 
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_AB_DEFAULT_PAB  <br/> |
|标识符:  <br/> |0x3D07  <br/> |
|数据类型：  <br/> |PT_BINARY  <br/> |
|区域：  <br/> |通讯簿  <br/> |
   
## <a name="remarks"></a>注解

不要使用此属性。 它保留供 MAPI 使用。
