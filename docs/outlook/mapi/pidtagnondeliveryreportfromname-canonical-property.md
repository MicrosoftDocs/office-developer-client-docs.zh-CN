# <a name="pidtagnondeliveryreportfromname-canonical-property"></a>PidTagNonDeliveryReportFromName 规范属性

  
  
**适用于**： Outlook 2013 |Outlook 2016 
  
如果原件报表上设置，则表示失败发送的名称-用户。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_NDR_FROM_NAME  <br/> |
|标识符：  <br/> |0x6905  <br/> |
|数据类型：  <br/> |PT_STRING8  <br/> |
|区域：  <br/> |常规消息  <br/> |
