# <a name="pidlidfax2richtextformat-canonical-property"></a>PidLidFax2RichTextFormat 规范属性

  
  
**适用于**： Outlook 2013 |Outlook 2016 
  
指定主联系人的传真地址的格式文本的格式。 
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidFax2RTF  <br/> |
|属性进行设置：  <br/> |PSETID_Address  <br/> |
|长 ID （盖）：  <br/> |0x000080C6  <br/> |
|数据类型：  <br/> |PT_BOOLEAN  <br/> |
|区域：  <br/> |联系人  <br/> |
