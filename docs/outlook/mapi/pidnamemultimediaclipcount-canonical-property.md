# <a name="pidnamemultimediaclipcount-canonical-property"></a>PidNameMultimediaClipCount 规范属性

  
  
**适用于**： Outlook 
  
指定文件的附加到文档中的多媒体剪辑数。
  
|||
|:-----|:-----|
|友好名称：  <br/> |无  <br/> |
|属性进行设置：  <br/> |PS_PUBLIC_STRINGS  <br/> |
|属性名称：  <br/> |MMClipCount  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |Common  <br/> |
