# <a name="pidtagcontactaddressbooksortflag-canonical-property"></a>PidTagContactAddressBookSortFlag 规范属性

  
  
**适用于**： Outlook 
  
包含一个时，应按文件下和 FALSE 排序联系人，他们应按显示名称排序时为 TRUE 的标志。 
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_CONTAB_SORT_FLAG  <br/> |
|标识符：  <br/> |0x6602  <br/> |
|数据类型：  <br/> |PT_BOOLEAN  <br/> |
|区域：  <br/> |联系人通讯簿  <br/> |
   
## <a name="remarks"></a>说明

这是联系人通讯簿配置文件一节的属性。
