# <a name="pidlidclassified-canonical-property"></a>PidLidClassified 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
指示邮件已分类。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidClassified  <br/> |
|属性集：  <br/> |PSETID_Common  <br/> |
|LONG ID (的一) ：  <br/> |0x000085B5  <br/> |
|数据类型：  <br/> |PT_BOOLEAN  <br/> |
|区域：  <br/> |常规消息  <br/> |
