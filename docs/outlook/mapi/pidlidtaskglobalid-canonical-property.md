# <a name="pidlidtaskglobalid-canonical-property"></a>PidLidTaskGlobalId 规范属性

  
  
**适用于**： Outlook 
  
查找现有任务，使用在收到任务响应或任务更新后的 GUID。
  
|||
|:-----|:-----|
|关联的属性：  <br/> |dispidTaskGlobalObjId  <br/> |
|属性进行设置：  <br/> |PSETID_Common  <br/> |
|长 ID （盖）：  <br/> |0x00008519  <br/> |
|数据类型：  <br/> |PT_BINARY  <br/> |
|区域：  <br/> |任务  <br/> |
   
## <a name="remarks"></a>备注

此属性保留未分配的任务未设置。
