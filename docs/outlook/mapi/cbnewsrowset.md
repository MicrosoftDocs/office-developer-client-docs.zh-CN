# <a name="cbnewsrowset"></a>CbNewSRowSet

  
  
**适用于**： Outlook 
  
计算要分配给新[SRowSet](srowset.md)结构，其中包含指定由[SRow](srow.md)结构的行数的字节数。 
  
|||
|:-----|:-----|
|头文件：  <br/> |Mapidefs.h  <br/> |
|相关的结构：  <br/> |**SRowSet** <br/> |
   
```cpp
CbNewSRowSet (_crow)
```

## <a name="parameters"></a>参数

 __crow_
  
> 要包含在新的**SRowSet**结构**SRow**结构指示的行数的计数。
