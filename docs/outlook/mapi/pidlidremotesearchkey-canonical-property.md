# <a name="pidlidremotesearchkey-canonical-property"></a>PidLidRemoteSearchKey 规范属性

  
  
**适用于**： Outlook 
  
指示搜索关键字的远程项目。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidRemoteSearchKey  <br/> |
|属性进行设置：  <br/> |PSETID_Remote  <br/> |
|长 ID （盖）：  <br/> |0x00008F06  <br/> |
|数据类型：  <br/> |PT_BINARY  <br/> |
|区域：  <br/> |远程邮件  <br/> |
