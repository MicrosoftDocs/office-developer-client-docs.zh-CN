# <a name="pidlidnoteheight-canonical-property"></a>PidLidNoteHeight 规范属性

  
  
**适用于**： Outlook 2013 |Outlook 2016 
  
以像素为单位指定显示一条消息窗口的高度。 
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidNoteHeight  <br/> |
|属性进行设置：  <br/> |PSETID_Note  <br/> |
|长 ID （盖）：  <br/> |0x00008B03  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |粘滞便笺  <br/> |
   
## <a name="remarks"></a>注解

此值必须大于零。
