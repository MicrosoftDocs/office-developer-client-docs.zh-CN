# <a name="pidtagcontactaddressbookstoreentryids-canonical-property"></a>PidTagContactAddressBookStoreEntryIds 规范属性

  
  
**适用于**： Outlook 
  
包含充当联系人地址的文件夹中找到簿条目存储的条目 Id。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_CONTAB_STORE_ENTRYIDS  <br/> |
|标识符：  <br/> |0x6626  <br/> |
|数据类型：  <br/> |PT_MV_BINARY  <br/> |
|区域：  <br/> |联系人通讯簿  <br/> |
   
## <a name="remarks"></a>说明

这是联系人通讯簿配置文件一节的属性。
