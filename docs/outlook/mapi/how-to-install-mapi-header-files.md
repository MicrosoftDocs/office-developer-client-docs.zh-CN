# <a name="install-mapi-header-files"></a>安装 MAPI 头文件

**适用于**：Outlook 2013 | Outlook 2016 
  
本主题介绍如何获取和安装 MAPI 头文件和库文件。

MAPI 头文件可为以下位置下载： [Outlook 2010: MAPI 头文件](https://www.microsoft.com/downloads/details.aspx?FamilyID=f8d01fc8-f7b5-4228-baa3-817488a66db1)。 默认情况下，MAPI 头文件安装在以下文件夹： C:\Program Files\Microsoft SDKs\Office\14.0\Include。
  
MAPI 头文件的列表，请参阅[通常使用头文件](commonly-used-header-files.md)。
