# <a name="install-mapi-header-files"></a>安装 MAPI 头文件

**适用于**：Outlook 2013 | Outlook 2016 
  
此话题介绍了如何获取和安装 MAPI 头文件和库文件。

MAPI 头文件可在 [Outlook 2010：MAPI 头文件](https://www.microsoft.com/download/details.aspx?id=12905)中下载。 默认情况下，MAPI 头文件安装在以下文件夹中：C:\Program Files\Microsoft SDKs\Office\14.0\Include。
  
有关 MAPI 头文件的列表，请参阅[常用头文件](commonly-used-header-files.md)。
