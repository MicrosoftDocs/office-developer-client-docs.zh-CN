# <a name="pidnamecharactercount-canonical-property"></a>PidNameCharacterCount 规范属性

  
  
**适用于**： Outlook 
  
指定附加到文档的文件的字符数。
  
|||
|:-----|:-----|
|友好名称：  <br/> |无  <br/> |
|属性进行设置：  <br/> |PS_PUBLIC_STRINGS  <br/> |
|属性名称：  <br/> |CharCount  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |Common  <br/> |
