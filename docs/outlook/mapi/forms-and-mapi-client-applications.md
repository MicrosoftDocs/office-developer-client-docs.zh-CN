# <a name="forms-and-mapi-client-applications"></a>表单和 MAPI 客户端应用程序

**适用于**：Outlook 2013 | Outlook 2016 
  
由于所有 MAPI 消息都是表单的实例，因此客户端应用程序可以像显示其他邮件对象一样显示表单对象并与之交互。
  
使用 MAPI 表单接口的客户端应用程序可以提供与表单对象的其他交互。 客户端应用程序可以使用由表单定义的任何特殊动词，为用户提供表单设计器要执行的完整交互范围。 客户端还可使用表单服务器公开的任何自定义属性来帮助用户在客户端界面中排序、预览或以其他方式管理表单。
  
有关详细信息，请参阅处理 [MAPI 窗体](handling-mapi-forms.md)。
