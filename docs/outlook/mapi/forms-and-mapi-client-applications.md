# <a name="forms-and-mapi-client-applications"></a>表单和 MAPI 客户端应用程序

**适用于**：Outlook 2013 | Outlook 2016 
  
由于所有 MAPI 邮件都是表单的实例, 因此客户端应用程序会像处理其他邮件对象一样, 显示和交互处理表单对象。
  
使用 MAPI 表单接口的客户端应用程序可以提供与表单对象的其他交互。 客户端应用程序可以使用由窗体定义的任何特殊谓词向用户提供窗体设计器所需的全部交互。 客户端还可以使用表单服务器公开的任何自定义属性, 以帮助用户在客户端界面中进行排序、预览或以其他方式管理表单。
  
有关详细信息, 请参阅[处理 MAPI 表单](handling-mapi-forms.md)。
