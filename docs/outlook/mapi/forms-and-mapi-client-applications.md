# <a name="forms-and-mapi-client-applications"></a>窗体和 MAPI 客户端应用程序

**适用于**： Outlook 
  
因为所有的 MAPI 邮件窗体的实例，客户端应用程序显示并与其交互表单对象，就像与其他消息对象。
  
使用 MAPI 表单接口的客户端应用程序可以提供与表单对象的其他交互。 客户端应用程序可以使用由表单定义任何特殊动词为用户提供的交互非常适用于窗体的设计器的全部。 客户端还可以使用窗体服务器公开可帮助用户排序、 预览，或以其他方式管理客户端的界面中的窗体的任何自定义属性。
  
有关详细信息，请参阅[处理 MAPI 表单](handling-mapi-forms.md)。
