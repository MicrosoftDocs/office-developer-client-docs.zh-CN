# <a name="pidlidfax3displayname-canonical-property"></a>PidLidFax3DisplayName 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
指定联系人的显示名称的其他传真地址。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidFax3DisplayName  <br/> |
|属性进行设置：  <br/> |PSETID_Address  <br/> |
|长 ID （盖）：  <br/> |0x000080D0  <br/> |
|数据类型：  <br/> |PT_UNICODE  <br/> |
|区域：  <br/> |联系人  <br/> |
