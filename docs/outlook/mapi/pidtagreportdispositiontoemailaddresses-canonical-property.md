# <a name="pidtagreportdispositiontoemailaddresses-canonical-property"></a>PidTagReportDispositionToEmailAddresses 规范属性

  
  
**适用于**： Outlook 
  
指示请求收款的邮件的报告收件人的电子邮件地址。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_REPORT_DISPOSITION_TO_EMAIL_ADDRESSES，PR_REPORT_DISPOSITION_TO_EMAIL_ADDRESSES_A，PR_REPORT_DISPOSITION_TO_EMAIL_ADDRESSES_W  <br/> |
|标识符：  <br/> |0x0084  <br/> |
|数据类型：  <br/> |PT_MV_STRING8 PT_UNICODE  <br/> |
|区域：  <br/> |MAPI 信封  <br/> |
