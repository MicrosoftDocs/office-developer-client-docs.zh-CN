<span data-ttu-id="0841c-109">邮件内容属性描述邮件的文本。</span><span class="sxs-lookup"><span data-stu-id="0841c-109">Message content properties describe the text of a message.</span></span> <span data-ttu-id="0841c-110">每个邮件类都有自己的一组内容属性。</span><span class="sxs-lookup"><span data-stu-id="0841c-110">Every message class has its own set of content properties.</span></span> <span data-ttu-id="0841c-111">MAPI 定义便笺和报表邮件的内容属性;由处理这些邮件类的客户端和邮件存储提供程序来设置适合其实现的属性。</span><span class="sxs-lookup"><span data-stu-id="0841c-111">MAPI defines content properties for note and report messages; it is up to the clients and message store providers that handle these classes of messages to set the properties appropriately for their implementations.</span></span> <span data-ttu-id="0841c-112">**PR_BODY (** [PidTagBody](pidtagbody-canonical-property.md)) PR_RTF_COMPRESSED ([PidTagRtfCompressed](pidtagrtfcompressed-canonical-property.md)) 是笔记消息的内容属性示例。 </span><span class="sxs-lookup"><span data-stu-id="0841c-112">**PR_BODY** ([PidTagBody](pidtagbody-canonical-property.md)) and **PR_RTF_COMPRESSED** ([PidTagRtfCompressed](pidtagrtfcompressed-canonical-property.md)) are examples of content properties for note messages.</span></span> <span data-ttu-id="0841c-113">**PR_BODY** 包含注释的无格式内容，PR_RTF_COMPRESSED包含注释格式化内容的压缩版本。 </span><span class="sxs-lookup"><span data-stu-id="0841c-113">**PR_BODY** contains the unformatted contents of a note, while **PR_RTF_COMPRESSED** contains the compressed version of a note's formatted contents.</span></span> <span data-ttu-id="0841c-114">有关属性标识符范围的信息，请参阅属性 [标识符范围](property-identifier-ranges.md)。</span><span class="sxs-lookup"><span data-stu-id="0841c-114">For more information about property identifier ranges, see [Property Identifier Ranges](property-identifier-ranges.md).</span></span>
  
<span data-ttu-id="0841c-115">对于新邮件类，客户端可以通过两种方式之一定义特定于内容的属性：</span><span class="sxs-lookup"><span data-stu-id="0841c-115">For new message classes, clients can define content-specific properties in one of two ways:</span></span>

<span data-ttu-id="0841c-124">信封属性提供用于将邮件从一个收件人传输到另一个收件人的信息。</span><span class="sxs-lookup"><span data-stu-id="0841c-124">Envelope properties provide information that is used to transmit a message data-stu-id="0841c-130">**PR_SUBJECT** ([PidTagSubject](pidtagsubject-canonical-property.md)) and **PR_MESSAGE_CLASS** ([PidTagMessageClass](pidtagmessageclass-canonical-property.md)) are examples of
