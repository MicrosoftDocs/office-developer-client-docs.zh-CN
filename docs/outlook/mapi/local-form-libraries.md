# <a name="local-form-libraries"></a>本地表单库

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
本地表单库直接存储在用户的计算机上, 通常位于名为 FRMCACHE 的文件中。 
  
客户端应用程序可以访问本地表单库中的表单, 而无需访问任何网络资源。 本地表单库是存储在关联的内容表中的规则的例外, 因为本地表单库只是磁盘上不属于任何 MAPI 文件夹层次结构的文件。
