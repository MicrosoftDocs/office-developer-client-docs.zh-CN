# <a name="local-form-libraries"></a>本地表单库

  
  
**适用于**： Outlook 
  
用户的计算机，通常在文件中调用 FRMCACHE.DAT 上直接存储在本地窗体库。 
  
客户端应用程序可以访问本地表单库中的窗体，而无需访问任何网络资源。 本地表单库的表单都存储在关联的内容表中，因为本地表单库只是不是任何 MAPI 文件夹层次结构的一部分的磁盘上的文件的规则的例外。
