# <a name="cbnewspropattrarray"></a>CbNewSPropAttrArray

  
  
**适用于**： Outlook 
  
计算应为包含指定的数目的 property 属性的新[SPropAttrArray](spropattrarray.md)结构分配的字节数。 
  
|||
|:-----|:-----|
|头文件：  <br/> |Imessage.h  <br/> |
|相关的结构：  <br/> |**SPropAttrArray** <br/> |
   
```cpp
CbNewSPropAttrArray (_cattr)
```

## <a name="parameters"></a>参数

 __cattr_
  
> 要包含在新的**SPropAttrArray**结构 property 属性的计数。
