# <a name="pidtagassociatedcontentcount-canonical-property"></a>PidTagAssociatedContentCount 规范属性

  
  
**适用于**： Outlook 2013 |Outlook 2016 
  
包含关联的内容表中的文件夹的项目数。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_ASSOC_CONTENT_COUNT  <br/> |
|标识符：  <br/> |0x3617  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |Folder  <br/> |
   
## <a name="remarks"></a>注解

此属性包含的文件夹的内容表中的条目数。
