# <a name="pidtagassociatedcontentcount-canonical-property"></a>PidTagAssociatedContentCount 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
包含文件夹的 "关联的内容" 表中的项目数。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_ASSOC_CONTENT_COUNT  <br/> |
|标识符:  <br/> |0x3617  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |Folder  <br/> |
   
## <a name="remarks"></a>说明

此属性包含文件夹的内容表中的条目数。
