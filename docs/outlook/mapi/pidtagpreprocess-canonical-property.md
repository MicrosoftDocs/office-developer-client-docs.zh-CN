# <a name="pidtagpreprocess-canonical-property"></a>PidTagPreprocess 规范属性

  
  
**适用于**： Outlook 
  
如果邮件需要预处理，包含 TRUE。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_PREPROCESS  <br/> |
|标识符：  <br/> |0x0E22  <br/> |
|数据类型：  <br/> |PT_BOOLEAN  <br/> |
|区域：  <br/> |MAPI 非可传送  <br/> |
   
## <a name="remarks"></a>说明

> [!IMPORTANT]
> 不要使用此属性。 它是通过 MAPI 供使用。
