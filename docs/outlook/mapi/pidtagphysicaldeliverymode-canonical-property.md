# <a name="pidtagphysicaldeliverymode-canonical-property"></a>PidTagPhysicalDeliveryMode 规范属性

  
  
**适用于**： Outlook 
  
指定特定收件人的邮件中包含定义物理传递模式 （例如，特殊传送） 标志位的掩码。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_PHYSICAL_DELIVERY_MODE  <br/> |
|标识符：  <br/> |0x0C0B  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |MAPI 收件人  <br/> |
