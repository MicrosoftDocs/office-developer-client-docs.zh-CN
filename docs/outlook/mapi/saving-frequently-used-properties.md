# <a name="saving-frequently-used-properties"></a>保存通常用于属性

  
  
**适用于**： Outlook 
  
存储需要花费时间和资源以检索和频繁访问的数据，以提高性能。 使用额外内存有时会重复检索更好的选择。 创建用于存储此数据缓存时，请使用警告和护理。 请记住设计不合理缓存可以对性能产生负面影响。
  
例如，最人际邮件 (IPM) 客户端需要以显示典型会话期间多次打开文件夹的 IPM 子树。 您可以通过为每个这些文件夹中存储的项标识符来提高性能。
