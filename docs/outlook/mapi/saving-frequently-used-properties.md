# <a name="saving-frequently-used-properties"></a>保存常用属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
通过存储需要检索和频繁访问的时间和资源的数据来提高性能。 有时，使用额外内存可能是重复检索的更好选择。 在创建用于存储此数据的缓存时，请谨慎操作。 请记住，设计不佳的缓存可能会对性能产生负面影响。
  
例如，IPM (大部分) 客户端都需要在典型会话期间多次显示和打开文件夹的 IPM 子树。 可以通过存储其中每个文件夹的条目标识符来提高性能。
