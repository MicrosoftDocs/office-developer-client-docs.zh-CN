# <a name="pidnamephishingstamp-canonical-property"></a>PidNamePhishingStamp 规范属性

  
  
**适用于**： Outlook 
  
包含消息值，该值标识和，可能需要网络钓鱼的标记邮件。
  
|||
|:-----|:-----|
|友好名称：  <br/> |无  <br/> |
|属性进行设置：  <br/> |PS_PUBLIC_STRINGS  <br/> |
|属性名称：  <br/> |http://schemas.microsoft.com/outlook/phishingstamp  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |安全邮件  <br/> |
