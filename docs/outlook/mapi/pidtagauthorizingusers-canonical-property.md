# <a name="pidtagauthorizingusers-canonical-property"></a>PidTagAuthorizingUsers 规范属性

  
  
**适用于**： Outlook 
  
包含用户有权发送一条消息的项标识符的列表。
  
|||
|:-----|:-----|
|关联的属性：  <br/> |PR_AUTHORIZING_USERS  <br/> |
|标识符:  <br/> |0x0003  <br/> |
|数据类型：  <br/> |PT_BINARY  <br/> |
|区域：  <br/> |Exchange  <br/> |
   
## <a name="remarks"></a>备注

消息存储不保持此属性。
