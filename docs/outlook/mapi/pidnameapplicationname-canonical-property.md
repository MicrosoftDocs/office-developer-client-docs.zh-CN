# <a name="pidnameapplicationname-canonical-property"></a>PidNameApplicationName 规范属性

  
  
**适用于**： Outlook 
  
指定附加到文档文件可能会打开的应用程序。
  
|||
|:-----|:-----|
|友好名称：  <br/> |无  <br/> |
|属性进行设置：  <br/> |PS_PUBLIC_STRINGS  <br/> |
|属性名称：  <br/> |应用程序名称  <br/> |
|数据类型：  <br/> |PT_UNICODE  <br/> |
|区域：  <br/> |Common  <br/> |
