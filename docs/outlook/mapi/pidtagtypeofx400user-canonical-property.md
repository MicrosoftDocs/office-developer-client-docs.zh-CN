# <a name="pidtagtypeofx400user-canonical-property"></a>PidTagTypeOfX400User 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
此属性包含要在报告中使用的邮件收件人的类型。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_TYPE_OF_MTS_USER  <br/> |
|标识符:  <br/> |0x0C1C  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |MAPI 收件人  <br/> |
