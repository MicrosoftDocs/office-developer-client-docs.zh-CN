# <a name="pidtagprofiletype-canonical-property"></a>PidTagProfileType 规范属性

  
  
**适用于**： Outlook 2013 |Outlook 2016 
  
包含信息存储的类型。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_PROFILE_TYPE  <br/> |
|标识符：  <br/> |0x660A  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |MAPI 配置文件配置  <br/> |
   
## <a name="remarks"></a>注解

此属性的可能值包括：
  
- 1-主用户
    
- 2-委托
    
- 3-公用存储
    
- 4-订阅
