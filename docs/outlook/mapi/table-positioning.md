# <a name="table-positioning"></a>表定位

  
  
**适用于**： Outlook 
  
始终由游标表示表格中的当前位置。 没有为表; 每个视图的一个游标其值由表的实施者设置。 当使用表的客户端或服务提供程序的调用可以更改表的位置时，将重置的游标的值。 可以使用更改表的位置：
  
- 书签。
    
- 分数的值。
    
- 筛选器。
