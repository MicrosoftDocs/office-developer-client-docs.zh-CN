# <a name="mapi-features-and-architecture"></a>MAPI 功能和体系结构

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
MAPI (API) 由一组常见的应用程序编程接口和一个动态链接库组成， (DLL) 组件。 这些接口用于创建和访问不同的邮件应用程序和邮件系统，为开发和使用提供统一的环境，并为这两者提供真正的独立。 DLL 包含 MAPI 子系统，该子系统管理前端邮件应用程序和后端邮件系统之间的交互，并提供用于频繁任务的常见用户界面。 MAPI 子系统充当中央交换所，统一各种邮件系统，并保护客户端免受其差异的防护。
