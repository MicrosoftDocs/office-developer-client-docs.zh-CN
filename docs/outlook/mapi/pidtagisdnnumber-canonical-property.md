# <a name="pidtagisdnnumber-canonical-property"></a>PidTagIsdnNumber 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
包含收件人的支持 ISDN 的电话号码。 
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_ISDN_NUMBER、PR_ISDN_NUMBER_A、PR_ISDN_NUMBER_W  <br/> |
|标识符:  <br/> |0x3A2D  <br/> |
|数据类型：  <br/> |PT_UNICODE、PT_STRING8  <br/> |
|区域：  <br/> |Address  <br/> |
   
## <a name="remarks"></a>注解

这些属性可提供收件人的标识和访问信息。 它们由收件人及其组织定义。
