# <a name="pidtagsearch-canonical-property"></a>PidTagSearch 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
包含用于高级搜索的容器对象。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_SEARCH  <br/> |
|标识符:  <br/> |0x3607  <br/> |
|数据类型：  <br/> |PT_OBJECT  <br/> |
|区域：  <br/> |通讯簿容器  <br/> |
   
## <a name="remarks"></a>备注

不支持高级搜索功能的容器无需提供 属性。
