# <a name="pidtagsearch-canonical-property"></a>PidTagSearch 规范属性

  
  
**适用于**： Outlook 
  
包含用于高级搜索的容器对象。
  
|||
|:-----|:-----|
|关联的属性：  <br/> |PR_SEARCH  <br/> |
|标识符:  <br/> |0x3607  <br/> |
|数据类型：  <br/> |PT_OBJECT  <br/> |
|区域：  <br/> |通讯簿容器  <br/> |
   
## <a name="remarks"></a>备注

不支持高级的搜索功能的容器没有提供该属性。
