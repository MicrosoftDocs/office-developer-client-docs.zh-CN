**适用于**： Outlook

|||
|:-----|:-----|
|相关属性：  <br/> |PR_CONTACT_ADDRTYPES，PR_CONTACT_ADDRTYPES_A，PR_CONTACT_ADDRTYPES_W  <br/> |
