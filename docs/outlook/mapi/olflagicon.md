# <a name="olflagicon"></a>OlFlagIcon

  
  
**适用于**： Outlook 
  
指定是否已标记某个项目，并且如果，与标志的项目相关联的颜色。
  
## <a name="quick-info"></a>快速信息

```cpp
enum OlFlagIcon { 
    olNoFlagIcon=0, 
    olPurpleFlagIcon=1, 
    olOrangeFlagIcon=2, 
    olGreenFlagIcon=3, 
    olYellowFlagIcon=4, 
    olBlueFlagIcon=5, 
    olRedFlagIcon=6, 
}; 

```

## <a name="members"></a>Members

 _olNoFlagIcon_
  
> 无标志已经与项目相关联。
    
 _olPurpleFlagIcon_
  
> 紫色已与已标记的项相关联。
    
 _olOrangeFlagIcon_
  
> 橙色已与已标记的项相关联。
    
 _olGreenFlagIcon_
  
> 绿色已与已标记的项相关联。
    
 _olYellowFlagIcon_
  
> 黄色已与已标记的项相关联。
    
 _olBlueFlagIcon_
  
> 蓝色已与已标记的项相关联。
    
 _olRedFlagIcon_
  
> 红色已与已标记的项相关联。
