# <a name="pidtagviewdescriptorversion-canonical-property"></a>PidTagViewDescriptorVersion 规范属性

  
  
**适用于**： Outlook 
  
指示视图描述符的版本。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_VD_VERSION  <br/> |
|标识符：  <br/> |0x7007  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |其他  <br/> |
   
## <a name="remarks"></a>说明

此属性的值必须为 8。
