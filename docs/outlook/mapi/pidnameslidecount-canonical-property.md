# <a name="pidnameslidecount-canonical-property"></a>PidNameSlideCount 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
附加到文档的文件中指定的幻灯片的数目。
  
|||
|:-----|:-----|
|友好名称：  <br/> |无  <br/> |
|属性进行设置：  <br/> |PS_PUBLIC_STRINGS  <br/> |
|属性名称：  <br/> |SlideCount  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |Common  <br/> |
