# <a name="pidlidchangehighlight-canonical-property"></a>PidLidChangeHighlight 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
指定指示会议对象如何更改的位字段。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidChangeHighlight  <br/> |
|属性集：  <br/> |PSETID_Appointment  <br/> |
|LONG ID (的一) ：  <br/> |0x00008204  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |会议  <br/> |
   
## <a name="remarks"></a>备注

此属性不是必需的。 [[MS-OXOCAL]](https://msdn.microsoft.com/library/09861fde-c8e4-4028-9346-e7c214cfdba1%28Office.15%29.aspx)中详细说明了可以设置的个人标志。
