# <a name="pidlidlogdocumentprinted-canonical-property"></a>PidLidLogDocumentPrinted 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
指示是否在日记过程中打印文档。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidLogDocPrinted  <br/> |
|属性集:  <br/> |PSETID_Log  <br/> |
|长 ID (盖子):  <br/> |0x0000870E  <br/> |
|数据类型：  <br/> |PT_BOOLEAN  <br/> |
|区域：  <br/> |分类账  <br/> |
