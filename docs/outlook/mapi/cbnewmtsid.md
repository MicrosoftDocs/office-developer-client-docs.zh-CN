# <a name="cbnewmtsid"></a>CbNewMTSID

  
  
**适用于**： Outlook 2013 |Outlook 2016 
  
计算指定大小的邮件传输代理标识符与新[MTSID](mtsid.md)结构应分配的字节数。 
  
|||
|:-----|:-----|
|头文件：  <br/> |Mapidefs.h  <br/> |
|相关的结构：  <br/> |**MTSID** <br/> |
   
```cpp
CbNewMTSID (_cb)
```

## <a name="parameters"></a>参数

 __cb_
  
> 要包含在新的**MTSID**结构中的消息传输代理标识符的字节数。
