# <a name="cbnewmtsid"></a>CbNewMTSID

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
计算应为具有指定大小的邮件传输代理标识符的新[MTSID](mtsid.md)结构分配的字节数。 
  
|||
|:-----|:-----|
|标头文件：  <br/> |mapidefs。h  <br/> |
|相关结构:  <br/> |**MTSID** <br/> |
   
```cpp
CbNewMTSID (_cb)
```

## <a name="parameters"></a>参数

 __cb_
  
> 要包含在新的**MTSID**结构中的邮件传输代理标识符的字节数。
