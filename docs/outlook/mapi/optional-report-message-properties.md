# <a name="optional-report-message-properties"></a>可选报告邮件属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
以下属性有时与报告邮件一起显示：
  
**表标题**

|**属性**|**说明**|
|:-----|:-----|
|**PR_ORIGINAL_AUTHOR_ADDRTYPE (** [PidTagOriginalAuthorAddressType](pidtagoriginalauthoraddresstype-canonical-property.md))   <br/> |包含邮件第一个版本的作者的地址类型，即转发或答复邮件之前的邮件。  <br/> |
|**PR_ORIGINAL_AUTHOR_EMAIL_ADDRESS (** [PidTagOriginalAuthorEmailAddress](pidtagoriginalauthoremailaddress-canonical-property.md))   <br/> |包含邮件第一个版本的作者的电子邮件地址，即转发或答复邮件之前的邮件。  <br/> |
|**PR_ORIGINAL_AUTHOR_ENTRYID (** [PidTagOriginalAuthorEntryId)](pidtagoriginalauthorentryid-canonical-property.md)  <br/> |包含邮件第一个版本（即转发或答复前的邮件）的作者的条目标识符。  <br/> |
|**PR_ORIGINAL_AUTHOR_NAME (** [PidTagOriginalAuthorName](pidtagoriginalauthorname-canonical-property.md))   <br/> |包含显示名称版本的邮件，即转发或答复之前的邮件的作者的收件人。  <br/> |
|**PR_ORIGINAL_AUTHOR_SEARCH_KEY (** [PidTagOriginalAuthorSearchKey](pidtagoriginalauthorsearchkey-canonical-property.md))   <br/> |包含邮件第一个版本的作者的搜索密钥，即转发或答复邮件之前的邮件。  <br/> |
|**PR_ORIGINAL_SENT_REPRESENTING_ADDRTYPE (** [PidTagOriginalSentRepresentingAddressType](pidtagoriginalsentrepresentingaddresstype-canonical-property.md))   <br/> |包含代表其发送原始邮件的邮件用户的地址类型。  <br/> |
|**PR_ORIGINAL_SENT_REPRESENTING_EMAIL_ADDRESS (** [PidTagOriginalSentRepresentingEmailAddress](pidtagoriginalsentrepresentingemailaddress-canonical-property.md))   <br/> |包含代表其发送原始邮件的邮件用户的电子邮件地址。  <br/> |
|**PR_ORIGINAL_SENT_REPRESENTING_ENTRYID (** [PidTagOriginalSentRepresentingEntryId](pidtagoriginalsentrepresentingentryid-canonical-property.md))   <br/> |包含代表其发送原始邮件的邮件用户的条目标识符。  <br/> |
|**PR_ORIGINAL_SENT_REPRESENTING_NAME (** [PidTagOriginalSentRepresentingName](pidtagoriginalsentrepresentingname-canonical-property.md))   <br/> |包含显示名称发送原始邮件的邮件用户的邮件的用户的联系人。  <br/> |
|**PR_ORIGINAL_SENT_REPRESENTING_SEARCH_KEY (** [PidTagOriginalSentRepresentingSearchKey](pidtagoriginalsentrepresentingsearchkey-canonical-property.md))   <br/> |包含代表其发送原始邮件的邮件用户的搜索密钥。  <br/> |
|**PR_ORIGINAL_SENDER_ADDRTYPE (** [PidTagOriginalSenderAddressType](pidtagoriginalsenderaddresstype-canonical-property.md))   <br/> |包含邮件第一个版本的发件人的地址类型，即转发或答复邮件之前的邮件。  <br/> |
|**PR_ORIGINAL_SENDER_EMAIL_ADDRESS (** [PidTagOriginalSenderEmailAddress](pidtagoriginalsenderemailaddress-canonical-property.md))   <br/> |包含邮件第一个版本的发件人的电子邮件地址，即转发或答复邮件之前的邮件。  <br/> |
|**PR_ORIGINAL_SENDER_ENTRYID (** [PidTagOriginalSenderEntryId](pidtagoriginalsenderentryid-canonical-property.md))   <br/> |包含邮件第一个版本的发件人的条目标识符，即转发或答复邮件之前的邮件。  <br/> |
|**PR_ORIGINAL_SENDER_NAME (** [PidTagOriginalSenderName)](pidtagoriginalsendername-canonical-property.md)  <br/> |包含显示名称第一个版本的邮件（即转发或答复邮件之前的邮件）的发件人地址。  <br/> |
|**PR_ORIGINAL_SENDER_SEARCH_KEY (** [PidTagOriginalSenderSearchKey](pidtagoriginalsendersearchkey-canonical-property.md))   <br/> |包含邮件第一个版本的发件人的搜索密钥，即转发或答复邮件之前的邮件。  <br/> |
