# <a name="pidtagprimarytelephonenumber-canonical-property"></a>PidTagPrimaryTelephoneNumber 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
包含收件人的主要电话号码。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_PRIMARY_TELEPHONE_NUMBER、PR_PRIMARY_TELEPHONE_NUMBER_A、PR_PRIMARY_TELEPHONE_NUMBER_W  <br/> |
|标识符:  <br/> |0x3A1A  <br/> |
|数据类型：  <br/> |PT_STRING8、PT_UNICODE  <br/> |
|区域：  <br/> |MAPI 邮件用户  <br/> |
   
## <a name="remarks"></a>注解

这些属性是提供收件人的标识和访问信息的属性示例。 它们由收件人和收件人的组织定义。
