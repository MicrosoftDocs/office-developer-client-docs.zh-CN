# <a name="pidlidemail2displayname-canonical-property"></a>PidLidEmail2DisplayName 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
为第二个电子邮件地址显示名称用户可读邮件。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidEmail2DisplayName  <br/> |
|属性集：  <br/> |PSETID_Address  <br/> |
|LONG ID (的一) ：  <br/> |0x00008090  <br/> |
|数据类型：  <br/> |PT_UNICODE  <br/> |
|区域：  <br/> |联系人  <br/> |
