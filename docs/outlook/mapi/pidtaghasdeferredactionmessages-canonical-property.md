# <a name="pidtaghasdeferredactionmessages-canonical-property"></a>PidTagHasDeferredActionMessages 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
如果邮件至少具有一个延迟操作规则，则包含 TRUE。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_HAS_DAMS  <br/> |
|标识符:  <br/> |0x3FEA  <br/> |
|数据类型：  <br/> |PT_BOOLEAN  <br/> |
|区域：  <br/> |Rules  <br/> |
