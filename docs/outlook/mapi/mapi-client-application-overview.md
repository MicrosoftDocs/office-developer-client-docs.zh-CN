# <a name="mapi-client-application-overview"></a>MAPI 客户端应用程序概述

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
mapi 客户端应用程序是使用 MAPI 编程接口的任何应用程序。 客户端应用程序将邮件任务作为主焦点或辅助焦点来实现。 邮件客户端应用程序 (例如发送和接收电子邮件的应用程序) 将消息作为其主焦点实现。 对于非邮件客户端应用程序 (如清单或配置应用程序), 消息传递是辅助功能。
  
邮件活动可以包括一个字处理应用程序, 该应用程序的 "**文件**" 菜单上有 "**发送**" 命令, 以启用文档发送、Microsoft Office Outlook 电子邮件、工作流自动化程序和公告牌服务。 
  
客户端应用程序既可以包括用户, 也可以在自动环境中创建交互式环境, 也可以在没有用户的情况下运行。 虽然 MAPI 提供了一组通用的对话框及其标准用户界面, 但客户端应用程序不需要提供用户界面。 事实上, 如果需要, 可以在应用程序中处理所有处理。 自动客户端应用程序的一个示例是设计用于定期将特定类型的项目路由到标准收件人的库存管理应用程序。
