# <a name="mapi-client-application-overview"></a>MAPI 客户端应用程序概述

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
MAPI 客户端应用程序是任何使用 MAPI 编程接口的应用程序。 客户端应用程序将消息传递任务作为主要或次要焦点来实现。 邮件客户端应用程序（如发送和接收电子邮件的应用程序）将消息传递作为主要重点实现。 对于非邮件客户端应用程序（如清单或配置应用程序）来说，消息传递是一项辅助功能。
  
例如，邮件活动可能包括一个字处理应用程序，该应用程序在其"文件"菜单上具有"发送"命令以允许发送文档、Microsoft Office Outlook电子邮件、工作流程自动化程序和公告板服务。 
  
客户端应用程序可以包括用户来创建交互式环境，也可以无需用户在自动化环境中运行。 虽然 MAPI 提供了一组具有其标准用户界面的常见对话框，但客户端应用程序不需要显示用户界面。 事实上，如果需要，可在应用程序中处理所有处理。 自动化客户端应用程序的一个示例是一个清单管理应用程序，它经过编程，可定期将特定类型的项目路由到标准收件人。
