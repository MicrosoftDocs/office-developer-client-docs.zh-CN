# <a name="pidlidtaskaccepted-canonical-property"></a>PidLidTaskAccepted 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
指示任务分配者是否已答复任务请求。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidTaskAccepted  <br/> |
|属性集：  <br/> |PSETID_Task  <br/> |
|LONG ID (的一) ：  <br/> |0x00008108  <br/> |
|数据类型：  <br/> |PT_BOOLEAN  <br/> |
|区域：  <br/> |任务  <br/> |
   
## <a name="remarks"></a>备注

对于新任务，客户端将此属性设置为 FALSE，或者当接受或拒绝任务时，客户端将此属性设置为 TRUE。 如果该属性未设置，则假定值为 FALSE。
