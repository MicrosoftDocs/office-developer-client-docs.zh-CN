# <a name="pidtagmailboxownername-canonical-property"></a>PidTagMailboxOwnerName 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
显示邮箱所有者的名称。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_MAILBOX_OWNER_NAME、PR_MAILBOX_OWNER_NAME_A、PR_MAILBOX_OWNER_NAME_W  <br/> |
|标识符:  <br/> |0x001F  <br/> |
|数据类型：  <br/> |PT_UNICODE、PT_UNICODE  <br/> |
|区域：  <br/> |邮件存储  <br/> |
