# <a name="cbnewflatentry"></a>CbNewFLATENTRY

  
  
**适用于**： Outlook 
  
计算应为包含指定的字节大小的项标识符的新[FLATENTRY](flatentry.md)结构分配的字节数。 
  
|||
|:-----|:-----|
|头文件：  <br/> |Mapidefs.h  <br/> |
|相关的结构：  <br/> |**FLATENTRY** <br/> |
   
```cpp
CbNewFLATENTRY (_cb)
```

## <a name="parameters"></a>参数

 __cb_
  
> 要包含在新的**FLATENTRY**结构中的项标识符的字节数。
