# <a name="pidtagbusinessfaxnumber-canonical-property"></a>PidTagBusinessFaxNumber 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
包含收件人商务传真机的电话号码。 
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_BUSINESS_FAX_NUMBER、PR_BUSINESS_FAX_NUMBER_A、PR_BUSINESS_FAX_NUMBER_W  <br/> |
|标识符:  <br/> |0x3A24  <br/> |
|数据类型：  <br/> |PT_UNICODE、PT_STRING8  <br/> |
|区域：  <br/> |联系人  <br/> |
   
## <a name="remarks"></a>备注

这些属性是提供有关收件人的标识和访问信息的属性示例。 这些属性由收件人和收件人的组织定义。
