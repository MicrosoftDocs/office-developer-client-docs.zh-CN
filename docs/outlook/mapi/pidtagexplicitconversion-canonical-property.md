# <a name="pidtagexplicitconversion-canonical-property"></a>PidTagExplicitConversion 规范属性

  
  
**适用于**： Outlook 2013 |Outlook 2016 
  
指示邮件发件人，已请求特定的收件人的邮件内容转换。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_EXPLICIT_CONVERSION  <br/> |
|标识符：  <br/> |0x0C01  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |MAPI 收件人  <br/> |
