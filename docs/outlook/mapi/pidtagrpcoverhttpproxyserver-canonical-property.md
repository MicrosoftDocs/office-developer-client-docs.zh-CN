# <a name="pidtagrpcoverhttpproxyserver-canonical-property"></a>PidTagRpcOverHttpProxyServer 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
包含用于将 Microsoft Outlook 连接到 Microsoft Exchange Server 的代理服务器的 URL，不带 URL 之前随附的 HTTP 协议或超文本传输协议安全 (HTTPS) 协议。
  
## 

|||
|:-----|:-----|
|相关属性：  <br/> |PR_ROH_PROXY_SERVER  <br/> |
|标识符:  <br/> |0x6622  <br/> |
|属性类型  <br/> |PT_UNICODE  <br/> |
|区域：  <br/> |其他  <br/> |
