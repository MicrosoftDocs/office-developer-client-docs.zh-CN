# <a name="pidtagrpcoverhttpproxyserver-canonical-property"></a>PidTagRpcOverHttpProxyServer 规范属性

  
  
**适用于**： Outlook 
  
包含用于连接到 Microsoft Exchange Server 的 Microsoft Outlook，不含 HTTP 协议或之前 URL 的安全超文本传输协议 (HTTPS) 协议的代理服务器的 URL。
  
## 

|||
|:-----|:-----|
|关联的属性：  <br/> |PR_ROH_PROXY_SERVER  <br/> |
|标识符:  <br/> |0x6622  <br/> |
|属性类型  <br/> |PT_UNICODE  <br/> |
|区域：  <br/> |其他  <br/> |
