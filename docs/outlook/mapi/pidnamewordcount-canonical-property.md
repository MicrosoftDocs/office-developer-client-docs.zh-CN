# <a name="pidnamewordcount-canonical-property"></a>PidNameWordCount 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
指定附加到文档的文件的字数。
  
|||
|:-----|:-----|
|友好名称:  <br/> |无  <br/> |
|属性集:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|属性名称:  <br/> |WordCount  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |常见  <br/> |
