# <a name="upread"></a>UPREAD

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
[上载读取状态状态](upload-read-status-state.md)期间上载项目的读取状态的信息。
  
## <a name="quick-info"></a>快速信息

```cpp
struct UPREAD 
{ 
    PUPREADE pupre; 
    UINT cEnt; 
};
```

## <a name="members"></a>成员

 _pupre_
  
>  排除**[UPREADE](upreade.md)** 条目的矢量。 
    
 _分币_
  
>  排除**UPREADE**条目的数量。
