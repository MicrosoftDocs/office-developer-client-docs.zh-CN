**适用于**：Outlook 2013 | Outlook 2016

|**Priority**|**MAPI 值**|**TNEF 值**|
|:-----|:-----|:-----|
|通用  <br/> |-1  <br/> |第三章  <br/> |
|高效  <br/> |0  <br/> |双面  <br/> |
|降低  <br/> |1  <br/> |1  <br/> |
