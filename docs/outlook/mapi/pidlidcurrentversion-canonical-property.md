# <a name="pidlidcurrentversion-canonical-property"></a>PidLidCurrentVersion 规范属性

  
  
**适用于**： Outlook 
  
包含一个 long 类型值，该值代表项目的 Outlook 应用程序的内部版本号。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidCurrentVersion  <br/> |
|属性进行设置：  <br/> |PSETID_Common  <br/> |
|长 ID （盖）：  <br/> |0x00008552  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |常规消息  <br/> |
