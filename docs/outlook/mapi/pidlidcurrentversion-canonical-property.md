# <a name="pidlidcurrentversion-canonical-property"></a>PidLidCurrentVersion 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
包含一个 long，代表项目Outlook应用程序内部版本。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidCurrentVersion  <br/> |
|属性集：  <br/> |PSETID_Common  <br/> |
|LONG ID (的一) ：  <br/> |0x00008552  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |常规消息  <br/> |
