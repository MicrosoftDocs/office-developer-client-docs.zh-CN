# <a name="pidtagspamtrustedsenders-canonical-property"></a>PidTagSpamTrustedSenders 规范属性

  
  
**适用于**： Outlook 
  
包含以分号分隔的电子邮件地址和组成的受信任的发件人的域列表。
  
|||
|:-----|:-----|
|关联的属性：  <br/> |PR_SPAM_TRUSTED_SENDERS_W  <br/> |
|长 ID （盖）：  <br/> |0x0418  <br/> |
|数据类型：  <br/> |PT_UNICODE  <br/> |
|区域：  <br/> |垃圾邮件  <br/> |
