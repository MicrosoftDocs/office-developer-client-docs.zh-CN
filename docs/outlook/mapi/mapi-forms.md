# <a name="mapi-forms"></a>MAPI 表单

  
  
**适用于**： Outlook 2013 |Outlook 2016 
  
阅读此概述 MAPI 表单体系结构之后, 将具有了解什么是 MAPI 表单以及与其他组件的 MAPI 子系统交互的方式。 本节旨在为您提供的概念的知识，您需要实现您自己的 MAPI 表单服务器。
  
在之前阅读本节，您应熟悉的[MAPI 表单概述](mapi-forms-overview.md)主题中的材料。 
  
> [!NOTE]
> MAPI 表单体系结构基于上组件对象模型 (COM)，因为开发表单服务器应用程序需要 COM 编程的知识。 有关 COM 的详细信息，请参阅 Windows SDK 中的 COM 和 ActiveX 对象服务一节。
