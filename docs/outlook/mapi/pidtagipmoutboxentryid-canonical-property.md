# <a name="pidtagipmoutboxentryid-canonical-property"></a>PidTagIpmOutboxEntryId 规范属性

  
  
**适用于**： Outlook 
  
包含标准人际邮件 (IPM) 发件箱文件夹的项标识符。 
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_IPM_OUTBOX_ENTRYID  <br/> |
|标识符：  <br/> |0x35E2  <br/> |
|数据类型：  <br/> |PT_BINARY  <br/> |
|区域：  <br/> |Folder  <br/> |
   
## <a name="remarks"></a>说明

在发件箱文件夹通常创建出站邮件。 应在提交此文件夹中放置人际邮件。
