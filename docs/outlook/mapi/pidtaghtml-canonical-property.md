# <a name="pidtaghtml-canonical-property"></a>PidTagHtml 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
包含以 HTML 格式的消息正文文本。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_HTML  <br/> |
|标识符：  <br/> |0x1013  <br/> |
|数据类型：  <br/> |PT_BINARY  <br/> |
|区域：  <br/> |常规消息  <br/> |
