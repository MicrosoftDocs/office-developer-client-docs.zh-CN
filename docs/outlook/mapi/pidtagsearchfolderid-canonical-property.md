# <a name="pidtagsearchfolderid-canonical-property"></a>PidTagSearchFolderId 规范属性

  
  
**适用于**： Outlook 
  
包含标识搜索文件夹的全局唯一标识符 (GUID)。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_WB_SF_ID  <br/> |
|标识符：  <br/> |0x6842  <br/> |
|数据类型：  <br/> |PT_BINARY  <br/> |
|区域：  <br/> |搜索  <br/> |
