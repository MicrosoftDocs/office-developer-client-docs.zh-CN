# <a name="pidtagread-canonical-property"></a>PidTagRead 规范属性

  
  
**适用于**： Outlook 
  
指示是否已读取表格行。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_READ  <br/> |
|标识符：  <br/> |0x0E69  <br/> |
|数据类型：  <br/> |PT_BOOLEAN  <br/> |
|区域：  <br/> |MAPI 非可传送  <br/> |
