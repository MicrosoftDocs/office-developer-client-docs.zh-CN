# <a name="pidtagsubjectmessageid-canonical-property"></a>PidTagSubjectMessageId 规范属性

  
  
**适用于**： Outlook 
  
包含为其生成报告的邮件复制的二进制值。 
  
|||
|:-----|:-----|
|关联的属性：  <br/> |PR_SUBJECT_IPM  <br/> |
|标识符:  <br/> |0x0038  <br/> |
|数据类型：  <br/> |PT_BINARY  <br/> |
|区域：  <br/> |MAPI 信封  <br/> |
   
## <a name="remarks"></a>备注

此属性，如**PR_REPORT_TAG** ([PidTagReportTag](pidtagreporttag-canonical-property.md)) 属性，可以用于与原始邮件相关的报告。
