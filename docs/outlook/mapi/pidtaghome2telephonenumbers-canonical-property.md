# <a name="pidtaghome2telephonenumbers-canonical-property"></a>PidTagHome2TelephoneNumbers 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
包含收件人住宅中的辅助电话号码列表。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_HOME2_TELEPHONE_NUMBER_A_MV  <br/> |
|标识符:  <br/> |0x3A2F  <br/> |
|数据类型：  <br/> |PT_MV_UNICODE8  <br/> |
|区域：  <br/> |MAPI 邮件用户  <br/> |
