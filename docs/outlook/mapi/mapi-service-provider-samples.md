# <a name="mapi-service-provider-samples"></a>MAPI 服务提供程序示例

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
本节包含三个 MAPI 服务提供商的代码示例：
  
- [下载Outlook MAPI 示例](downloading-the-outlook-mapi-samples.md)
    
    本主题介绍如何下载示例。
    
- [通讯簿提供程序示例](address-book-provider-sample.md)
    
    此示例支持显示名称和电子邮件地址（从平面二进制文件读取）的单个只读容器。
    
- [邮件存储提供程序示例](message-store-provider-sample.md)
    
    此示例使用目录作为文件夹，将 OLE 2.0 文档文件用作邮件。 此示例支持默认邮件存储的所有必需功能。
    
- [传输提供程序示例](transport-provider-sample.md)
    
    此示例使用文件和目录传输和接收消息。
    
若要在 Windows 7、Windows Vista、Windows Server 2008、Windows XP 或 Windows Server 2003 操作系统上运行这些示例，必须先设置 MAPI 运行时系统。 有关详细信息，请参阅 [MAPI 编程概述](mapi-programming-overview.md)。
