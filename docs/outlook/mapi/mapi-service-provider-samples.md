# <a name="mapi-service-provider-samples"></a>MAPI 服务提供程序示例

  
  
**适用于**： Outlook 
  
本部分包含三个 MAPI 服务提供商的代码示例：
  
- [下载 Outlook MAPI 示例](downloading-the-outlook-mapi-samples.md)
    
    本主题介绍如何下载示例。
    
- [通讯簿提供程序示例](address-book-provider-sample.md)
    
    此示例显示名称和电子邮件地址，从平面二进制文件读取支持单个只读的容器。
    
- [邮件存储区提供程序示例](message-store-provider-sample.md)
    
    此示例使用目录为文件夹和 OLE 2.0 文档文件作为邮件。 此示例为默认邮件存储区支持所有必需的功能。
    
- [传输提供程序示例](transport-provider-sample.md)
    
    此示例使用文件和目录传输和接收消息。
    
若要运行这些示例的任何 Windows 7、 Windows Vista、 Windows Server 2008、 Windows XP 或 Windows Server 2003 操作系统上，必须先设置 MAPI 运行时系统。 有关详细信息，请参阅[MAPI 编程的概述](mapi-programming-overview.md)。
