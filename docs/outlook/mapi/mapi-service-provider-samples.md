# <a name="mapi-service-provider-samples"></a>MAPI 服务提供程序示例

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
本节包含三个 MAPI 服务提供程序的代码示例:
  
- [下载 Outlook MAPI 示例](downloading-the-outlook-mapi-samples.md)
    
    本主题介绍如何下载示例。
    
- [通讯簿提供程序示例](address-book-provider-sample.md)
    
    此示例支持从简单二进制文件读取的显示名称和电子邮件地址的单个只读容器。
    
- [邮件存储提供程序示例](message-store-provider-sample.md)
    
    本示例将目录作为文件夹, 将 OLE 2.0 文档文件作为邮件使用。 该示例支持默认邮件存储区所需的所有功能。
    
- [传输提供程序示例](transport-provider-sample.md)
    
    此示例使用文件和目录传输和接收邮件。
    
若要在 windows 7、windows Vista、windows server 2008、windows XP 或 windows Server 2003 操作系统上运行这些示例, 必须首先设置 MAPI 运行时系统。 有关详细信息，请参阅 [MAPI 编程概述](mapi-programming-overview.md)。
