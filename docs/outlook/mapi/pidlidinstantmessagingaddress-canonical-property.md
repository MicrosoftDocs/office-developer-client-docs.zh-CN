# <a name="pidlidinstantmessagingaddress-canonical-property"></a>PidLidInstantMessagingAddress 规范属性

  
  
**适用于**： Outlook 
  
指定联系人的即时消息地址。
  
|||
|:-----|:-----|
|关联的属性：  <br/> |dispidInstMsg  <br/> |
|属性进行设置：  <br/> |PSETID_Address  <br/> |
|长 ID （盖）：  <br/> |0x00008062  <br/> |
|数据类型：  <br/> |PT_UNICODE  <br/> |
|区域：  <br/> |联系人  <br/> |
