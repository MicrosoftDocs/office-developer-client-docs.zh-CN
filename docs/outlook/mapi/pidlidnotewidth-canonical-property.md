# <a name="pidlidnotewidth-canonical-property"></a>PidLidNoteWidth 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
指定可见邮件窗口的宽度（以像素为单位）。 
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidNoteWidth  <br/> |
|属性集：  <br/> |PSETID_Note  <br/> |
|LONG ID (的一) ：  <br/> |0x00008B02  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |便笺  <br/> |
   
## <a name="remarks"></a>备注

此值必须大于零。
