# <a name="client-shutdown-in-mapi"></a>MAPI 中的客户端关闭 
  
**适用于**： Outlook 2013 |Outlook 2016 
  
开始在 Microsoft Outlook 2010 中，现在包括 Microsoft Outlook 2013 MAPI 客户端可以关闭方式相同之前，或者他们可使用快速关闭。 对于要成功进行的快速关闭，MAPI 客户端、 MAPI 提供程序和客户端计算机的管理员必须支持快速关闭。 
  
本节中的主题介绍执行快速关闭的客户端的 MAPI 支持。
  
[快速关闭概述](fast-shutdown-overview.md)
  
> 本主题介绍快速关闭的基本机制。
    
[快速关闭用户选项](fast-shutdown-user-options.md)
  
> 本主题介绍为管理员的用户的 MAPI 客户端采用在用户级别的快速关闭可用的选项。
    
[快速关闭的最佳实践](best-practices-for-fast-shutdown.md)
  
> 本主题建议使用的快速关闭接口帮助 MAPI 客户端关闭期间防止数据丢失的最佳做法。
