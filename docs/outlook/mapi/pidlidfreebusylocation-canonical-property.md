# <a name="pidlidfreebusylocation-canonical-property"></a>PidLidFreeBusyLocation 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
指定一个 URL 路径, 客户端可以从该路径检索联系人的忙/闲信息, 如[[OXCICAL]](https://msdn.microsoft.com/library/a685a040-5b69-4c84-b084-795113fb4012%28Office.15%29.aspx)中所指定的 iCal 文件。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidFreeBusyLocation  <br/> |
|属性集:  <br/> |PSETID_Address  <br/> |
|长 ID (盖子):  <br/> |0x000080D8  <br/> |
|数据类型：  <br/> |PT_UNICODE  <br/> |
|区域：  <br/> |Contact  <br/> |
