# <a name="pidlidfreebusylocation-canonical-property"></a>PidLidFreeBusyLocation 规范属性

  
  
**适用于**： Outlook 
  
指定客户端可以从中检索忙/闲信息的联系人中指定[[MS OXCICAL]](http://msdn.microsoft.com/library/a685a040-5b69-4c84-b084-795113fb4012%28Office.15%29.aspx)为 iCal 文件的 URL 路径。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidFreeBusyLocation  <br/> |
|属性进行设置：  <br/> |PSETID_Address  <br/> |
|长 ID （盖）：  <br/> |0x000080D8  <br/> |
|数据类型：  <br/> |PT_UNICODE  <br/> |
|区域：  <br/> |联系人  <br/> |
