# <a name="pidlidemail1emailaddress-canonical-property"></a>PidLidEmail1EmailAddress 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
指定联系人的第一个电子邮件地址。
  
|||
|:-----|:-----|
|相关属性：  <br/> |dispidEmail1EmailAddress  <br/> |
|属性集:  <br/> |PSETID_Address  <br/> |
|长 ID (盖子):  <br/> |0x00008083  <br/> |
|数据类型：  <br/> |PT_UNICODE  <br/> |
|区域：  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>注解

此属性的值必须适用于为此电子邮件地址指定的地址类型。
