# <a name="pidtagsearchownerid-canonical-property"></a>PidTagSearchOwnerId 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
包含将通知发送到索引器的进程 ID, 该进程准备好编制索引的对象。
