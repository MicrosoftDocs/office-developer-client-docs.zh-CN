# <a name="pidtagproofofsubmission-canonical-property"></a>PidTagProofOfSubmission 规范属性

  
  
**适用于**： Outlook 
  
此属性包含 TRUE 如果邮件发件人请求证明邮件传输系统已提交的邮件传递到最初预期接收人。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_PROOF_OF_SUBMISSION  <br/> |
|标识符：  <br/> |0x0E26  <br/> |
|数据类型：  <br/> |PT_BINARY  <br/> |
|区域：  <br/> |MAPI 非可传送  <br/> |
