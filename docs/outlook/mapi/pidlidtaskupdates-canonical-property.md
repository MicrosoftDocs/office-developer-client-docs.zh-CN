# <a name="pidlidtaskupdates-canonical-property"></a>PidLidTaskUpdates 规范属性

  
  
**适用于**： Outlook 
  
指示是否已发送的任务更新所分配的任务更改时请求任务受理人。
  
|||
|:-----|:-----|
|关联的属性：  <br/> |dispidTaskUpdates  <br/> |
|属性进行设置：  <br/> |PSETID_Task  <br/> |
|长 ID （盖）：  <br/> |0x0000811B  <br/> |
|数据类型：  <br/> |PT_BOOLEAN  <br/> |
|区域：  <br/> |任务  <br/> |
