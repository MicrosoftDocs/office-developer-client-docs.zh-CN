# <a name="pidtagproofofsubmissionrequested-canonical-property"></a>PidTagProofOfSubmissionRequested 规范属性

  
  
**适用于**： Outlook 
  
包含 ASN.1 证明提交值。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_PROOF_OF_SUBMISSION_REQUESTED  <br/> |
|标识符：  <br/> |0x0028  <br/> |
|数据类型：  <br/> |PT_BOOLEAN  <br/> |
|区域：  <br/> |MAPI 信封  <br/> |
   
## <a name="remarks"></a>说明

此属性对应于 X.400 提交信封每封邮件属性。
