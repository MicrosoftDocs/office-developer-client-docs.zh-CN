# <a name="optional-transport-provider-features"></a>可选的传输提供程序功能

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
传输提供程序可以实现的可选功能包括：
  
- 注册特定于传输提供程序的邮件和收件人选项。
    
- 维护配置文件（如有必要）以将配置信息和凭据存储到邮件系统。
    
- 对邮件系统所需的凭据执行任何验证。
    
- 使用 [IMAPISupport：：Notify](imapisupport-notify.md) 方法为感兴趣的客户端应用程序支持事件通知。 
    
- 显示配置属性表和向导对话框，以使用户能够配置传输提供程序的设置。
    
- 向客户端应用程序提供邮件送达报告。
