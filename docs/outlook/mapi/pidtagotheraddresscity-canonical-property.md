# <a name="pidtagotheraddresscity-canonical-property"></a>PidTagOtherAddressCity 规范属性

  
  
**适用于**： Outlook 
  
用户的其他位置，例如城镇或市/县包含邮件的名称。
  
|||
|:-----|:-----|
|关联的属性：  <br/> |PR_OTHER_ADDRESS_CITY，PR_OTHER_ADDRESS_CITY_A，PR_OTHER_ADDRESS_CITY_W  <br/> |
|标识符:  <br/> |0x3A5F  <br/> |
|数据类型：  <br/> |PT_UNICODE PT_STRING8  <br/> |
|区域：  <br/> |Address  <br/> |
   
## <a name="remarks"></a>备注

由用户或用户的组织定义这些属性。
