# <a name="pidtagredirectionhistory-canonical-property"></a>PidTagRedirectionHistory 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
包含有关已传递邮件涵盖的路由的信息。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_REDIRECTION_HISTORY  <br/> |
|标识符:  <br/> |0x002C  <br/> |
|数据类型：  <br/> |PT_BINARY  <br/> |
|区域：  <br/> |MAPI 信封  <br/> |
   
## <a name="remarks"></a>备注

当自动代理自动前向邮件时，使用此属性。
