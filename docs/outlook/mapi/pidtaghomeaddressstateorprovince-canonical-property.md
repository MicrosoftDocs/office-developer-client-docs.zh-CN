# <a name="pidtaghomeaddressstateorprovince-canonical-property"></a>PidTagHomeAddressStateOrProvince 规范属性

  
  
**适用于**： Outlook 2013 |Outlook 2016 
  
包含用户的地址的州或省部分。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_HOME_ADDRESS_STATE_OR_PROVINCE，PR_HOME_ADDRESS_STATE_OR_PROVINCE_A，PR_HOME_ADDRESS_STATE_OR_PROVINCE_W  <br/> |
|标识符：  <br/> |0x3A5C  <br/> |
|数据类型：  <br/> |PT_STRING8 PT_UNICODE  <br/> |
|区域：  <br/> |MAPI 邮件用户  <br/> |
   
## <a name="remarks"></a>注解

由用户或用户的组织定义这些属性。
