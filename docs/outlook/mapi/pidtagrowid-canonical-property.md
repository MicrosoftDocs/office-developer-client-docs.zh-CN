# <a name="pidtagrowid-canonical-property"></a>PidTagRowid 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
包含收件人表或状态表中收件人的唯一标识符。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_ROWID  <br/> |
|标识符:  <br/> |0x3000  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |MAPI common  <br/> |
   
## <a name="remarks"></a>备注

此属性是一个临时值，仅在拥有该表的对象的生命周期内有效。 它显示为表格的一列，但不存储。
