# <a name="pidtagreplyrequested-canonical-property"></a>PidTagReplyRequested 规范属性

  
  
**适用于**： Outlook 
  
包含 TRUE，则邮件发件人请求来自收件人的答复。
  
|||
|:-----|:-----|
|关联的属性：  <br/> |PR_REPLY_REQUESTED  <br/> |
|标识符:  <br/> |0x0C17  <br/> |
|数据类型：  <br/> |PT_BOOLEAN  <br/> |
|区域：  <br/> |MAPI 收件人  <br/> |
