# <a name="pidtagmobiletelephonenumber-canonical-property"></a>PidTagMobileTelephoneNumber 规范属性

  
  
**适用于**： Outlook 
  
包含收件人的移动电话号码。
  
|||
|:-----|:-----|
|关联的属性：  <br/> |PR_MOBILE_TELEPHONE_NUMBER、 PR_MOBILE_TELEPHONE_NUMBER_A、 PR_MOBILE_TELEPHONE_NUMBER_W、 PR_CELLULAR_TELEPHONE_NUMBER、 PR_CELLULAR_TELEPHONE_NUMBER_A、 PR_CELLULAR_TELEPHONE_NUMBER_W  <br/> |
|标识符:  <br/> |0x3A1C  <br/> |
|数据类型：  <br/> |PT_UNICODE PT_STRING8  <br/> |
|区域：  <br/> |通讯簿  <br/> |
   
## <a name="remarks"></a>备注

这些属性提供标识和访问收件人的信息。 它们是按收件人和组织定义的。
