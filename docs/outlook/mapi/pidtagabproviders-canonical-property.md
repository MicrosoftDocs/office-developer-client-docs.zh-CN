# <a name="pidtagabproviders-canonical-property"></a>PidTagAbProviders 规范属性

  
  
**适用于**： Outlook 
  
包含当前配置文件中的通讯簿提供程序标识符的列表。 
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_AB_PROVIDERS  <br/> |
|标识符：  <br/> |0x3D01  <br/> |
|数据类型：  <br/> |PT_BINARY  <br/> |
|区域：  <br/> |通讯簿  <br/> |
   
## <a name="remarks"></a>说明

不要使用此属性。 它是通过 MAPI 供使用。
