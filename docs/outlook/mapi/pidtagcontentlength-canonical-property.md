# <a name="pidtagcontentlength-canonical-property"></a>PidTagContentLength 规范属性

  
  
**适用于**： Outlook 
  
包含消息长度，以字节为单位，传递给客户端应用程序或服务提供商，以确定是否可以发送消息的长度。 
  
|||
|:-----|:-----|
|关联的属性：  <br/> |PR_CONTENT_LENGTH  <br/> |
|标识符:  <br/> |0x0009  <br/> |
|数据类型：  <br/> |PT_LONG  <br/> |
|区域：  <br/> |Exchange  <br/> |
