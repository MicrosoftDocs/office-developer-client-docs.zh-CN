# <a name="pidtagotheraddressstateorprovince-canonical-property"></a>PidTagOtherAddressStateOrProvince 规范属性

  
  
**适用于**：Outlook 2013 | Outlook 2016 
  
包含的州或省其他地址中使用的名称。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_OTHER_ADDRESS_STATE_OR_PROVINCE，PR_OTHER_ADDRESS_STATE_OR_PROVINCE_A，PR_OTHER_ADDRESS_STATE_OR_PROVINCE_W  <br/> |
|标识符：  <br/> |0x3A62  <br/> |
|数据类型：  <br/> |PT_UNICODE PT_STRING8  <br/> |
|区域：  <br/> |Address  <br/> |
   
## <a name="remarks"></a>注解

由用户或用户的组织定义这些属性。
