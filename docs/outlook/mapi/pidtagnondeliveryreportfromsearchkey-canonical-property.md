# <a name="pidtagnondeliveryreportfromsearchkey-canonical-property"></a>PidTagNonDeliveryReportFromSearchKey 规范属性

  
  
**适用于**： Outlook 
  
如果原件报表上设置，表示搜索关键字的失败的发送-作为用户。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_NDR_FROM_SEARCH_KEY  <br/> |
|标识符：  <br/> |0x6906  <br/> |
|数据类型：  <br/> |PT_BINARY  <br/> |
|区域：  <br/> |常规消息  <br/> |
