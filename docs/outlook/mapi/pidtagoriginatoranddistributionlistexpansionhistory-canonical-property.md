# <a name="pidtagoriginatoranddistributionlistexpansionhistory-canonical-property"></a>PidTagOriginatorAndDistributionListExpansionHistory 规范属性

  
  
**适用于**： Outlook 
  
包含有关原始邮件发件人和通讯组列表扩展历史记录信息。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_ORIGINATOR_AND_DL_EXPANSION_HISTORY  <br/> |
|标识符：  <br/> |0x1002  <br/> |
|数据类型：  <br/> |PT_BINARY  <br/> |
|区域：  <br/> |MIME  <br/> |
   
## <a name="remarks"></a>说明

在生成报告使用此属性。
