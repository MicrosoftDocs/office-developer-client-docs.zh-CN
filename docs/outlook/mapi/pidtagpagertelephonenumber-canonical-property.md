# <a name="pidtagpagertelephonenumber-canonical-property"></a>PidTagPagerTelephoneNumber 规范属性

  
  
**适用于**： Outlook 2013 |Outlook 2016 
  
包含收件人的寻呼机电话号码。
  
|||
|:-----|:-----|
|相关属性：  <br/> |PR_PAGER_TELEPHONE_NUMBER、 PR_PAGER_TELEPHONE_NUMBER_A、 PR_PAGER_TELEPHONE_NUMBER_W、 PR_BEEPER_TELEPHONE_NUMBER、 PR_BEEPER_TELEPHONE_NUMBER_A、 PR_BEEPER_TELEPHONE_NUMBER_W  <br/> |
|标识符：  <br/> |0x3A21  <br/> |
|数据类型：  <br/> |PT_UNICODE PT_STRING8  <br/> |
|区域：  <br/> |Address  <br/> |
   
## <a name="remarks"></a>注解

这些属性提供标识和访问收件人的信息。 它们是按收件人和组织定义的。
