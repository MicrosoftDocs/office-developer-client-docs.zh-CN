# <a name="isocialprofile--isocialperson"></a>ISocialProfile : ISocialPerson

代表登录用户。 
  
## <a name="members"></a>Members

下表显示了**ISocialProfile**接口上可用的成员。 
  
|**名称**|**成员类型**|**说明**|
|:-----|:-----|:-----|
|[AreFriendsOrColleagues](isocialprofile-arefriendsorcolleagues.md) <br/> |方法  <br/> |确定指定的用户是否朋友。  <br/> |
|[GetActivitiesOfFriendsAndColleagues](isocialprofile-getactivitiesoffriendsandcolleagues.md) <br/> |方法  <br/> |Outlook Social Connector 2013 以来已弃用此方法。  <br/> |
|[SetStatus](isocialprofile-setstatus.md) <br/> |方法  <br/> |当前不支持此方法。  <br/> |
   
## <a name="remarks"></a>说明

Outlook Social Connector (OSC) 提供程序必须实现此接口与 OSC 进行通信。
