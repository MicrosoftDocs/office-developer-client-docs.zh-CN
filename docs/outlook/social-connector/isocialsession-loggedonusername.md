# <a name="isocialsessionloggedonusername"></a>ISocialSession::LoggedOnUserName

返回一个字符串, 表示登录时使用的用户名。
  
```cpp
[propget] HRESULT _stdcall LoggedOnUserName([out, retval] BSTR* result);
```

## <a name="property-value"></a>属性值

一个字符串, 表示登录用户的用户名。
