# <a name="osc-sample-templates"></a>OSC 示例模板

Social Connector 2013 Outlook参考随附了 C++、C# 和 Visual Basic 中的多个提供程序模板。 可以从 Social [Connector 2013：Outlook模板 下载模板](https://code.msdn.microsoft.com/Outlook-Social-Connector-73fd8d2c)。
  
使用你选择的语言构建提供程序时，可以使用提供程序模板。 若要最大程度地减小提供程序的下载大小并尽可能缩短提供程序加载时间，应该使用 C++ 生成生产提供程序。 如果要为计算机上安装了 .NET Framework 3.5 SP1 或更高版本的 .NET Framework 的用户开发提供程序，还可以考虑使用 C# 或 Visual Basic 编写托管提供程序。
