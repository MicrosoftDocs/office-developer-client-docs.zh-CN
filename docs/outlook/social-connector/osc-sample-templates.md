# <a name="osc-sample-templates"></a>.osc 示例模板

随附 Outlook Social Connector 2013 provider 引用是 c + +、c # 和 Visual Basic 中的几个提供程序模板。 您可以在[Outlook Social Connector 2013: 提供程序模板](https://code.msdn.microsoft.com/Outlook-Social-Connector-73fd8d2c)中下载模板。
  
当您使用您选择的语言构建提供程序时, 提供程序模板可供使用。 若要最大限度地减少提供程序的下载大小并最大限度地减少提供程序加载时间, 应使用 c + + 生成生产提供程序。 如果要为在其计算机上安装了 .net framework 3.5 SP1 或更高版本的 .net framework 的用户开发提供程序, 则还可以考虑使用 c # 或 Visual Basic 编写托管提供程序。
