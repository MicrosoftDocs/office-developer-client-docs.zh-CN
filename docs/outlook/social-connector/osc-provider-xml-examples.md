# <a name="osc-provider-xml-examples"></a>OSC 提供程序 XML 示例

本节包含使用功能、**好友** 和 **activityFeed** XML 元素的 XML 示例，这些示例符合 Outlook Social Connector (OSC) XML 架构。 
  
- [Capabilities XML](capabilities-xml-example.md) 示例 **演示** OSC 提供程序的功能 XML，用于指定 OSC 的功能和要求。 
    
- [好友 XML 示例](friends-xml-example.md) 显示 **社交网络上** 两个人的好友 XML。 
    
- [活动源 XML 示例](activity-feed-xml-example.md) 显示四个 **活动的 activityFeed** XML：两个配置文件图片更新、一个状态更新和一个博客文章。 XML 还指定用于显示相应活动的三个活动显示模板。
