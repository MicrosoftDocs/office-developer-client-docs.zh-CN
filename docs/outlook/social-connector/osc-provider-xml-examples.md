# <a name="osc-provider-xml-examples"></a>OSC 提供程序 XML 示例

此部分包含使用**功能**、**朋友**和**activityFeed** XML 元素，遵守 Outlook Social Connector (OSC) 提供程序的 XML 架构的 XML 示例。 
  
- [功能 XML 示例](capabilities-xml-example.md)演示**功能**OSC 提供程序，指定其功能和要求 OSC XML。 
    
- [朋友 XML 示例](friends-xml-example.md)演示用于社交网络上的两个人员**朋友**XML。 
    
- [活动源 XML 示例](activity-feed-xml-example.md)演示**activityFeed**四项活动的 XML： 两个配置文件图片更新、 状态更新和博客文章。 XML 还会指定用于显示相应的活动的三个活动显示模板。
