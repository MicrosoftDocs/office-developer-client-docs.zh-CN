# <a name="osc-provider-xml-examples"></a>.osc 提供程序 XML 示例

本节包含的 xml 示例使用**功能**、**好友**和**microsoft.office.server.activityfeed** xml 元素, 遵循 Outlook Social Connector (.osc) 提供程序 XML 架构。 
  
- [功能 xml 示例](capabilities-xml-example.md)显示了 .osc 提供程序的**功能**xml, 指定了 .osc 的功能和要求。 
    
- [好友 xml 示例](friends-xml-example.md)显示社交网络中两个人的**好友**xml。 
    
- [活动源 xml 示例](activity-feed-xml-example.md)显示了四个活动的**microsoft.office.server.activityfeed** XML: 两个配置文件图片更新、一个状态更新和一个博客文章。 XML 还指定三个活动显示模板, 用于显示相应的活动。
