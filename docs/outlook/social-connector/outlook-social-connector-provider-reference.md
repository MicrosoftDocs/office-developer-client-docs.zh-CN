# <a name="outlook-social-connector-provider-reference"></a>Outlook Social Connector 提供程序参考

Outlook Social Connector 2013 提供了用于个人和专业通信的通信中心。 Outlook Social Connector (.osc) 适用于 sharepoint Server、sharepoint Workspace、Lync 客户端和所有支持状态信息的 Office 客户端应用程序以及联系人卡片支持 .osc。 

在特定的 outlook 中, 只需选择 outlook 项目 (如电子邮件或会议请求, 然后单击该项目中的发件人或收件人) 即可在不离开 Outlook 的情况下, 用户可以在其上的人员窗格活动、照片和状态更新中看到此人喜爱的社交网络。 用户可以方便地看到所有 Outlook 电子邮件、附件和从该人收到的会议。 在组织环境中, sharepoint 网站上的用户也可以在 sharepoint 网站上的人员窗格文档更新和此人员的其他网站活动中查看。
  
社交网络可以开发用于 .osc 的提供程序, 以便在支持服务器和应用程序中同步和显示社交网络更新。 受欢迎的社交网络 (如 LinkedIn、Facebook 和 Windows Live) 为 .osc 提供提供程序。 
  
Outlook Social Connector 2013 Provider 参考介绍了如何使用 .osc 提供程序可扩展性开发一个 .osc 提供程序。 
  
如果你是第一次开发 Outlook 解决方案，请参阅[选择某个 API 或技术开发适用于 Outlook 的解决方案](../selecting-an-api-or-technology-for-developing-solutions-for-outlook.md)，以确定最适合你需求的 API 和技术。
