# <a name="deploying-a-provider"></a>部署提供程序

部署 Outlook Social Connector (OSC) 提供程序的建议的方法是使用 Windows Installer (.msi) 程序包。 下载到本地硬盘上的安装包之后, Office 最终用户安装 OSC 提供程序。 提供程序安装完成后，用户可以连接到 OSC 提供程序，并提供到社交网络帐户的登录凭据。 如果用户决定卸载 OSC 提供程序，用户可以使用**程序和功能**控制面板中卸载。
