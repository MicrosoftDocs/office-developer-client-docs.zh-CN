# <a name="reserved-keywords-access-custom-web-app"></a>Access 自定义 Web (保留的) 

Microsoft Access 使用保留的关键字来定义、操作和访问数据库。 保留的关键字是 Access 用于分析和理解这些语句的语言语法的一SQL部分。 
  
> [!IMPORTANT]
> Microsoft 不再建议在 SharePoint 中创建和使用 Access Web 应用程序。作为备选方法，请考虑使用 [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/)，生成适用于 Web 和移动设备的无代码业务解决方案。
