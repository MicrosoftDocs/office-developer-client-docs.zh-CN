# <a name="min-function-access-custom-web-app"></a>Min 函数 （访问自定义 web 应用程序）

返回的最小值中查询或表中的表达式。
  
> [!IMPORTANT]
> [!重要信息] Microsoft 不再建议在 SharePoint 中创建和使用 Access Web 应用程序。作为备选方法，请考虑使用 [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/)，生成适用于 Web 和移动设备的无代码业务解决方案。 
  
## <a name="syntax"></a>语法

 **最小值**（*表达式*） 
  
**Min**函数包含以下参数。 
  
|**参数名称**|**说明**|
|:-----|:-----|
| *Expression*  <br/> |标识包含要计算的数据或表达式的计算使用该字段中的数据的字段的字符串表达式。 在*表达式*中的操作数可包括表字段、 一个常量或函数 （这可能是固有或用户定义但不是其他 SQL 聚合函数） 的名称。  <br/> |
