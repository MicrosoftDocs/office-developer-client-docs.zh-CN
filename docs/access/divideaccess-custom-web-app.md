# <a name="-divide-access-custom-web-app"></a>/ (Divide)  (Access 自定义 Web 应用) 

将一个数字除以另一个数字。
  
> [!IMPORTANT]
> Microsoft 不再建议在 SharePoint 中创建和使用 Access Web 应用程序。作为备选方法，请考虑使用 [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/)，生成适用于 Web 和移动设备的无代码业务解决方案。 
  
## <a name="syntax"></a>语法

 *一个*   /  *divisor* 
  
 *一个*  要除以的数值表达式。 可以是数值数据类别的任何数据类型的任何有效数据类型，但 datetime 数据类型。 
  
 *Divisor*  是除数的数值表达式。 可以是数值数据类别的任何数据类型的任何有效数据类型，但 datetime 数据类型。 
  
## <a name="return-type"></a>返回类型

返回数据类型优先级较高的参数的变量。 
  
如果整数 *除* 数除以整数除数，则结果是结果的任何小数部分被截断的整数。 
  
## <a name="remarks"></a>备注

/ 实际值返回的值是第一个表达式除以第二个表达式的商。
