# <a name="-divide-access-custom-web-app"></a>/(除) (Access 自定义 web 应用程序)

将一个数除以另一个数。
  
> [!IMPORTANT]
> Microsoft 不再建议在 SharePoint 中创建和使用 Access Web 应用程序。作为备选方法，请考虑使用 [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/)，生成适用于 Web 和移动设备的无代码业务解决方案。 
  
## <a name="syntax"></a>语法

 *除数*  /  *除数* 
  
 被*除数* 是要进行除法运算的数值表达式。 可以是数值数据类型类别的任意一种数据类型的任何有效表达式, datetime 数据类型除外。 
  
 *除数* 是用来除以除数的数值表达式。 可以是数值数据类型类别的任意一种数据类型的任何有效表达式, datetime 数据类型除外。 
  
## <a name="return-type"></a>返回类型

返回优先级较高的参数的数据类型。 
  
如果用整数** *除数*除以整数, 则结果是一个整数, 其中包含结果的任何小数部分截断。 
  
## <a name="remarks"></a>注解

由/运算符返回的实际值是第一个表达式的商除以第二个表达式。
