# <a name="-divide-access-custom-web-app"></a>/ （划分） （访问自定义 web 应用程序）

用由另一个数。
  
> [!IMPORTANT]
> Microsoft 不再建议在 SharePoint 中创建和使用 Access Web 应用程序。作为备选方法，请考虑使用 [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/)，生成适用于 Web 和移动设备的无代码业务解决方案。 
  
## <a name="syntax"></a>语法

 *被除数*  /  *除数* 
  
 *被除数* 划分的数值表达式。 可以是任何有效的数字数据类型分类，除 datetime 数据类型的数据类型之一的表达式。 
  
 *除数* 是要除数的数值表达式。 可以是任何有效的数字数据类型分类，除 datetime 数据类型的数据类型之一的表达式。 
  
## <a name="return-type"></a>返回类型

返回具有更高的优先级参数的数据类型。 
  
如果整数*被除数*除以整数*除数*，结果将为整数类型的值已被截断结果的小数部分。 
  
## <a name="remarks"></a>说明

返回的实际值 / 运算符为除以第二个表达式的第一个表达式的商。
