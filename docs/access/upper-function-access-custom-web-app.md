# <a name="upper-function-access-custom-web-app"></a>Upper 函数 （访问自定义 web 应用程序）

返回带小写字符数据的文本表达式转换为大写。
  
> [!IMPORTANT]
> [!重要信息] Microsoft 不再建议在 SharePoint 中创建和使用 Access Web 应用程序。作为备选方法，请考虑使用 [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/)，生成适用于 Web 和移动设备的无代码业务解决方案。 
  
## <a name="syntax"></a>语法

 **右上**(*TextExpression*) 
  
**Upper**函数包含以下参数。 
  
|**参数名称**|**说明**|
|:-----|:-----|
| *TextExpression*  <br/> |文本表达式。  <br/> |
