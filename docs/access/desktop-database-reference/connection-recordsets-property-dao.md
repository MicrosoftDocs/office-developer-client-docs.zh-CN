# <a name="connectionrecordsets-property-dao"></a>Connection.Recordsets Property (DAO)


**适用于**： Access 2013 |Office 2013

返回一个 **Recordsets** 集合，该集合包含指定连接的所有已打开的记录集。只读。

## <a name="syntax"></a>语法

*表达式*。记录集

*表达式*代表**Connection**对象的变量。
