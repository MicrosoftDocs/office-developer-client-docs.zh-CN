# <a name="understanding-recordset-structure"></a>了解记录集结构


**适用于**： Access 2013 |Office 2013

每个 **Recordset** 都有一个 **Fields** 集合，该集合由一个或多个 **Field** 对象组成。 **Field** 对象通常表示表格列。下列主题将解释如何在 **Fields** 集合中浏览以及如何获得有关每个字段的信息，然后将讨论可通过 **Field** 对象使用哪种类型的信息以及如何使用它们。

  - [Fields 集合](the-fields-collection.md)

  - [Field 对象](the-field-object.md)

  - [使用 Recordset](working-with-recordsets.md)
