# <a name="documentdatecreated-property-dao"></a>Document.DateCreated 属性 (DAO)


**适用于**： Access 2013、 Office 2013

返回对象的创建日期和时间。只读 **Variant**。

## <a name="syntax"></a>语法

*表达式*。DateCreated

*表达式*一个代表**Document**对象的变量。
