# <a name="levelname-property-ado-md"></a>LevelName 属性 (ADO MD)


**适用于**：Access 2013、Office 2013

指示成员级别的名称。

## <a name="return-values"></a>返回值

返回一个 **String** 值，并且该值为只读。

## <a name="remarks"></a>注解

使用 **LevelName** 属性可检索成员所属级别的名称。该属性对应于 [Level](level-object-ado-md.md) 对象的 [Name](name-property-ado-md.md) 属性。
