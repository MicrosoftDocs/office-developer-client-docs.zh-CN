# <a name="querydeffields-property-dao"></a>QueryDef.Fields Property (DAO)


**适用于**： Access 2013、 Office 2013

返回一个 **[Fields](fields-collection-dao.md)** 集合，该集合表示指定对象的所有存储 **[Field](field-object-dao.md)** 对象。只读。

*表达式*。字段

*表达式*一个代表**QueryDef**对象的变量。
