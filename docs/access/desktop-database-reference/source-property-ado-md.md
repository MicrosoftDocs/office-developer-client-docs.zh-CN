# <a name="source-property-ado-md"></a>Source 属性 (ADO MD)


**适用于**： Access 2013、 Office 2013

指示单元格集中数据的来源。

## <a name="settings-and-return-values"></a>设置和返回值

设置或返回一个 **Variant** 值，该值对于关闭的 [Cellset](cellset-object-ado-md.md) 对象为可读写，对于打开的 **Cellset** 对象为只读。该 **Variant** 应包含有效的 **String** ，如 MDX 查询。
