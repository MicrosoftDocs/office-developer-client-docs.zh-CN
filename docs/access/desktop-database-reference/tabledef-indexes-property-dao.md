# <a name="tabledefindexes-property-dao"></a>TableDef.Indexes Property (DAO)


**适用于**： Access 2013、 Office 2013

返回一个 **Indexes** 集合，该集合包含指定表的所有已存储 **Index** 对象。只读。

## <a name="syntax"></a>语法

*表达式*。索引

*表达式*一个代表**TableDef**对象的变量。
