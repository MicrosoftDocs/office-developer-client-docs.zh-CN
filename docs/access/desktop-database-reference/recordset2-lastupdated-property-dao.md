# <a name="recordset2lastupdated-property-dao"></a>Recordset2.LastUpdated Property (DAO)


**适用于**： Access 2013、 Office 2013

返回基表的最近更改日期和时间。只读 **Variant**。

## <a name="syntax"></a>语法

*表达式*。LastUpdated

*表达式*一个表示**Recordset2**对象的变量。

## <a name="remarks"></a>注解

日期和时间的设置取自创建或上次更新基表的计算机。
