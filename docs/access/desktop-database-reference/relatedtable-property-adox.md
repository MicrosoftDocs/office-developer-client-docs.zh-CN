# <a name="relatedtable-property-adox"></a>RelatedTable 属性 (ADOX)


**适用于**： Access 2013 |Office 2013

指示相关表的名称。

## <a name="settings-and-return-values"></a>设置和返回值

设置和返回一个 **String** 值。

## <a name="remarks"></a>备注

默认值为空字符串 ("")。

如果键为外键，则 **RelatedTable** 为包含该键的表的名称。
