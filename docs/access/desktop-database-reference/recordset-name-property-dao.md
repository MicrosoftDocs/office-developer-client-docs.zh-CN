# <a name="recordsetname-property-dao"></a>Recordset.Name 属性 (DAO)


**适用于**： Access 2013、 Office 2013

返回指定对象的名称。只读 **String**。

## <a name="syntax"></a>语法

*表达式*。名称

*表达式*一个表示**Recordset**对象的变量。

## <a name="remarks"></a>注解

使用 SQL 语句打开的 **Recordset** 对象的 **Name** 属性是 SQL 语句的前 256 个字符。
