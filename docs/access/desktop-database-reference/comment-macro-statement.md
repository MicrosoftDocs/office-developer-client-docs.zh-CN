# <a name="comment-macro-statement"></a>Comment 宏语句


**适用于**： Access 2013、 Office 2013

**Comment** 语句可用于在宏中提供说明性注释。

## <a name="remarks"></a>注释

**Comment** 长度不能超过 1000 个字符。但默认情况下，只显示第一行注释。
