# <a name="indexfields-property-dao"></a>Index.Fields 属性 (DAO)


**适用于**： Access 2013、 Office 2013

返回一个 **Fields** 集合，该集合代表指定对象的所有存储的 **Field** 对象。可读/写。

## <a name="syntax"></a>语法

*表达式*。字段

*表达式*一个代表**Index**对象的变量。
