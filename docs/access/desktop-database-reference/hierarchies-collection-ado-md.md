# <a name="hierarchies-collection-ado-md"></a>Hierarchies 集合 (ADO MD)


**适用于**： Access 2013、 Office 2013

包含某一维度中的一系列 [Hierarchy](hierarchy-object-ado-md.md) 对象。

## <a name="remarks"></a>说明

**Hierarchies** 是一个标准 ADO 集合。使用集合的属性和方法，可以执行下列操作：

  - 使用 [Count](count-property-ado.md) 属性获取集合中的对象数。

  - 使用默认的 [Item](item-property-ado.md) 属性返回集合中的某个对象。

  - 使用 [Refresh](refresh-method-ado.md) 方法更新来自提供程序的集合中的对象。
