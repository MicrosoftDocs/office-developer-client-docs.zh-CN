# <a name="field2name-property-dao"></a>Field2.Name 属性 (DAO)


**适用于**： Access 2013、 Office 2013

返回或设置指定对象的名称。对于尚未追加到集合中的对象，该属性为可读/写 **String** 类型；对于已追加到集合中的对象，该属性为只读 **String** 类型。

## <a name="syntax"></a>语法

*表达式*。名称

*表达式*一个代表**Field2**对象的变量。

## <a name="remarks"></a>注解

**Field2** 对象名称的最大长度为 64 个字符。
