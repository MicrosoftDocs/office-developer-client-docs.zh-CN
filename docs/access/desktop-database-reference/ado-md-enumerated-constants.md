# <a name="ado-md-enumerated-constants"></a>ADO MD 枚举常量


**适用于**： Access 2013、 Office 2013

为帮助调试，ADO MD 枚举常量为每个常量各列出一个值。但是，该值纯粹是参考性的，并且可能随不同的 ADO MD 版本而变化。您的代码应只依赖于枚举常量的名称，而不是实际值。

定义了下列枚举。
