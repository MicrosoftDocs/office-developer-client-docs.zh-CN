# <a name="dbengineworkspaces-property-dao"></a>DBEngine.Workspaces 属性 (DAO)


**适用于**： Access 2013、 Office 2013

返回一个 **Workspaces** 集合，该集合包含所有可见的活动 **Workspace** 对象。只读。

## <a name="syntax"></a>语法

*表达式*。工作区

*表达式*一个代表**DBEngine**对象的变量。
