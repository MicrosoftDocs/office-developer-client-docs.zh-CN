# <a name="uniquename-property-ado-md"></a>UniqueName 属性 (ADO MD)


**适用于**： Access 2013、 Office 2013

指示当前对象的明确的名称。

## <a name="return-values"></a>返回值

返回一个 **String** 值，并且该值为只读。
