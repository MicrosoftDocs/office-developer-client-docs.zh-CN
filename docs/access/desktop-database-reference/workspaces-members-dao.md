# <a name="workspaces-members-dao"></a>Workspaces Members (DAO)


**适用于**： Access 2013、 Office 2013

Workspaces 集合包含 DBEngine 对象的所有活动、未隐藏的 Workspace 对象。（隐藏的 Workspace 对象并不追加到集合中，而是通过它们分配到的变量进行引用）。
