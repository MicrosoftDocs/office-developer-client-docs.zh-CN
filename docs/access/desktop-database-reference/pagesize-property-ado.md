**适用于**： Access 2013 |Office 2013

指示由多少记录构成 [Recordset](recordset-object-ado.md) 中的一页。

=======
## <a name="settings-and-return-values"></a>设置和返回值

设置或返回一个 **Long** 值，指示页上的记录数。默认值为 10。
