1.  <span data-ttu-id="dfd31-p105">在命令提示符下运行以下语句。您必须设置路径以包括 Microsoft SDK for Java Bin 目录，或从该位置运行命令。一般情况下，Microsoft SDK for Java 的安装位置与 Visual Studio 一样。以下语句为单个命令语句。</span><span class="sxs-lookup"><span data-stu-id="dfd31-p105">Run the following from a command prompt.     ```
