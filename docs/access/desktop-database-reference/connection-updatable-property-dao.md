# <a name="connectionupdatable-property-dao"></a>连接可更新属性 (DAO)


**适用于**：Access 2013、Office 2013

返回一个值，该值指示是否可以更改 DAO 对象。 **Boolean** 类型，只读。

## <a name="syntax"></a>语法

*表达式*。更新

*表达式*一个返回**Connection**对象的表达式。
