# <a name="view-an-activex-controls-about-box"></a>查看 ActiveX 控件的关于框

**适用于**： Access 2013 |Office 2013

若要查看**有关**框中显示的版本和 ActiveX 控件的版权信息，请单击 Microsoft Access 属性表中的**关于**属性框。 然后单击属性框右侧的**生成**按钮。
