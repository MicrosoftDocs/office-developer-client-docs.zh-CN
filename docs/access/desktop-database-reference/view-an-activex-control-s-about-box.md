# <a name="view-an-activex-control-about-box"></a>查看有关框的一个 ActiveX 控件

**适用于**： Access 2013、 Office 2013

若要查看**有关**框中显示的一个 ActiveX 控件的版本和版权信息：

1. 选择**关于**属性框的 Microsoft Access 属性表中。

2. 选择属性框右侧的**生成**按钮。
