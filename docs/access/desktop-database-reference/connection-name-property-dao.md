# <a name="connectionname-property-dao"></a>Connection.Name Property (DAO)


**适用于**： Access 2013 |Office 2013

返回 **[Connection](connection-object-dao.md)** 的名称。

## <a name="syntax"></a>语法

*表达式*。名称

*表达式*代表**Connection**对象的变量。

## <a name="remarks"></a>注解

可以将对象的 **Name** 属性与代码中的 Visual Basic for Applications **Dim** 语句一起使用，以创建该对象的其他实例。
