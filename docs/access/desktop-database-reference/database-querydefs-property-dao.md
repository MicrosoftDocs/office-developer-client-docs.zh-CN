# <a name="databasequerydefs-property-dao"></a>Database.QueryDefs 属性 (DAO)


**适用于**： Access 2013、 Office 2013

返回一个 **QueryDefs** 集合，该集合包含指定数据库的所有 **QueryDef** 对象。只读。

## <a name="syntax"></a>语法

*表达式*。QueryDefs

*表达式*一个代表**Database**对象的变量。
