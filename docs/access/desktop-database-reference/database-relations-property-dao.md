# <a name="databaserelations-property-dao"></a>数据库关系属性 (DAO)


**适用于**：Access 2013、Office 2013

返回一个 **Relations** 集合，该集合包含指定数据库的所有存储的 **Relation** 对象。 此为只读属性。

## <a name="syntax"></a>语法

*表达式*。公关

*表达式*一个代表**Database**对象的变量。
