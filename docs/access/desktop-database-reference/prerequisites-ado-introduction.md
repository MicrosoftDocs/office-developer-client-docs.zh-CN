# <a name="prerequisites-ado-introduction"></a>Prerequisites (ADO Introduction)

**适用于**： Access 2013、 Office 2013

*ADO 程序员指南 》* 将很有用到开发人员提供多种背景。 读者应当至少具有使用 Microsoft Visual Basic 进行应用程序开发的中级经验，因为指南中的大多数示例都是用该语言编写的。 其他示例 Microsoft Visual c + +、 Java、 Visual Basic、 Scripting Edition (VBScript) 中，和 Microsoft JScript 中写入。

由于 ADO 用于访问多种数据源中的数据，因此读者可能还需要对基础关系数据库管理系统概念、联机分析处理 (OLAP) 概念有一定的了解，并且基本熟悉 Internet 和 Internet 协议。

ADO 属于 Microsoft 数据访问 (UDA) 策略。 （有关 UDA 的详细信息，请参阅[在 Microsoft 数据访问的 ADO 角色](the-role-of-ado-in-microsoft-data-access.md)，本章后面部分。）因此，它与互操作的 OLE DB 技术。 OLE DB 以 Microsoft 组件对象模型 (COM) 为基础。 因此，为了理解 ADO 中的某些更高级概念，熟悉 COM 也是有用的。
