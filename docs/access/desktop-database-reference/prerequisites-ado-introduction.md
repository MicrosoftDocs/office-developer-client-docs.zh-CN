# <a name="prerequisites-ado-introduction"></a>Prerequisites (ADO Introduction)


**适用于**： Access 2013、 Office 2013

*ADO 程序员指南 》* 将很有用到开发人员提供多种背景。 读者应当至少具有使用 Microsoft Visual Basic 进行应用程序开发的中级经验，因为指南中的大多数示例都是用该语言编写的。 其他示例则是用 Microsoft Visual C++、Java、Visual Basic Scripting Edition (VBScript) 和 Microsoft JScript 编写的。

由于 ADO 用于访问多种数据源中的数据，因此读者可能还需要对基础关系数据库管理系统概念、联机分析处理 (OLAP) 概念有一定的了解，并且基本熟悉 Internet 和 Internet 协议。

ADO 是 Microsoft 数据访问 (UDA) 策略的一部分（有关 UDA 的详细信息，请参阅本章随后的主题 [ADO 在 Microsoft 数据访问中的角色](the-role-of-ado-in-microsoft-data-access.md)）。它以这种角色与 OLE DB 技术进行交互。OLE DB 以 Microsoft 组件对象模型 (COM) 为基础。因此，为了理解 ADO 中的某些更高级概念，熟悉 COM 也是有用的。
