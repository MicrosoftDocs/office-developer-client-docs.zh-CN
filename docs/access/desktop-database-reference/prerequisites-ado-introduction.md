# <a name="prerequisites-ado-introduction"></a>先决条件 (ADO 简介)

**适用于**：Access 2013、Office 2013

《 *ADO 程序员指南》* 将为具有多种背景的开发人员提供有用的证明。 读者应当至少具有使用 Microsoft Visual Basic 进行应用程序开发的中级经验，因为指南中的大多数示例都是用该语言编写的。 其他示例编写在 Microsoft Visual c + +、Java、Visual Basic、脚本编写版 (VBScript) 和 microsoft JScript 中。

由于 ADO 用于访问多种数据源中的数据，因此读者可能还需要对基础关系数据库管理系统概念、联机分析处理 (OLAP) 概念有一定的了解，并且基本熟悉 Internet 和 Internet 协议。

ADO 是 Microsoft 数据访问 (UDA) 策略的一部分（有关 UDA 的详细信息，请参阅本章随后的主题 ADO 在 Microsoft 数据访问中的角色）。 (有关 UDA 的详细信息, 请参阅本章后面[的 ADO 在 Microsoft 数据访问中的角色](the-role-of-ado-in-microsoft-data-access.md)。)因此, 它与 OLE DB 技术互操作。 OLE DB 以 Microsoft 组件对象模型 (COM) 为基础。 因此，为了理解 ADO 中的某些更高级概念，熟悉 COM 也是有用的。
