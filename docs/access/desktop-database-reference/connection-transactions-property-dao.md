# <a name="connectiontransactions-property-dao"></a>Connection.Transactions 属性 (DAO)


**适用于**： Access 2013、 Office 2013

返回一个值，该值指示对象是否支持事务。只读 **Boolean** 类型。

## <a name="syntax"></a>语法

*表达式*。事务

*表达式*代表**Connection**对象的变量。
