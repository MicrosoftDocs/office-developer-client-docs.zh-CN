# <a name="propertiescount-property-dao"></a>Properties.Count 属性 (DAO)


**适用于**： Access 2013、 Office 2013

使用 **Count** 属性可以确定指定集合中的项目个数。 **Long** 类型，只读。

## <a name="syntax"></a>语法

*表达式*。计数

*表达式*一个代表**Properties**对象的变量。
