# <a name="propertiescount-property-dao"></a>Properties 属性 (DAO)


**适用于**：Access 2013、Office 2013

**Count** 属性可用于确定指定集合中项的数目。 **Long** 类型，只读。

## <a name="syntax"></a>语法

*表达式*一个代表**Properties**对象的变量。
