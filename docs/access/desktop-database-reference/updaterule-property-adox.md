# <a name="updaterule-property-adox"></a>UpdateRule 属性 (ADOX)


**适用于**： Access 2013、 Office 2013

指示更新主键时执行的操作。

## <a name="settings-and-return-values"></a>设置和返回值

设置和返回一个 **Long** 值，该值可为 [RuleEnum](ruleenum.md) 常量之一。默认值为 **adRINone** 。

## <a name="remarks"></a>备注

对于已追加到集合中的 [Key](key-object-adox.md) 对象，此属性为只读。
