# <a name="databaserecordsets-property-dao"></a>Database. recordset 属性 (DAO)


**适用于**：Access 2013、Office 2013

返回一个 **Recordsets** 集合，该集合包含指定数据库的所有已打开的记录集。 此为只读属性。

## <a name="syntax"></a>语法

*表达式*。集

*表达式*一个代表**Database**对象的变量。
