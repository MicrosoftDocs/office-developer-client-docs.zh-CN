# <a name="workspacedatabases-property-dao"></a>Workspace.Databases 属性 (DAO)


**适用于**： Access 2013、 Office 2013

返回一个 **Databases** 集合，该集合代表指定 **Workspace** 中的已打开的数据库。只读。

## <a name="syntax"></a>语法

*表达式*。数据库

*表达式*一个代表**Workspace**对象的变量。
