# <a name="recordsetsrefresh-method-dao"></a>Recordsets.Refresh Method (DAO)


**适用于**： Access 2013 |Office 2013

不支持此对象。

## <a name="syntax"></a>语法

*表达式*。刷新

*表达式*一个代表**Recordsets**对象的变量。

## <a name="remarks"></a>注解

**Refresh** 方法不能用于非永久的集合，例如 **Connections**、 **Databases**、 **Recordsets** 、 **Workspaces** 或 **Connection** 对象的 **QueryDefs** 集合。
