# <a name="databasetabledefs-property-dao"></a>Database.TableDefs 属性 (DAO)


**适用于**： Access 2013、 Office 2013

返回一个 **TableDefs** 集合，该集合包含指定数据库中存储的所有 **TableDef** 对象。只读。

## <a name="syntax"></a>语法

*表达式*。TableDefs

*表达式*一个代表**Database**对象的变量。
