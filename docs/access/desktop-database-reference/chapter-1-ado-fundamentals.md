# <a name="chapter-1-ado-fundamentals"></a>第 1 章：ADO 基础知识


**适用于**： Access 2013 |Office 2013

本章对 ADO 库进行简单介绍，将讨论 ADO 的用途，介绍 ADO 层次结构中的对象，并展示一个使用很多 ADO 对象来检索、编辑、更新数据源中的数据的简单 ADO 示例应用程序。最后，本章将介绍对于了解编写 ADO 应用程序非常重要的两个问题：OLE DB 提供程序和错误。
