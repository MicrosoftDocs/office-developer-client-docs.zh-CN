# <a name="containerdocuments-property-dao"></a>Container.Documents 属性 (DAO)


**适用于**： Access 2013、 Office 2013

返回一个 **Documents** 集合，该集合包含指定对象中存储的所有文档。只读。

## <a name="syntax"></a>语法

*表达式*。文档

*表达式*代表**容器**对象的变量。
