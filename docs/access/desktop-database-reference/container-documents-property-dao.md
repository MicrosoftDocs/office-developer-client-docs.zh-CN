# <a name="containerdocuments-property-dao"></a>"Documents" 属性 (DAO)


**适用于**：Access 2013、Office 2013

返回一个 **Documents** 集合，该集合包含指定对象中存储的所有文档。 此为只读属性。

## <a name="syntax"></a>语法

*表达式*。单据

*表达式*一个代表**Container**对象的变量。
