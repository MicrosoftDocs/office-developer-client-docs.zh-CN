# <a name="exitforeachrecord-macro-action"></a>ExitForEachRecord 宏操作


**适用于**：Access 2013、Office 2013

使用 **ExitForEachRecord** 操作可立即退出 **[ForEachRecord](foreachrecord-data-block.md)** 数据块。


> [!NOTE]
> [!注释] **ExitForEachRecord** 操作仅适用于数据宏。
