# <a name="internet-server-error-access-denied"></a>Internet 服务器错误： 访问被拒绝


**适用于**： Access 2013 |Office 2013

如果出现此错误消息，通常表示 Microsoft Internet 信息服务 (IIS) 返回以下状态：

HTTP\_状态\_拒绝 401
