# <a name="recordset2name-property-dao"></a>Recordset2.Name 属性 (DAO)


**适用于**：Access 2013、Office 2013

返回指定对象的名称。 只读的 **字符串** 。

## <a name="syntax"></a>语法

*表达式*。别名

*表达式*一个代表**Recordset2**对象的变量。

## <a name="remarks"></a>注解

使用 SQL 语句打开的 **Recordset** 对象的 **Name** 属性是 SQL 语句的前 256 个字符。
