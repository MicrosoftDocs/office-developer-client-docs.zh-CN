# <a name="connectionrecordsaffected-property-dao"></a>RecordsAffected 属性 (DAO)


**适用于**：Access 2013、Office 2013

返回最近调用的 **[Execute](connection-execute-method-dao.md)** 方法所影响的记录数。

## <a name="syntax"></a>语法

*表达式*。RecordsAffected

*表达式*一个代表**Connection**对象的变量。
