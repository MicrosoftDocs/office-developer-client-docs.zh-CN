# <a name="tabledefname-property-dao"></a>TableDef.Name 属性 (DAO)


**适用于**： Access 2013、 Office 2013

返回或设置指定对象的名称。可读/写 **String** 类型。

## <a name="syntax"></a>语法

*表达式*。名称

*表达式*一个代表**TableDef**对象的变量。

## <a name="remarks"></a>注解

**TableDef** 对象名称的最大长度为 64 个字符。
