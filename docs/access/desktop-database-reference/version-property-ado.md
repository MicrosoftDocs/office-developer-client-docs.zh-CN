# <a name="version-property-ado"></a>Version 属性 (ADO)


**适用于**： Access 2013、 Office 2013

指示 ADO 版本号。

## <a name="return-value"></a>返回值

返回一个指示版本的 **String** 值。

## <a name="remarks"></a>备注

使用 **Version** 属性可以返回 ADO 实现的版本号。

提供程序的版本可作为 [Properties](properties-collection-ado.md) 集合中的动态属性。
