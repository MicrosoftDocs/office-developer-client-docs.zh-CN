# <a name="querydefparameters-property-dao"></a>QueryDef.Parameters 属性 (DAO)


**适用于**： Access 2013、 Office 2013

返回一个 **[Parameters](parameters-collection-dao.md)** 集合，该集合包含指定 [QueryDef](parameter-object-dao.md) 的所有 ****Parameter**** 对象。只读。

## <a name="syntax"></a>语法

*表达式*。参数

*表达式*一个代表**QueryDef**对象的变量。
