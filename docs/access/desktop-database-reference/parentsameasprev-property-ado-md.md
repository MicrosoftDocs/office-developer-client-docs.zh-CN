# <a name="parentsameasprev-property-ado-md"></a>ParentSameAsPrev 属性 (ADO MD)


**适用于**：Access 2013、Office 2013

指示此位置成员的父级是否与紧挨的前一个成员的父级相同。

## <a name="return-values"></a>返回值

返回一个 **Boolean** 值，并且是只读的。

## <a name="remarks"></a>注解

此属性仅在 [Position](position-object-ado-md.md) 对象所属的 [Member](member-object-ado-md.md) 对象上受支持。从 [Level](level-object-ado-md.md) 对象所属的 **Member** 对象引用此属性时，会发生错误。
