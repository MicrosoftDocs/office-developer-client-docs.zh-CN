# <a name="databasecontainers-property-dao"></a>Database.Containers Property (DAO)


**适用于**： Access 2013、 Office 2013

返回一个 **Containers** 集合，该集合代表指定数据库中的所有 **Container** 对象。只读。

## <a name="syntax"></a>语法

*表达式*。容器

*表达式*一个代表**Database**对象的变量。
