# <a name="dbengineerrors-property-dao"></a>DBEngine.Errors 属性 (DAO)


**适用于**： Access 2013、 Office 2013

返回一个 **Errors** 集合，该集合包含指定对象的所有存储的 **Error** 对象。只读。

*表达式*。错误
