# <a name="workspaceconnections-property-dao"></a>Workspace.Connections Property (DAO)


**适用于**： Access 2013、 Office 2013

返回一个 **Connections** 集合，该集合代表指定 **Workspace** 中的当前连接。只读。

## <a name="syntax"></a>语法

*表达式*。连接

*表达式*一个代表**Workspace**对象的变量。
