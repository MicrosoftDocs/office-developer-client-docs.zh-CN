# <a name="updating-joined-results-unique-table"></a>更新联接结果：Unique Table


**适用于**：Access 2013、Office 2013

## <a name="updating-joined-results-unique-table"></a>更新 JOINed 结果：唯一表

对于以前使用 **Unique Table** 动态属性针对多个基表执行 JOIN 操作而形成的 **Recordset**，可以使用 ADO 来严格控制对特定基表进行的修改。 有关使用**unique table**的详细信息, 请参阅 ADO 程序员参考主题中的**独特表**和**更新再次同步**动态属性。
