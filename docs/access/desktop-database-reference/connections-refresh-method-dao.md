# <a name="connectionsrefresh-method-dao"></a>连接。 Refresh 方法 (DAO)


**适用于**：Access 2013、Office 2013

**[Connections](connections-collection-dao.md)** 集合不支持。

**Refresh** 方法不能用于非永久的集合（例如 **Connections**、 **Databases**、 **Recordsets** 和 **Workspaces**）或 **Connection** 对象的 **QueryDefs** 集合。
