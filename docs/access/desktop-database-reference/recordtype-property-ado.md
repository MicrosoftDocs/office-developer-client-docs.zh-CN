# <a name="recordtype-property-ado"></a>RecordType 属性 (ADO)


**适用于**： Access 2013、 Office 2013

指示 [Record](record-object-ado.md) 对象的类型。

## <a name="return-value"></a>返回值

返回一个 [RecordTypeEnum](recordtypeenum.md) 值。

## <a name="remarks"></a>备注

**RecordType** 为只读属性。
