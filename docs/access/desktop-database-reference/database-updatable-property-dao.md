# <a name="databaseupdatable-property-dao"></a>数据库可更新属性 (DAO)


**适用于**：Access 2013、Office 2013

返回一个值，该值指示是否可以更改 DAO 对象。 只读 **Boolean** 类型。

## <a name="syntax"></a>语法

*表达式*。更新

*表达式*一个代表**Database**对象的变量。
