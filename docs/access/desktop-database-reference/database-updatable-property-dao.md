# <a name="databaseupdatable-property-dao"></a>Database.Updatable Property (DAO)


**适用于**： Access 2013、 Office 2013

返回一个值，该值指示是否可以更改 DAO 对象。只读 **Boolean**。

## <a name="syntax"></a>语法

*表达式*。可更新

*表达式*一个代表**Database**对象的变量。
