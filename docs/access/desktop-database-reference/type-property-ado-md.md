# <a name="type-property-ado-md"></a>Type 属性 (ADO MD)


**适用于**： Access 2013、 Office 2013

指示当前成员的类型。

## <a name="return-values"></a>返回值

返回一个 [MemberTypeEnum](membertypeenum.md) 值，并且该值为只读。

## <a name="remarks"></a>备注

此属性仅在 [Level](member-object-ado-md.md) 对象所属的 [Member](level-object-ado-md.md) 对象上受支持。从 **Position** 对象所属的 [Member](position-object-ado-md.md) 对象引用此属性时，会发生错误。
