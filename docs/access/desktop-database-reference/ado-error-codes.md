# <a name="ado-error-codes"></a>ADO 错误代码


**适用于**： Access 2013、 Office 2013

除了 [Errors](error-object-ado.md) 集合的 [Error](errors-collection-ado.md) 对象中返回的提供程序错误以外，ADO 本身也可以将错误返回到运行时环境的异常处理机制。使用编程语言的捕获错误机制（如 Microsoft Visual Basic 中的 **On Error** 语句，Microsoft Visual C++ 或 Microsoft Visual J++ 中的 **try-catch** 块），可以捕获 ADO 错误。

有关 ADO 错误代码的列表，请参阅 [ErrorValueEnum](errorvalueenum.md)。
