# <a name="databasename-property-dao"></a>Database.Name 属性 (DAO)


**适用于**： Access 2013、 Office 2013

返回指定对象的名称。只读 **String**。

## <a name="syntax"></a>语法

*表达式*。名称

*表达式*一个代表**Database**对象的变量。
