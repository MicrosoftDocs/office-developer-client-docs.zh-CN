# <a name="connectionquerydefs-property-dao"></a>QueryDefs 属性 (DAO)


**适用于**：Access 2013、Office 2013

返回一个 **QueryDefs** 集合，该集合包含指定连接的所有 **QueryDef** 对象。 此为只读属性。

## <a name="syntax"></a>语法

*表达式*。QueryDefs

*表达式*一个代表**Connection**对象的变量。
