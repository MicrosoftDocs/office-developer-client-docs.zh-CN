# <a name="programming-with-ado-md"></a>ADO MD 编程


**适用于**： Access 2013、 Office 2013

若要将 ADO MD 与您的开发工具一起使用，就应建立对 ADO MD 类型库的引用。ADO MD 库的描述为"Microsoft ActiveX Data Objects (Multi-dimensional) Library"。ADO MD 库的文件名为 msadomd.dll，程序 ID (ProgID) 为"ADOMD"。有关如何建立对库的引用的更多信息，请参阅您的开发工具的文档。
