# <a name="step-1-set-up-the-application"></a>步骤 1：设置应用程序


**适用于**： Access 2013 |Office 2013

## <a name="step-1-set-up-the-application"></a>步骤 1：设置应用程序

创建一个具有脚本权限的名为“XMLPersist”的 IIS 虚拟目录。在该虚拟目录所指向的文件夹中新建两个文本文件：一个名为“XMLResponse.asp”，另一个名为“Default.htm”。
