# <a name="position-object-ado-md"></a>Position 对象 (ADO MD)


**适用于**： Access 2013、 Office 2013

表示由一个成员或不同维度的多个成员组成的成员集，该成员集定义了轴上的某个点。

## <a name="remarks"></a>说明

使用 **Position** 对象的集合和属性，可以执行下列操作：

  - 使用 **Ordinal** 属性返回沿 **Axis** 的 [Position](axis-object-ado-md.md) 的序号位置。

  - 使用 [Members](members-collection-ado-md.md) 集合返回构成沿 **Axis** 的位置的成员。
