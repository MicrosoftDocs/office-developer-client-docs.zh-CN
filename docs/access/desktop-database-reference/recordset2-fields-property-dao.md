# <a name="recordset2fields-property-dao"></a>Recordset2 属性 (DAO)


**适用于**：Access 2013、Office 2013

返回一个 **Fields** 集合，该集合表示指定对象的所有存储 **Field** 对象。 此为只读属性。

## <a name="syntax"></a>语法

*表达式*。字段

*表达式*一个代表**Recordset2**对象的变量。
