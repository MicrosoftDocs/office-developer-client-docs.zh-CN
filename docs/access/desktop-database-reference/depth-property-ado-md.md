# <a name="depth-property-ado-md"></a>Depth 属性 (ADO MD)


**适用于**： Access 2013、 Office 2013

指示 [Level](level-object-ado-md.md) 和层次结构的根之间的级数。

## <a name="return-values"></a>返回值

返回一个整数，并且该值为只读。

## <a name="remarks"></a>说明

层次结构的根所在 **Level** 的 **Depth** 值为零 (0)。
