# <a name="document-members-dao"></a>Document Members (DAO)


**适用于**： Access 2013 |Office 2013

Document 对象包含有关对象实例的信息。该对象可以是数据库、保存的表、查询或关系（仅适用于 Microsoft Access 数据库引擎数据库）。
