# <a name="undorecord-macro-action-access-custom-web-app"></a>UndoRecord 宏操作 (Access 自定义 web 应用程序)

放弃对当前记录所做的更改。
  
> [!IMPORTANT]
> Microsoft 不再建议在 SharePoint 中创建和使用 Access Web 应用程序。作为备选方法，请考虑使用 [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/)，生成适用于 Web 和移动设备的无代码业务解决方案。 
  
## <a name="remarks"></a>说明

**UndoRecord**操作等效于使用操作栏中的 "**取消**" 按钮放弃更改。
