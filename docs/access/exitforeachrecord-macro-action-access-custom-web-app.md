# <a name="exitforeachrecord-macro-action-access-custom-web-app"></a>ExitForEachRecord 宏操作 （访问自定义 web 应用程序）

使用 **ExitForEachRecord** 操作可立即退出 **[ForEachRecord](foreachrecord-data-block-access-custom-web-app.md)** 数据块。 
  
> [!IMPORTANT]
> Microsoft 不再建议在 SharePoint 中创建和使用 Access Web 应用程序。作为备选方法，请考虑使用 [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/)，生成适用于 Web 和移动设备的无代码业务解决方案。 
  
> [!NOTE]
> [!注释] **ExitForEachRecord** 操作仅适用于数据宏。
