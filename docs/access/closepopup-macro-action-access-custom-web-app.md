# <a name="closepopup-macro-action-access-custom-web-app"></a>ClosePopup 宏操作 (Access 自定义 web 应用程序)

您可以使用**ClosePopup**操作关闭活动文档弹出窗口视图。 
  
> [!IMPORTANT]
> [!重要信息] Microsoft 不再建议在 SharePoint 中创建和使用 Access Web 应用程序。作为备选方法，请考虑使用 [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/)，生成适用于 Web 和移动设备的无代码业务解决方案。 
  
## <a name="remarks"></a>注解

如果弹出窗口中的当前记录处于编辑模式, 将自动保存对记录所做的更改。
