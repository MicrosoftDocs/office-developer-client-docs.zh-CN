# <a name="closepopup-macro-action-access-custom-web-app"></a>ClosePopup 宏操作 (Access 自定义 Web 应用) 

可以使用 **ClosePopup** 操作关闭活动文档弹出视图。 
  
> [!IMPORTANT]
> Microsoft 不再建议在 SharePoint 中创建和使用 Access Web 应用程序。作为备选方法，请考虑使用 [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/)，生成适用于 Web 和移动设备的无代码业务解决方案。 
  
## <a name="remarks"></a>备注

如果弹出视图中的当前记录在编辑模式下，则会自动保存对记录的更改。
