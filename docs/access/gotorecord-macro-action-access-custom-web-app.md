# <a name="gotorecord-macro-action-access-custom-web-app"></a>GoToRecord 宏操作 （访问自定义 web 应用程序）

您可以使用**GoToRecord**操作使视图中指定的记录成为当前记录。 
  
> [!IMPORTANT]
> Microsoft 不再建议在 SharePoint 中创建和使用 Access Web 应用程序。作为备选方法，请考虑使用 [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/)，生成适用于 Web 和移动设备的无代码业务解决方案。 
  
## <a name="setting"></a>设置

**GoToRecord** 操作具有下列参数。 
  
|**操作参数**|**说明**|
|:-----|:-----|
|**Record** <br/> |要使成为当前记录的记录。 选择**上一个**、**下一步**，**第一个**或**最后一个**。 默认值为**下一步**。  <br/> |
