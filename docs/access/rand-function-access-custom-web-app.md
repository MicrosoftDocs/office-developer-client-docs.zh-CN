# <a name="rand-function-access-custom-web-app"></a>Rand 函数 （访问自定义 web 应用程序）

返回一个伪随机数字 0 与 1 之间。
  
> [!IMPORTANT]
> Microsoft 不再建议在 SharePoint 中创建和使用 Access Web 应用程序。作为备选方法，请考虑使用 [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/)，生成适用于 Web 和移动设备的无代码业务解决方案。 
  
## <a name="syntax"></a>语法

 **Rand**（[*种子*]） 
  
**Rand**函数包含以下参数。 
  
|**参数名称**|**说明**|
|:-----|:-----|
| *Seed*  <br/> |提供的种子值整数表达式。 如果未指定*种子*，随机分配种子值。  <br/> |
   
## <a name="remarks"></a>说明

重复调用具有相同的种子的**Rand**函数返回相同的结果。
