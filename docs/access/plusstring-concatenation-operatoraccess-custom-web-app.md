# <a name="-string-concatenation-operator-access-custom-web-app"></a>+ (字符串连接运算符)  (Access 自定义 Web 应用) 

+ (加) 是启用字符串串联的字符串连接运算符。 所有其他字符串操作都通过使用字符串函数（如 **Substring）进行处理**。 
  
> [!IMPORTANT]
> Microsoft 不再建议在 SharePoint 中创建和使用 Access Web 应用程序。作为备选方法，请考虑使用 [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/)，生成适用于 Web 和移动设备的无代码业务解决方案。 
  
## <a name="syntax"></a>语法

 *TextExpression* **+***TextExpression* 
  
 *TextExpression*  任何有效的文本表达式。
