# <a name="newrecord-macro-action-access-custom-web-app"></a>NewRecord 宏操作 (Access 自定义 web 应用程序)

在当前视图中创建一个新的空白记录。
  
> [!IMPORTANT]
> [!重要信息] Microsoft 不再建议在 SharePoint 中创建和使用 Access Web 应用程序。作为备选方法，请考虑使用 [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/)，生成适用于 Web 和移动设备的无代码业务解决方案。 
  
## <a name="remarks"></a>注解

您可以使用**NewRecord**操作在视图的末尾添加空白记录, 以便可以输入新数据。 
  
**NewRecord**操作等效于使用操作栏中的 "**添加**" 按钮添加记录。
