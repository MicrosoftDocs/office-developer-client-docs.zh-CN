# <a name="len-function-access-custom-web-app"></a>Len 函数 （访问自定义 web 应用程序）

返回指定的字符串表达式，不包括尾随空白的字符数。
  
> [!IMPORTANT]
> [!重要信息] Microsoft 不再建议在 SharePoint 中创建和使用 Access Web 应用程序。作为备选方法，请考虑使用 [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/)，生成适用于 Web 和移动设备的无代码业务解决方案。 
  
## <a name="syntax"></a>语法

 **Len**(*TextExpression*) 
  
**Len**函数包含以下参数。 
  
|**参数名称**|**说明**|
|:-----|:-----|
| *TextExpression*  <br/> |字符串表达式。  <br/> |
