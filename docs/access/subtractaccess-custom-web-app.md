# <a name="--subtract-access-custom-web-app"></a>- (Access)  (自定义 Web 应用) 

减去两个数字。 此算术减法运算符还可以从日期中减去一个数字（以天表示）。
  
> [!IMPORTANT]
> Microsoft 不再建议在 SharePoint 中创建和使用 Access Web 应用程序。作为备选方法，请考虑使用 [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/)，生成适用于 Web 和移动设备的无代码业务解决方案。 
  
## <a name="syntax"></a>语法

 *表达式*   -  *表达式* 
  
 *表达式* 数字类别中任何一种数据类型的任何有效表达式，位值除外 **数据类型。** 不能与日期或时间数据类型一同使用。 
  
## <a name="return-value"></a>返回值

返回数据类型优先级较高的参数的变量。
