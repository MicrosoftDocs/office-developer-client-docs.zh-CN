# <a name="not-access-custom-web-app"></a>不是 (Access 自定义 web 应用)

对布尔输入求反。
  
> [!IMPORTANT]
> Microsoft 不再建议在 SharePoint 中创建和使用 Access Web 应用程序。作为备选方法，请考虑使用 [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/)，生成适用于 Web 和移动设备的无代码业务解决方案。 
  
## <a name="syntax"></a>语法

[**非**] *BooleanExpression* 
  
**Not**运算符包含下列参数。 
  
|**参数名称**|**说明**|
|:-----|:-----|
| *BooleanExpression*  <br/> |一个有效的布尔表达式。  <br/> |
   
## <a name="remarks"></a>说明

下表显示了使用**Not**运算符比较 TRUE 和 FALSE 值的结果。 
  
||**Not**|
|:-----|:-----|
|**TRUE** <br/> |False  <br/> |
|**该值** <br/> |True  <br/> |
