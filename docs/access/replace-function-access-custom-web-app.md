# <a name="replace-function-access-custom-web-app"></a>Replace 函数 （访问自定义 web 应用程序）

使用另一个字符串值替换所有出现指定的字符串值。
  
> [!IMPORTANT]
> Microsoft 不再建议在 SharePoint 中创建和使用 Access Web 应用程序。作为备选方法，请考虑使用 [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/)，生成适用于 Web 和移动设备的无代码业务解决方案。 
  
## <a name="syntax"></a>语法

 **替换**（*TextExpression*，*模式*中，*替换*） 
  
**Replace**函数包含以下参数。 
  
|**参数名称**|**说明**|
|:-----|:-----|
| *TextExpression*  <br/> |要搜索的字符串表达式。  <br/> |
| *Pattern*  <br/> |要找的子字符串。  *模式*不能为空字符串 ("")。  <br/> |
| *Replacement*  <br/> |替换字符串。  <br/> |
