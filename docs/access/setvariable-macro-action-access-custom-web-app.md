# <a name="setvariable-macro-action-access-custom-web-app"></a>SetVariable Macro Action (Access 自定义 Web app) 

可以使用 **SetVariable** 操作创建一个临时变量，并设置为特定值。 然后，可以在后续操作中将变量用作条件或参数，也可以将变量用于 UI 或宏 (用户界面) 变量。 
  
> [!IMPORTANT]
> Microsoft 不再建议在 SharePoint 中创建和使用 Access Web 应用程序。作为备选方法，请考虑使用 [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/)，生成适用于 Web 和移动设备的无代码业务解决方案。 
  
## <a name="setting"></a>设置

**SetVariable** 操作具有下列参数。 
  
|**操作参数**|**说明**|
|:-----|:-----|
|"变量" <br/> |请输入临时变量的名称。  <br/> |
|**值 =** <br/> |请输入将用来设置此临时变量的值的表达式。 不要将等号的表达式 () **=** 表达式。 You can click the **Build** button ![Formula](media/buildbut_ZA06047218.gif "Formula") to use the Expression Builder to set this argument.  <br/> |
