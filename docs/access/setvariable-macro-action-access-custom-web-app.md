# <a name="setvariable-macro-action-access-custom-web-app"></a>SetVariable 宏操作 (Access 自定义 web 应用程序)

您可以使用**SetVariable**操作创建一个临时变量并将其设置为特定值。 然后, 可以将该变量用作后续操作中的条件或参数, 也可以在另一个用户界面 (UI) 宏中使用该变量。 
  
> [!IMPORTANT]
> Microsoft 不再建议在 SharePoint 中创建和使用 Access Web 应用程序。作为备选方法，请考虑使用 [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/)，生成适用于 Web 和移动设备的无代码业务解决方案。 
  
## <a name="setting"></a>设置

**SetVariable**操作具有下列参数。 
  
|**操作参数**|**说明**|
|:-----|:-----|
|"变量" <br/> |请输入临时变量的名称。  <br/> |
|**值 =** <br/> |请输入将用来设置此临时变量的值的表达式。 不要在表达式前面加等号 (**=** )。 可以单击 "**生成**" 按钮![公式](media/buildbut_ZA06047218.gif "公式")以使用表达式生成器设置此参数。  <br/> |
