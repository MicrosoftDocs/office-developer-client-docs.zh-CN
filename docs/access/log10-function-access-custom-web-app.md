# <a name="log10-function-access-custom-web-app"></a>Log10 函数 （访问自定义 web 应用程序）

返回以 10 为底的对数。
  
> [!IMPORTANT]
> Microsoft 不再建议在 SharePoint 中创建和使用 Access Web 应用程序。作为备选方法，请考虑使用 [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/)，生成适用于 Web 和移动设备的无代码业务解决方案。 
  
## <a name="syntax"></a>语法

 **Log10**(*NumericExpression*) 
  
**Log10**函数包含以下参数。 
  
|**参数名称**|**说明**|
|:-----|:-----|
| *NumericExpression*  <br/> |要计算其以 10 为底的对数的正数。  <br/> |
   
## <a name="remarks"></a>说明

**日志**功能允许您更改的底的对数，而不是使用基本 10。
