# <a name="replicate-function-access-custom-web-app"></a>复制 Access (Web 应用的复制) 

将文本值重复指定次数。
  
> [!IMPORTANT]
> Microsoft 不再建议在 SharePoint 中创建和使用 Access Web 应用程序。作为备选方法，请考虑使用 [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/)，生成适用于 Web 和移动设备的无代码业务解决方案。 
  
## <a name="syntax"></a>语法

 **复制** (*TextExpression* *、NumberOfTimes)* 
  
**复制** 函数包含下列参数。 
  
|**参数名称**|**说明**|
|:-----|:-----|
| *TextExpression*  <br/> |文本表达式。  <br/> |
| *NumberOfTimes*  <br/> |指定  *TextExpression*  重复次数的整数。  <br/> |
